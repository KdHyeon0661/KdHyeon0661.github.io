# 극좌표계에서의 곡선의 길이와 면적

## 1. 극좌표계 복습 (요점만)

한 점은 $$ (r,\theta) $$ 로 표현.  
직교좌표로의 변환은
$$
x=r\cos\theta,\qquad y=r\sin\theta.
$$
곡선은 보통 $$ r=f(\theta) $$ 로 주어집니다.

> 표현은 유일하지 않습니다: $$(r,\theta)\equiv(-r,\theta+\pi)\equiv(r,\theta+2k\pi)$$.

---

## 2. 면적 공식 (Area in Polar Coordinates)

### 2.1 원리와 공식
부채꼴의 미소면적은
$$
dA=\frac12\,r^2\,d\theta.
$$
따라서 $$\theta\in[\theta_1,\theta_2]$$ 에서
$$
A=\frac12\int_{\theta_1}^{\theta_2} \bigl[r(\theta)\bigr]^2\,d\theta.
$$

> **중요 포인트**
> - $$r(\theta)$$가 음수가 되어도 $$r^2$$이므로 면적 integrand는 양수입니다.  
> - 단, **각도 구간**을 잘못 잡으면 같은 영역을 **중복 계산**할 수 있습니다(자기 교차/꽃잎 곡선 주의).

### 2.2 예제 A — 레무니스케이트 반루프(문제에서 준 형태)
$$
r=\sqrt{\cos(2\theta)},\qquad \theta\in\Bigl[-\frac{\pi}{4},\frac{\pi}{4}\Bigr].
$$
면적
$$
A=\frac12\int_{-\pi/4}^{\pi/4}\cos(2\theta)\,d\theta
=\frac12\left[\frac12\sin(2\theta)\right]_{-\pi/4}^{\pi/4}
=\frac12.
$$

### 2.3 예제 B — 원 $$r=a$$ 의 전면적
$$
A=\frac12\int_0^{2\pi} a^2\,d\theta=\pi a^2.
$$

### 2.4 예제 C — 카디오이드 $$r=a(1+\cos\theta)$$ 의 전면적
$$
A=\frac12\int_0^{2\pi} a^2(1+\cos\theta)^2\,d\theta
=\frac{a^2}{2}\int_0^{2\pi}\bigl(1+2\cos\theta+\cos^2\theta\bigr)\,d\theta.
$$
여기서 $$\int_0^{2\pi}\cos\theta\,d\theta=0,\quad \int_0^{2\pi}\cos^2\theta\,d\theta=\pi.$$
따라서
$$
A=\frac{a^2}{2}(2\pi+\pi)=\frac{3\pi a^2}{2}.
$$

### 2.5 예제 D — 장미곡선 한 꽃잎의 면적 $$r=a\cos(n\theta)$$
한 꽃잎(예: $$-\frac{\pi}{2n}\le\theta\le \frac{\pi}{2n}$$)에 대해
$$
A_{\text{petal}}=\frac12\int r^2\,d\theta
=\frac{a^2}{2}\int_{-\pi/(2n)}^{\pi/(2n)}\cos^2(n\theta)\,d\theta
=\frac{\pi a^2}{4n}.
$$

## 3. 곡선의 길이 공식 (Arc Length in Polar Coordinates)

### 3.1 유도
매개화 $$x=r\cos\theta,\ y=r\sin\theta$$ 에서
$$
ds=\sqrt{\Bigl(\frac{dx}{d\theta}\Bigr)^2+\Bigl(\frac{dy}{d\theta}\Bigr)^2}\,d\theta
=\sqrt{r^2+\Bigl(\frac{dr}{d\theta}\Bigr)^2}\,d\theta.
$$
따라서
$$
L=\int_{\theta_1}^{\theta_2}\sqrt{\,r(\theta)^2+\bigl(r'(\theta)\bigr)^2\,}\,d\theta.
$$

### 3.2 예제 E — 나선 $$r=a\theta$$, $$0\le\theta\le 2\pi$$
$$
r'=a\quad\Rightarrow\quad
L=\int_0^{2\pi}\sqrt{(a\theta)^2+a^2}\,d\theta
=a\int_0^{2\pi}\sqrt{\theta^2+1}\,d\theta.
$$
원시함수는
$$
\int\sqrt{\theta^2+1}\,d\theta
=\frac12\Bigl(\theta\sqrt{\theta^2+1}+\operatorname{asinh}\theta\Bigr),
$$
따라서
$$
L=\frac{a}{2}\left[\,\theta\sqrt{\theta^2+1}+\operatorname{asinh}\theta\,\right]_0^{2\pi}
=\frac{a}{2}\Bigl(2\pi\sqrt{1+4\pi^2}+\operatorname{asinh}(2\pi)\Bigr).
$$

### 3.3 예제 F — 원 $$r=a$$ 의 둘레
$$
r'=0\ \Rightarrow\ L=\int_0^{2\pi}\sqrt{a^2}\,d\theta=2\pi a.
$$

### 3.4 예제 G — 카디오이드 $$r=a(1+\cos\theta)$$ 의 길이
$$
r'=-a\sin\theta,\quad r^2+(r')^2=a^2\bigl((1+\cos\theta)^2+\sin^2\theta\bigr)
=2a^2(1+\cos\theta)=4a^2\cos^2\frac{\theta}{2}.
$$
따라서
$$
L=\int_0^{2\pi}2a\left|\cos\frac{\theta}{2}\right|\,d\theta
=4a\int_0^{\pi}|\cos u|\,du
=8a.
$$

> **길이 계산 팁**: $$\sqrt{r^2+(r')^2}$$ 에서 삼각함수 항등식(예: $$1+\cos\theta=2\cos^2(\theta/2)$$)을 적극 활용하면 단순화가 큽니다.

## 4. 두 곡선 사이의 면적

바깥쪽 곡선이 $$r_1(\theta)$$, 안쪽이 $$r_2(\theta)$$ 라면
$$
A=\frac12\int_{\theta_1}^{\theta_2}\Bigl(r_1(\theta)^2-r_2(\theta)^2\Bigr)\,d\theta.
$$
경계각 $$\theta_1,\theta_2$$ 는 보통 **교점 방정식** $$r_1(\theta)=r_2(\theta)$$ 으로 얻습니다.

### 4.1 예제 H — $$r=2$$ 와 $$r=\cos\theta$$ 사이 ( $$0\le\theta\le\frac{\pi}{2}$$ )
$$
A=\frac12\int_{0}^{\pi/2}\bigl(4-\cos^2\theta\bigr)\,d\theta
=\frac12\left( 4\cdot\frac{\pi}{2}-\int_0^{\pi/2}\cos^2\theta\,d\theta\right).
$$
$$
\int_0^{\pi/2}\cos^2\theta\,d\theta=\frac{\pi}{4}
\ \Rightarrow\ 
A=\frac12\left(2\pi-\frac{\pi}{4}\right)=\frac{7\pi}{8}.
$$

### 4.2 예제 I — 교점 각 구해 나누어 적분(개략)
예를 들어 $$r_1=2\sin\theta$$, $$r_2=2\cos\theta$$ 의 **공통 부분**을 구하려면
$$
2\sin\theta=2\cos\theta\ \Rightarrow\ \tan\theta=1\ \Rightarrow\ \theta=\frac{\pi}{4}.
$$
사분면 대칭을 고려해 적절한 구간(예: $$[0,\pi/4]$$ 등)을 선택,  
해당 구간에서 **더 큰 반지름**이 바깥쪽 곡선이 되도록 분할/합산합니다.

## 5. 실전에서 자주 생기는 함정 & 체크리스트

- **각도 구간 설정**: 꽃잎/자기 교차 곡선은 같은 영역을 **두 번 통과**할 수 있음 → 구간을 꽃잎 단위로 잡고 **배수 곱**.
- **음의 반지름**: $$r<0$$ 이면 반대 방향 점. 면적 integrand는 $$r^2$$이라 괜찮지만, **경계각**을 올바르게 선택해야 중복/누락 방지.
- **길이 적분의 절댓값**: 길이 integrand는 본질적으로 양수(루트). 다만 **분할점(뾰족점, $$r=0$$)** 에서 미분가능성 확인 후 **구간 분할**.
- **교점 각 찾기**: 해석해가 없으면 **수치해**(이분법/뉴턴법)로 찾은 뒤 구간을 나누어 적분.
- **대칭성 활용**: 짝/홀성, $$\theta\to -\theta$$, $$\theta\to\theta+\pi$$ 등 대칭을 이용해 **적분 범위 축소** 후 배수.
- **특이 적분**: $$\sqrt{r^2+(r')^2}$$ 가 급변하는 구간은 **적응형 수치적분**을 사용.

## 6. 파이썬 수치 코드 (적응형 심슨, 교점 탐색 포함)

> 외부 라이브러리 없이 스케치 가능한 형태입니다. 실제 블로그/노트북에서 돌릴 수 있습니다.

```python
import math

# ---- 적응형 심슨 공통 루틴 ----
def _simpson(F, a, b):
    c = (a + b) / 2.0
    return (b - a) * (F(a) + 4.0 * F(c) + F(b)) / 6.0

def _adapt(F, a, b, S, tol):
    c = (a + b) / 2.0
    S1 = _simpson(F, a, c)
    S2 = _simpson(F, c, b)
    if abs(S1 + S2 - S) < 15 * tol:
        return S1 + S2 + (S1 + S2 - S) / 15.0
    return _adapt(F, a, c, S1, tol / 2.0) + _adapt(F, c, b, S2, tol / 2.0)

def integrate(F, a, b, tol=1e-10):
    return _adapt(F, a, b, _simpson(F, a, b), tol)

# ---- 면적/길이 ----
def area_polar(r, th1, th2, tol=1e-10):
    G = lambda th: 0.5 * (r(th) ** 2)
    return integrate(G, th1, th2, tol)

def length_polar(r, rp, th1, th2, tol=1e-10):
    G = lambda th: math.hypot(r(th), rp(th))  # sqrt(r^2 + r'^2)
    return integrate(G, th1, th2, tol)

# ---- 교점(각) 찾기: 이분법 ----
def bisect_theta(h, a, b, tol=1e-12, maxit=100):
    fa, fb = h(a), h(b)
    if fa == 0.0: return a
    if fb == 0.0: return b
    if fa * fb > 0:
        raise ValueError("No sign change in [a,b]; cannot bisect.")
    for _ in range(maxit):
        c = (a + b) / 2.0
        fc = h(c)
        if abs(fc) < tol or (b - a) < tol:
            return c
        if fa * fc < 0:
            b, fb = c, fc
        else:
            a, fa = c, fc
    return (a + b) / 2.0

# ---- 예제: 문제에서 준 것들 검증 ----
# 1) 레무니스케이트 반루프: r = sqrt(cos(2θ)), θ ∈ [-π/4, π/4] → 면적 = 1/2
r1 = lambda th: math.sqrt(max(0.0, math.cos(2*th)))
A1 = area_polar(r1, -math.pi/4, math.pi/4)
print("레무니스케이트 반루프 면적 ≈", A1)  # ≈ 0.5

# 2) 원 r=a 의 둘레/면적: a=1
r_circle  = lambda th: 1.0
rp_circle = lambda th: 0.0
A2 = area_polar(r_circle, 0.0, 2*math.pi)  # ≈ π
L2 = length_polar(r_circle, rp_circle, 0.0, 2*math.pi)  # ≈ 2π
print("원 면적 ≈", A2, ", 둘레 ≈", L2)

# 3) 나선 r=aθ 의 길이(0~2π), a=1
r_sp  = lambda th: th
rp_sp = lambda th: 1.0
L3 = length_polar(r_sp, rp_sp, 0.0, 2*math.pi)
print("나선 길이(수치) ≈", L3)

# 4) 카디오이드 r = 1 + cosθ 의 면적/길이
r_c  = lambda th: 1.0 + math.cos(th)
rp_c = lambda th: -math.sin(th)
A4 = area_polar(r_c, 0.0, 2*math.pi)  # 이론값 1.5π
L4 = length_polar(r_c, rp_c, 0.0, 2*math.pi)  # 이론값 8
print("카디오이드 면적 ≈", A4, " (이론 1.5π), 길이 ≈", L4, " (이론 8)")

# 5) 두 곡선 사이 면적: r=2 와 r=cosθ, θ∈[0,π/2] → 7π/8
r_out = lambda th: 2.0
r_in  = lambda th: math.cos(th)
A5 = integrate(lambda th: 0.5*(r_out(th)**2 - r_in(th)**2), 0.0, math.pi/2)
print("두 곡선 사이 면적 ≈", A5, " (이론 7π/8)")
```

## 8. 연습문제(해설 스케치)

1) $$r=2\sin\theta$$ 의 전면적.  
\(\quad\)대칭 이용해 $$[0,\pi]$$ 적분: $$A=\frac12\int_0^\pi 4\sin^2\theta\,d\theta=2\cdot\frac{\pi}{2}=\pi.$$

2) $$r=2\cos\theta$$ 와 $$r=2\sin\theta$$ 의 공통영역 면적.  
\(\quad\)교점 $$\theta=\pi/4$$. 1사분면에서 대칭성 활용, 구간 분할 후 합산.

3) $$r=a(1-\cos\theta)$$ (내접 루프 없는 카디오이드)의 면적과 길이.  
\(\quad\)면적은 2.4의 변형으로 $$\frac{3\pi a^2}{2}$$, 길이는 동일하게 $$8a$$.

4) $$r^2=a^2\cos 2\theta$$ 한 루프의 면적.  
\(\quad\)$$|\theta|\le \pi/4$$ 에서 $$r^2\ge 0$$, 2.1 방식으로 $$A=\frac{a^2}{2}$$.
