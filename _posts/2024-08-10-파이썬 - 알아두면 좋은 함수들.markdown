# 파이썬 내장 함수 예제 모음

파이썬 내장 함수는 강력하면서도 간단합니다.

## 출력 및 입력

### `print()` / `input()` / `format()`

```python
name = input("이름을 입력하세요: ")
print("안녕하세요,", name)
print("오늘은 {}년 {}월입니다.".format(2025, 6))
```

## 반복과 순회

### `len()`

```python
print(len("hello"))       # 5
print(len([1, 2, 3]))      # 3
```

### `range()`

```python
for i in range(3):
    print(i)  # 0, 1, 2
```

### `enumerate()`

```python
colors = ['red', 'blue']
for idx, color in enumerate(colors):
    print(idx, color)
```

### `zip()`

```python
names = ['Tom', 'Jane']
scores = [90, 85]
for name, score in zip(names, scores):
    print(f"{name}: {score}")
```

### `reversed()`

```python
for ch in reversed("abc"):
    print(ch)  # c b a
```

### `sorted()`

```python
numbers = [3, 1, 2]
print(sorted(numbers))  # [1, 2, 3]
```

## 수치 관련 함수

### `sum()`, `max()`, `min()`

```python
nums = [1, 5, 3]
print(sum(nums))  # 9
print(max(nums))  # 5
print(min(nums))  # 1
```

### `abs()`

```python
print(abs(-7))  # 7
```

### `round()`

```python
print(round(3.14159, 2))  # 3.14
```

### `pow()`

```python
print(pow(2, 3))  # 8
```

### `divmod()`

```python
q, r = divmod(10, 3)
print(q, r)  # 3 1
```

## 조건과 논리 함수

### `all()` / `any()`

```python
print(all([True, True, False]))   # False
print(any([False, False, True]))  # True
```

### `bool()`

```python
print(bool(""))     # False
print(bool("hi"))   # True
```

### `isinstance()` / `callable()`

```python
print(isinstance(5, int))     # True
print(callable(print))        # True
print(callable(3))            # False
```

## 타입 변환 함수

### `int()`, `float()`, `str()`

```python
print(int("10"))     # 10
print(float("3.14")) # 3.14
print(str(123))      # "123"
```

### `chr()` / `ord()`

```python
print(chr(65))  # 'A'
print(ord('A')) # 65
```

### `bin()` / `oct()` / `hex()`

```python
print(bin(10))  # '0b1010'
print(oct(10))  # '0o12'
print(hex(10))  # '0xa'
```

## 고차 함수

### `map()`

```python
nums = ['1', '2', '3']
print(list(map(int, nums)))  # [1, 2, 3]
```

### `filter()`

```python
nums = [1, 2, 3, 4]
even = filter(lambda x: x % 2 == 0, nums)
print(list(even))  # [2, 4]
```

### `sorted(key=...)`

```python
words = ["hi", "hello", "a"]
print(sorted(words, key=len))  # ['a', 'hi', 'hello']
```

### `eval()`

```python
expr = "3 + 5 * (2 - 1)"
print(eval(expr))  # 8
```

### `id()` / `hash()`

```python
a = 100
print(id(a))       # 객체 고유 ID (주소)
print(hash("abc")) # 문자열 해시값
```

## 기타 함수

### `dir()`

```python
print(dir([1, 2, 3]))  # 리스트가 가진 속성과 메서드
```

### `help()`

```python
help(str)  # 문자열 타입의 도움말 보기
```

### `globals()` / `locals()`

```python
x = 10
print('x' in globals())  # True
print('x' in locals())   # True
```

### `getattr()`, `setattr()`, `hasattr()`, `delattr()`

```python
class Person:
    name = "Tom"

p = Person()
print(getattr(p, 'name'))     # Tom
print(hasattr(p, 'name'))     # True
setattr(p, 'age', 20)
print(p.age)                  # 20
delattr(p, 'name')
print(hasattr(p, 'name'))     # False
```
