# 단면적과 회전체에 의한 입체의 체적

입체의 체적을 적분으로 계산하는 대표적 방법 두 가지를 정리한다.

1) **단면적 적분법**: 축에 수직인 얇은 단면의 **넓이** \(A(x)\)를 적분  
2) **회전체 적분법**: 곡선을 축 둘레로 회전시켜 얻는 입체의 **체적**을 원판/원주껍질로 분해해 적분

핵심은 “**슬라이스의 넓이 또는 둘레·반지름**을 정확히 표현”하는 것이다.

---

## 1. 단면적에 의한 체적 (Volume by Cross-Sections)

### 1.1 개념
입체를 \(x\)-축에 **수직**으로 아주 얇게 자르면, 각 위치 \(x\)에서의 단면 넓이를 \(A(x)\)라 할 수 있다.  
이때 전체 체적은
$$
V=\int_{a}^{b}A(x)\,dx.
$$

- \([a,b]\): 입체가 존재하는 \(x\)-구간  
- \(A(x)\): 그 \(x\)에서의 **단면 넓이**

> 같은 원리로 \(y\)-축 수직 단면이면 \(V=\int_{c}^{d}A(y)\,dy\).

### 1.2 자주 쓰는 단면 형태의 넓이
단면의 **한 변(또는 지름)**이 구간의 **폭** \(W(x)\)일 때(예: 위곡선 \(-\) 아래곡선),
- **정사각형**: \(A(x)=W(x)^2\)
- **직사각형(높이 \(h\))**: \(A(x)=W(x)\,h\)
- **정삼각형**: \(A(x)=\dfrac{\sqrt{3}}{4}W(x)^2\)
- **직각이등변삼각형**: \(A(x)=\dfrac{1}{2}W(x)^2\)
- **반원**: \(A(x)=\dfrac{\pi}{8}W(x)^2\)  \(\big(\)반지름 \(=\tfrac{W}{2}\), 넓이 \(=\tfrac12\pi(\tfrac{W}{2})^2\big)\)

> **포인트**: 대부분 \(A(x)\)가 **\(W(x)^2\)에 비례**한다. 도형 상수만 맞추면 된다.

### 1.3 예제 A — 정사각형 단면
- 밑면: \(y=\sqrt{x}\) 와 \(y=0\) 사이, \(x\in[0,4]\)
- 각 \(x\)에서 **정사각형 단면**, 변 \(=W(x)=\sqrt{x}-0=\sqrt{x}\)

따라서
$$
A(x)=\big(\sqrt{x}\big)^2=x
\quad\Rightarrow\quad
V=\int_0^4 x\,dx=\left[\tfrac12x^2\right]_0^4=8.
$$

\(\boxed{V=8}\)

### 1.4 예제 B — 반원 단면
- 폭 \(W(x)=2-x\) (예: 밑면이 \(x\in[0,2]\)에서 위 직선 \(y=2-x\)와 아래 \(y=0\)로 주어진 띠)
- 단면이 **반원**이므로
  \(A(x)=\dfrac{\pi}{8}W(x)^2=\dfrac{\pi}{8}(2-x)^2\)

체적:
$$
V=\int_0^2 \frac{\pi}{8}(2-x)^2\,dx
=\frac{\pi}{8}\int_0^2 (2-x)^2\,dx
=\frac{\pi}{8}\cdot\frac{8}{3}
=\frac{\pi}{3}.
$$

\(\boxed{V=\tfrac{\pi}{3}}\)

## 2. 회전체의 체적 (Solids of Revolution)

하나의 곡선(혹은 영역)을 축 둘레로 회전시켜 만든 입체의 체적을 적분으로 구한다.

### 2.1 원판/와셔법 (Disk/Washer Method)
- **축에 수직**으로 얇게 자르면 단면은 **원판**(또는 **와셔**: 구멍 있는 원판)
- **x축 회전** (곡선 \(y=R(x)\) 위쪽만 회전):
  $$
  V=\pi\int_a^b \big(R(x)\big)^2\,dx
  $$
- **x축 회전(와셔)** — 위 \(y=R(x)\), 아래 \(y=r(x)\)로 둘러싼 영역을 회전:
  $$
  V=\pi\int_a^b\Big(\big(R(x)\big)^2-\big(r(x)\big)^2\Big)\,dx
  $$
- 동일 아이디어로 **y축 회전**이면 \(y\)로 적분:
  $$
  V=\pi\int_c^d\Big(\big(R(y)\big)^2-\big(r(y)\big)^2\Big)\,dy
  $$

#### 예제 C — 와셔: \(y=x\) vs \(y=x^2\)를 x축 회전
- 영역: \(x\in[0,1]\), 위 \(R(x)=x\), 아래 \(r(x)=x^2\)
$$
V=\pi\int_0^1\big(x^2-x^4\big)\,dx
=\pi\left[\tfrac13x^3-\tfrac15x^5\right]_0^1
=\pi\left(\tfrac13-\tfrac15\right)
=\frac{2\pi}{15}.
$$

\(\boxed{V=\tfrac{2\pi}{15}}\)

#### 예제 D — 축이 수평선 \(y=c\)일 때(와셔)
- 영역: \(y=\sqrt{x}\), \(y=0\), \(x\in[0,4]\)  
- **수평선 \(y=2\)** 둘레 회전
  - 바깥 반지름: \(R(x)=\text{축}\,2\)에서 아래 경계 \(0\)까지 거리 \(=2\)
  - 안쪽 반지름: \(r(x)=2-\sqrt{x}\) (축에서 위 경계까지)
  - 주의: **반지름은 거리**이므로 항상 **양수**로 두고, **바깥 ≥ 안쪽** 관계를 확인
$$
\begin{aligned}
V&=\pi\int_0^4\Big(R^2-r^2\Big)\,dx
=\pi\int_0^4\Big(4-(2-\sqrt{x})^2\Big)\,dx\\
&=\pi\int_0^4\big(4-(4-4\sqrt{x}+x)\big)\,dx
=\pi\int_0^4 (4\sqrt{x}-x)\,dx\\
&=\pi\left[ \frac{4\cdot 2}{3}x^{3/2}-\frac12x^2 \right]_0^4
=\pi\left( \frac{8}{3}\cdot 8 - 8 \right)
=\pi\left(\frac{64}{3}-8\right)
=\frac{40\pi}{3}.
\end{aligned}
$$

\(\boxed{V=\tfrac{40\pi}{3}}\)

### 2.2 원주껍질법 (Shell Method)
- 축에 **평행**한 얇은 **원주껍질**(shell)들의 합  
- **y축 회전**(가장 흔함): 반지름 \(=x\), 높이 \(=f(x)-g(x)\)
  $$
  V=2\pi\int_a^b \big(\text{반지름}\big)\big(\text{높이}\big)\,dx
  =2\pi\int_a^b x\,[f(x)-g(x)]\,dx
  $$
- **수직선 \(x=c\)** 둘레 회전:
  $$
  V=2\pi\int_a^b |x-c|\,[f(x)-g(x)]\,dx
  $$
- **x축 회전**은 \(y\)-적분으로(반지름 \(=y\)):
  $$
  V=2\pi\int_c^d y\,[x_{\text{right}}(y)-x_{\text{left}}(y)]\,dy
  $$

#### 예제 E — y축 회전: \(y=x\) vs \(y=x^2\)
- 영역: \(x\in[0,1]\), 높이 \(h(x)=x-x^2\), 반지름 \(=x\)
$$
V=2\pi\int_0^1 x\,(x-x^2)\,dx
=2\pi\int_0^1 (x^2-x^3)\,dx
=2\pi\left[\tfrac13-\tfrac14\right]
=\frac{\pi}{6}.
$$

\(\boxed{V=\tfrac{\pi}{6}}\)

> 같은 문제를 **와셔( \(y\)-적분 )**로 풀어도 같은 값이 나온다.  
> 방법 선택의 요령은 아래 “실전 팁” 참고.

#### 예제 F — 수직선 \(x=1\) 둘레 회전(셸)
- 영역: \(y=1-x^2\)와 \(y=0\), \(x\in[0,1]\)  
- 반지름 \(=|x-1|=1-x\), 높이 \(=1-x^2\)
$$
\begin{aligned}
V&=2\pi\int_0^1 (1-x)(1-x^2)\,dx
=2\pi\int_0^1 (1-x-x^2+x^3)\,dx\\
&=2\pi\left[1-\tfrac12-\tfrac13+\tfrac14\right]
=2\pi\cdot\frac{5}{12}
=\frac{5\pi}{6}.
\end{aligned}
$$

\(\boxed{V=\tfrac{5\pi}{6}}\)

## 3. 원판 vs 와셔 vs 셸 — 언제 무엇을 쓸까?

- **회전축에 수직**으로 자르기 쉬우면 **원판/와셔**가 편함  
  (반지름/내반지름만 표현하면 끝)
- **회전축에 평행**한 폭(높이)이 단순하면 **셸**이 유리  
  (특히 \(y\)-축 회전 & \(x\)-함수로 주어진 경우)
- **오프셋 축**(예: \(x=c\), \(y=c\))은  
  - 와셔: 반지름에 “축까지의 거리”를 반영  
  - 셸: 반지름이 \(|x-c|\) 또는 \(|y-c|\) 로 깔끔
- **곡선 사이 영역**일 때  
  - 와셔: \(\pi(R^2-r^2)\) 구조로 **제곱**을 먼저 정리  
  - 셸: \(2\pi(\text{반지름})(\text{높이})\) 로 **곱**이 단순

> **경험칙**:  
> - **y축 회전 + \(y=f(x)\)들** → 셸이 대체로 간단  
> - **x축 회전 + \(y=f(x)\)들** → 원판/와셔가 간단

## 5. 파이썬 예제 코드

> 교육/검증용 스니펫. 실제 실행엔 `numpy`, `matplotlib`, (선택) `sympy`가 필요.

### 5.1 단면적 적분으로 체적 구하기 (수치 근사)
```python
import numpy as np

def volume_by_cross_section(A, a, b, n=20000):
    """
    A(x): 단면 넓이 함수
    [a,b]에서 사다리꼴법으로 수치 적분
    """
    x = np.linspace(a, b, n+1)
    y = A(x)
    return np.trapz(y, x)

# 예제 A 재현: A(x)=x on [0,4] -> V=8
A = lambda x: x
V = volume_by_cross_section(A, 0.0, 4.0)
print("단면적 적분 체적(근사):", V)
```

### 5.2 와셔/원판법 (x-적분)
```python
import numpy as np

def volume_washer_x(R, r, a, b, n=20000):
    """
    R(x): 바깥 반지름, r(x): 안쪽 반지름 (없으면 0 함수)
    V = π ∫ (R^2 - r^2) dx
    """
    x = np.linspace(a, b, n+1)
    integrand = R(x)**2 - r(x)**2
    return np.pi * np.trapz(integrand, x)

# 예제 C: y=x(바깥), y=x^2(안쪽), [0,1]
R = lambda x: x
r = lambda x: x**2
Vw = volume_washer_x(R, r, 0.0, 1.0)
print("와셔 체적(근사):", Vw)
```

### 5.3 셸법 (y축 회전, x-적분)
```python
import numpy as np

def volume_shell_yaxis(height, a, b, n=20000):
    """
    y축 회전, 반지름= x, 높이 = height(x)
    V = 2π ∫ x * height(x) dx
    """
    x = np.linspace(a, b, n+1)
    integrand = x * height(x)
    return 2*np.pi * np.trapz(integrand, x)

# 예제 E: height(x)=x - x^2, [0,1]
height = lambda x: x - x**2
Vs = volume_shell_yaxis(height, 0.0, 1.0)
print("셸 체적(근사):", Vs)
```

### 5.4 오프셋 축 셸 (수직선 x=c)
```python
import numpy as np

def volume_shell_x_equals_c(height, c, a, b, n=20000):
    """
    수직선 x=c 둘레 회전
    반지름 = |x - c|, 높이 = height(x)
    """
    x = np.linspace(a, b, n+1)
    integrand = np.abs(x - c) * height(x)
    return 2*np.pi * np.trapz(integrand, x)

# 예제 F: c=1, height(x)=1 - x^2, x∈[0,1]
height = lambda x: 1 - x**2
Vx1 = volume_shell_x_equals_c(height, 1.0, 0.0, 1.0)
print("x=1 둘레 셸 체적(근사):", Vx1)
```

### 5.5 (선택) Sympy로 상징적 확인
```python
import sympy as sp
x = sp.symbols('x', real=True)

# 예제 C 확인: π ∫_0^1 (x^2 - x^4) dx
expr = sp.integrate(sp.pi*(x**2 - x**4), (x, 0, 1))
sp.simplify(expr)
# -> 2*pi/15
```

## 6. 고급 토픽 한 줌

- **Cavalieri의 원리**: 모든 \(x\)에서 단면 넓이가 같으면 체적도 같다.  
- **파푸스(파푸스-굴딘) 정리**: 평면도형을 축 둘레로 회전시킬 때,  
  체적 \(=\) (도형의 넓이) \(\times\) (도형 무게중심이 그리는 원둘레 길이).  
  복잡한 적분을 피할 때 강력하다(단, 적용 조건 확인 필수).  
- **매개변수/극좌표** 회전체: \(x=x(t),y=y(t)\)나 \(r=f(\theta)\)에서도  
  반지름·높이를 올바로 표현하면 같은 방식으로 계산 가능.

## 7. 요약

| 주제 | 핵심 공식 | 코멘트 |
|---|---|---|
| 단면 적분 | \(V=\displaystyle\int A(x)\,dx\) | 단면 도형 상수를 정확히 |
| 원판법 | \(V=\displaystyle\pi\int R(x)^2\,dx\) | 축에 수직 절단 |
| 와셔법 | \(V=\displaystyle\pi\int\big(R^2-r^2\big)\,dx\) | 구멍이 있으면 제곱 차 |
| 셸법(y축) | \(V=\displaystyle 2\pi\int x\cdot h(x)\,dx\) | 축에 평행 절단 |
| 셸( \(x=c\) ) | \(V=\displaystyle 2\pi\int |x-c|\cdot h(x)\,dx\) | 오프셋 축에 효과적 |
| 선택 요령 | 위 표 참고 | 더 단순한 표현을 주는 방법 선택 |

> **기억할 것**: 반지름/높이는 **거리**이므로 항상 양수.  
> 와셔는 **제곱**, 셸은 **곱**.  
> 경계(교점)와 구간 분할을 정확히!

### 연습 과제 (스스로 풀어보기)

1) \(y=\sqrt{x}\)와 \(y=\tfrac{x}{2}\) 사이 영역을 **x축** 둘레로 회전. 와셔로 \(V\) 계산.  
2) 같은 영역을 **y축** 둘레로 회전. 셸로 \(V\) 계산.  
3) 단면이 **정삼각형**(변 \(= \sqrt{x}\))인 입체를 \(x\in[0,4]\)에서 단면 적분으로 계산.  
4) 영역 \(0\le y\le 1-x^2\)를 **수평선 \(y=2\)** 둘레로 회전. 와셔로 \(V\) 계산.
