# 수열의 극한과 급수의 수렴 판정법

## 0. 개요: “수열 → 부분합 → 급수”

- **수열** \(\{a_n\}\): 숫자의 순서 있는 나열.
- **급수** \(\sum_{n=1}^\infty a_n\): 부분합 \(S_N=\sum_{n=1}^N a_n\)의 극한 \(S=\lim_{N\to\infty}S_N\)이 존재하면 **수렴**.

**기본 사실**  
- (Divergence Test) \(\sum a_n\)이 수렴하려면 \(\displaystyle \lim_{n\to\infty} a_n = 0\) 이어야 한다. (역은 성립 X)
- **절대수렴**: \(\sum |a_n|\) 수렴 ⇒ \(\sum a_n\)도 수렴.
- **조건수렴**: \(\sum a_n\) 수렴하지만 \(\sum |a_n|\) 발산.

---

## 1. 수열의 극한 — 정의, 기본법칙, 도구

### 1.1 \(\varepsilon\)–\(N\) 정의
$$
\lim_{n\to\infty} a_n = L \iff
\forall \varepsilon>0,\ \exists N\ \text{s.t.}\ n>N\Rightarrow |a_n-L|<\varepsilon.
$$

### 1.2 기본 연산법칙
수렴 수열 \(a_n\to A, b_n\to B\)이면
- \(a_n\pm b_n\to A\pm B\), \quad \(a_nb_n\to AB\), \quad \(a_n/b_n\to A/B\) (단 \(B\neq0\)).

### 1.3 유계·단조수열 수렴정리
- 단조증가(감소)이고 위(아래)로 유계면 수렴.

**예제 A (단조+유계):**  
\(a_1=1,\ a_{n+1}=\sqrt{1+a_n}\).  
증가: \(a_{n+1}-a_n=\sqrt{1+a_n}-a_n>0\) (충분히 큰 \(n\)에서 성립함을 귀납으로 확인).  
유계: \(a_n<2\) (귀납). 극한 \(L=\sqrt{1+L}\Rightarrow L^2-L-1=0\Rightarrow L=\frac{1+\sqrt5}{2}\).

### 1.4 샌드위치(끼인) 정리
\(a_n\le c_n\le b_n\)이고 \(\lim a_n=\lim b_n=L\)이면 \(\lim c_n=L\).

**예제 B:**  
\(\displaystyle a_n=\frac{\sin n}{n}\). \(-\frac1n\le a_n\le \frac1n\), 양 끝이 0으로 가므로 \(a_n\to0\).

### 1.5 부분수열, \(\limsup/\liminf\)
- \(\limsup a_n=\lim_{N\to\infty}\sup_{n\ge N} a_n\), \(\liminf a_n=\lim_{N\to\infty}\inf_{n\ge N} a_n\).
- 수렴하면 \(\liminf=\limsup=L\).

**예제 C (진동):** \(a_n=(-1)^n\Rightarrow\liminf=-1,\ \limsup=1\) (수렴 X).

### 1.6 스톨츠–체사로(Stolz–Cesàro) (수열형 로피탈)
\(b_n\) 증가·무한대,  
\(\displaystyle \lim_{n\to\infty}\frac{a_{n+1}-a_n}{b_{n+1}-b_n}=L \Rightarrow \lim_{n\to\infty}\frac{a_n}{b_n}=L.\)

**예제 D:** \(\displaystyle \lim_{n\to\infty}\frac{\ln n}{n^p}=0\) (여기선 연속함수 극한으로도 즉시 보임; 스톨츠로도 가능).

## 2. 급수의 기초 — 부분합, 코시 기준, 재배열

### 2.1 코시 기준(필요충분)
\(\sum a_n\) 수렴 \(\iff\) \(\forall\varepsilon>0\), \(\exists N:\ \forall m>n\ge N\), \(|\sum_{k=n+1}^m a_k|<\varepsilon\).

### 2.2 재배열(rearrangement)
- 절대수렴이면 재배열해도 합이 **불변**.
- 조건수렴이면 합을 임의의 실수로 **재배열** 가능(리만 재배열 정리) → **수치계산시 주의!**

---

## 3. **양항** 급수 전용 판정: 비교·극한비교·적분·응축(Cauchy condensation)

### 3.1 비교 판정(Comparison)
양의 \(a_n,b_n\).  
\(0\le a_n\le b_n\)이고 \(\sum b_n\) 수렴 ⇒ \(\sum a_n\) 수렴.  
\(a_n\ge b_n\ge0\)이고 \(\sum b_n\) 발산 ⇒ \(\sum a_n\) 발산.

**예제 1:** \(\displaystyle \sum \frac{n}{n^3+1}\).  
\(\frac{n}{n^3+1}\le \frac{n}{n^3}=\frac{1}{n^2}\). \(p=2>1\) 수렴 ⇒ 원급수 수렴.

### 3.2 극한 비교 판정(Limit Comparison)
양의 \(a_n,b_n\),  
\(\displaystyle c=\lim_{n\to\infty}\frac{a_n}{b_n}\in(0,\infty)\Rightarrow \sum a_n,\sum b_n\) **동시 수렴/발산**.

> **끝값이 0 또는 \(\infty\)**인 경우:  
> \(\lim a_n/b_n=0\)이고 \(\sum b_n\) 수렴 ⇒ \(\sum a_n\) 수렴.  
> \(\lim a_n/b_n=\infty\)이고 \(\sum b_n\) 발산 ⇒ \(\sum a_n\) 발산.

**예제 2:** \(\displaystyle \sum \frac{\sqrt{n}+1}{n^{3/2}+n}\).  
\(a_n\sim \frac{\sqrt{n}}{n^{3/2}}=\frac1{n}\Rightarrow\) 조화급수와 동치 → **발산**.

### 3.3 적분 판정(Integral Test)
단조감소 \(f(n)=a_n>0\).  
\(\displaystyle \int_1^\infty f(x)\,dx\) 수렴 ⇔ \(\sum a_n\) 수렴.

- **p-급수**: \(\sum 1/n^p\)은 \(p>1\) 수렴, \(p\le1\) 발산.
- **로그 보정**: \(\sum \frac{1}{n(\ln n)^p}\)은 \(p>1\) 수렴, \(p\le1\) 발산 (단 \(n\ge2\)).

**예제 3:** \(\displaystyle \sum_{n\ge2}\frac{1}{n(\ln n)^{1+\epsilon}}\) with \(\epsilon>0\) ⇒ 수렴.  
**예제 4:** \(\displaystyle \sum_{n\ge2}\frac{1}{n\ln n}\) ⇒ 발산.

> **오차추정(적분판정)**:  
> \(R_N=\sum_{n>N}a_n\)은 \(\displaystyle \int_{N+1}^\infty f\le R_N\le \int_{N}^\infty f\).

### 3.4 코시 응축(Cauchy Condensation) — 감소 양항에 강력
\(a_n\downarrow\) (단조감소, \(a_n\ge0\)).  
\(\displaystyle \sum_{n=1}^\infty a_n\) 수렴 \(\iff\) \(\displaystyle \sum_{k=0}^\infty 2^k a_{2^k}\) 수렴.

**예제 5:** \(a_n=1/n\) ⇒ 응축 후 \(\sum 2^k \cdot 1/2^k=\sum 1\) 발산 ⇒ 원래도 발산.  
**예제 6:** \(a_n=1/[n(\ln n)^p]\) (감소 성립 가정) ⇒ 응축 후 \(\sum 1/k^p\)로 귀결.

## 4. 비율·근 판정(전체/절대값 대상): 계승·지수형에 특효

### 4.1 비판정법 (Ratio Test)
$$
L=\lim_{n\to\infty}\left|\frac{a_{n+1}}{a_n}\right|.
$$
- \(L<1\) ⇒ **절대수렴**  
- \(L>1\) 또는 \(=\infty\) ⇒ **발산**  
- \(L=1\) ⇒ **판정불가**

**예제 7:** \(\displaystyle \sum \frac{3^n}{n!}\):  
\(\left|\frac{a_{n+1}}{a_n}\right|=\frac{3}{n+1}\to0<1\Rightarrow\) 수렴.

**예제 8 (주의: \(L=1\))**: \(\sum \frac{1}{n}\) 또는 \(\sum \frac{1}{n^2}\) 모두 \(L=1\). 하나는 발산, 하나는 수렴 → **다른 판정 필요**.

**예제 9:** \(\displaystyle \sum \frac{(n!)^2}{(2n)!}\).  
\(\frac{a_{n+1}}{a_n}=\frac{(n+1)^2}{(2n+2)(2n+1)}=\frac{n+1}{2(2n+1)}\to \frac14<1\) ⇒ 절대수렴.  
(참고: \(a_n\sim \sqrt{\pi n}/4^n\), 지수적 감쇠)

**예제 10 (발산):** \(\sum \frac{n!}{3^n}\): \(\frac{a_{n+1}}{a_n}=\frac{n+1}{3}\to\infty\) ⇒ 발산.

### 4.2 근판정법 (Root Test)
$$
L=\lim_{n\to\infty}\sqrt[n]{|a_n|}.
$$
- \(L<1\) ⇒ 절대수렴, \(L>1\) ⇒ 발산, \(L=1\) ⇒ 불가.

**예제 11:** \(\sum \left(\frac{n}{n+1}\right)^n\).  
\(\sqrt[n]{a_n}=\frac{n}{n+1}\to1\Rightarrow\) 불가 → 다른 판정: \(\left(\frac{n}{n+1}\right)^n\to e^{-1}\)이므로 항이 0이 아니어서 **급수 발산**(Divergence Test!).

## 5. 교대·진동 급수: 라이프니츠, 디리클레, 아벨

### 5.1 라이프니츠(교대급수) 판정
\(\displaystyle \sum (-1)^{n+1}b_n\), \(b_n\downarrow\), \(b_n\to 0\) ⇒ **수렴**.  
오차: \(|S-S_N|\le b_{N+1}\).

**예제 12:** \(\displaystyle \sum (-1)^{n+1}\frac{1}{n}\) (**교대 조화급수**) ⇒ 수렴(단, 절대수렴 X).

**예제 13 (조건수렴):** \(\displaystyle \sum (-1)^n \frac{\ln n}{n}\).  
\(\frac{\ln n}{n}\downarrow\) (충분히 큰 \(n\)), \(\to0\) ⇒ **수렴**(절대값은 \(\sum \ln n/n\) 발산).

### 5.2 디리클레(Dirichlet)·아벨(Abel) — 곱형태에서 유용
- **Dirichlet**: 부분합이 유계인 \(A_n=\sum_{k=1}^n a_k\), \(b_n\downarrow\to 0\) ⇒ \(\sum a_n b_n\) 수렴.
- **Abel**: \(\sum a_n\) 부분합 유계, \(\{b_n\}\) 단조·유계 ⇒ \(\sum a_n b_n\) 수렴.

**예제 14:** \(\displaystyle \sum \frac{\sin n}{n}\) — Dirichlet로 **조건수렴** (절대수렴 X).

---

## 6. 그 외 고급 비율류: Raabe–Duhamel (경계 강화)

**Raabe 판정**  
양의 \(a_n\), \(\displaystyle R=\lim_{n\to\infty} n\left(\frac{a_n}{a_{n+1}}-1\right)\).
- \(R>1\) ⇒ 수렴, \(R<1\) ⇒ 발산, \(R=1\) ⇒ 불가.

경계가 \(L=1\)에서 멈추는 비판정법을 한 단계 더 밀어붙일 때 사용.

---

## 7. “어떤 판정법을 쓸까?” — 결정 트리

1) 항이 0으로 **안**가면 → 즉시 **발산**.  
2) **양항**인가?  
   - 네 → **비교/극한비교/적분/응축**부터.  
   - 계승·지수·멱지수 혼합이면 **비/근 판정** 시도.  
3) **부호 교대**인가?  
   - 네 → **라이프니츠**, 또는 **Dirichlet/Abel**.  
4) 비/근의 \(L=1\)로 **막히면** → 다른 판정(비교, 응축, 적분, Raabe).  
5) 절대수렴 여부가 중요하면 \(\sum|a_n|\) 먼저 판단.

## 8. 대표 예제 모음 (풀이는 요지 중심)

### 예제 A — 극한비교
\[
\sum_{n=1}^\infty \frac{n}{n^3+1},\quad b_n=\frac{1}{n^2}.
\]
\(\frac{a_n}{b_n}=\frac{n^3}{n^3+1}\to1\Rightarrow\) **수렴**.

### 예제 B — 적분판정(로그 보정)
\[
\sum_{n\ge 2}\frac{1}{n(\ln n)^{p}}.
\]
\(p>1\) 수렴, \(p\le1\) 발산.

### 예제 C — 비판정(계승 vs 지수)
\[
\sum \frac{3^n}{n!}\ \text{수렴},\qquad 
\sum \frac{n!}{3^n}\ \text{발산}.
\]

### 예제 D — 근판정
\[
\sum \left(\frac{n}{n+1}\right)^{n^2}.
\]
\(\sqrt[n]{|a_n|}=\left(\frac{n}{n+1}\right)^{n}\to e^{-1}<1\Rightarrow\) **절대수렴**.

### 예제 E — 교대 + 조건수렴
\[
\sum (-1)^{n}\frac{\ln n}{n} \ \text{수렴(조건)},\quad
\sum \frac{\ln n}{n}\ \text{발산}.
\]

### 예제 F — 디리클레
\[
\sum \frac{\sin n}{n} \ \text{수렴(조건)}.
\]

### 예제 G — 응축(강력)
\[
\sum \frac{1}{n(\ln n) (\ln\ln n)^{p}}
\]
(충분히 큰 \(n\)에서 감소) ⇒ 응축 후 \( \sum \frac{1}{k(\ln k)^{p}} \)와 유사 → \(p>1\) 수렴.

## 9. 오차·속도: 계산 실무 팁

- **교대급수 오차**: \(|S-S_N|\le b_{N+1}\).  
  목표 오차 \(\varepsilon\)이면 \(b_{N+1}<\varepsilon\)가 되게 \(N\) 선택.
- **적분판정 오차**: \(R_N\)이 \(\int_N^\infty f\)와 \(\int_{N+1}^\infty f\) 사이.
- **비/근 판정 \(L<1\)**: 실제 수렴 속도는 **지수적**(빠름).

## 10. 실습 코드 (Python) — 부분합·경험적 판단·오차

> 실행 환경에 따라 `numpy`, `mpmath`, `sympy`를 설치하세요.

### 10.1 부분합 계산기
```python
import math

def partial_sums(term, N):
    """term(n) = a_n, N까지 부분합 리스트 반환"""
    S, out = 0.0, []
    for n in range(1, N+1):
        a = term(n)
        S += a
        out.append(S)
    return out

# 예: 교대 조화급수
def alt_harmonic(n): return ((-1)**(n+1))/n

S = partial_sums(alt_harmonic, 20)
print("S_20 =", S[-1])  # 대략 ln 2 ≈ 0.693에 접근
```

### 10.2 비/근 판정의 L 추정
```python
def estimate_ratio(term, n0=10000):
    a1 = term(n0)
    a2 = term(n0+1)
    return abs(a2/a1)

def estimate_root(term, n0=10000):
    a = abs(term(n0))
    return a**(1.0/n0)

# 예: a_n = 3**n / math.factorial(n)
def a(n): return (3.0**n)/math.factorial(n)
print("Ratio L~", estimate_ratio(a))
print("Root  L~", estimate_root(a))
```

### 10.3 교대급수 오차 확인
```python
def alt_series(term_b, N):
    # sum (-1)^(n+1) b_n
    s = 0.0
    for n in range(1, N+1):
        s += ((-1)**(n+1)) * term_b(n)
    return s

def bn(n): return math.log(n)/n  # b_n = ln n / n (n>=2 권장)
for N in (10, 100, 1000):
    sN = alt_series(lambda k: bn(k if k>1 else 2), N)
    err_bound = bn(N+1)  # |S - S_N| ≤ b_{N+1}
    print(N, sN, "err ≤", err_bound)
```

### 10.4 적분판정 잔차 근사 (수치적)
```python
import mpmath as mp

def remainder_integral(f, N):
    # ∫_N^∞ f(x) dx 를 수치 구적(상황에 따라 조정)
    return mp.quad(lambda x: f(x), [N, mp.inf])

f = lambda x: 1.0/(x*(mp.log(x)**2))
print("R_100 ~", remainder_integral(f, 100))  # 수렴 사례(p=2)
```

## 11. “실수하기 쉬운 함정” 체크리스트

- [ ] \(\lim a_n=0\)은 **필요조건**일 뿐. (조화급수 반례!)  
- [ ] **비/근 판정에서 \(L=1\)** → 결론 없음. 다른 판정으로 전환.  
- [ ] **교대급수**: 단조감소와 극한0을 꼭 확인. (초기 몇 항 이후 단조면 OK)  
- [ ] **응축법**: \(a_n\)의 단조감소가 핵심 가정.  
- [ ] **적분판정**: \(f\)가 양의·단조감소 조건 확인.  
- [ ] **조건수렴 재배열** 주의 — 수치합산 순서에 따라 값이 흔들린다.

## 12. 요약 표

| 구분 | 내용 | 키 포인트 | 대표 예제 |
|---|---|---|---|
| 발산판정(기본) | \(\lim a_n\neq0\Rightarrow\) 발산 | 맨 먼저 확인 | \((n/(n+1))^n\to e^{-1}\) |
| 비교/극한비교 | 양항 전용 | \(a_n\sim b_n\)이면 운명 동행 | \(\sum n/(n^3+1)\) |
| 적분판정 | \(f\downarrow, f(n)=a_n\) | \(p\)-급수·로그보정 | \(\sum 1/(n(\ln n)^p)\) |
| 응축(Cauchy) | \(a_n\downarrow\) | 로그보정류에 강함 | \(\sum 1/(n\ln n)\) |
| 비판정 | 계승·지수형 | \(L<1\) 절대수렴, \(=1\) 불가 | \(\sum 3^n/n!\) |
| 근판정 | 거듭제곱 꼴 | \(L<1\) 절대수렴 | \(\sum ((n)/(n+1))^{n^2}\) |
| 교대(Leibniz) | \(b_n\downarrow\to0\) | 조건수렴·오차 \(\le b_{N+1}\) | \(\sum (-1)^{n+1}/n\) |
| Dirichlet/Abel | 곱형 진동 | 부분합 유계 + 단조 | \(\sum \sin n / n\) |

## 13. 마무리

- **전략**: (1) 항의 극한 → (2) 양항 여부 → (3) 구조 파악(계승/지수/로그) → (4) 적합한 판정 선택 → (5) 막히면 다른 판정.  
- **실무**: 절대수렴이면 안심, 조건수렴은 **오차·재배열** 민감성에 주의.  
- **코드**로 부분합·비/근 한계값을 대략 파악하고, 최종 판단은 **이론 판정법**으로 확정하세요.
