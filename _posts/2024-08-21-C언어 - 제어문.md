# 제어문: 조건문과 반복문

이번 편에서는 프로그램의 흐름을 제어하는 **조건문**과 **반복문**을 배워봅니다.

---

## 조건문 (if, else, else if)

```c
int score = 85;

if (score >= 90) {
    printf("A 학점\n");
} else if (score >= 80) {
    printf("B 학점\n");
} else {
    printf("C 이하\n");
}
// B 학점
```

> 조건은 반드시 괄호 `()`로 묶고, 블록 `{}` 사용 권장.
> 한 줄의 경우 `{}`을 쓰지 않아도 되나 변경 사항이 있을 때, 실수할 가능성이 있음.

---

## switch 문

```c
int menu = 2;

switch (menu) {
    case 1:
        printf("커피\n");
        break;
    case 2:
        printf("차\n");
        break;
    default:
        printf("잘못된 선택\n");
}
//차
```

- `break`는 각 case 블록의 실행 종료를 위해 필수
- `default`는 모든 case에 해당하지 않을 때 실행

---

## 반복문

### while 문

```c
int i = 0;
while (i < 5) {
    printf("%d\n", i);
    i++;
}

```

### for 문

```c
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}
```

### do-while 문

```c
int i = 0;
do {
    printf("%d\n", i);
    i++;
} while (i < 5);
```

> 최소 1회는 실행됨

---

## break / continue

```c
for (int i = 0; i < 5; i++) {
    if (i == 2) continue;  // 2는 출력 생략
    if (i == 4) break;     // 4부터 중단
    printf("%d\n", i);
}
```

---

## 정리

| 문법 | 기능 |
|------|------|
| `if ~ else` | 조건 분기 |
| `switch` | 여러 경우 분기 처리 |
| `while` | 조건에 따른 반복 |
| `for` | 카운터 기반 반복 |
| `do-while` | 최소 1회 실행 반복 |
| `break` / `continue` | 반복 제어 |
