# 평면 곡선의 길이 (Arc Length of a Plane Curve)

> 한 줄 요약: **곡선의 길이**는 “작은 선분 길이의 적분”이다.  
> 함수형 \(y=f(x)\), 역함수형 \(x=g(y)\), 매개변수형 \((x(t),y(t))\), 극좌표형 \(r=r(\theta)\) 모두에서
> \[
> \text{길이}=\int \sqrt{(\text{수평 변화})^2+(\text{수직 변화})^2}\,.
> \]
> 를 일관된 방식으로 얻는다.

---

## 1) 직관과 정의

곡선을 아주 작은 직선 조각으로 쪼개면, 각 조각의 길이는 피타고라스 정리에 의해
\[
ds=\sqrt{(dx)^2+(dy)^2}
\]
가 된다. 곡선을 따라 누적합(=적분)하면 전체 길이 \(L=\int ds\).

---

## 2) \(y=f(x)\) 형식 (함수형)

연속 미분 가능 \(f\)에 대해, \(x\in[a,b]\)에서
\[
L \;=\; \int_a^b \sqrt{1+\big(f'(x)\big)^2}\,dx.
\]

### 유도
\(ds=\sqrt{(dx)^2+(dy)^2}=\sqrt{1+(dy/dx)^2}\,dx\).

### 간단 체크리스트
- 미분 가능(혹은 **절대연속**이면 충분), \(f'\)가 구간에서 적절히 적분 가능해야 함.
- **수직 접선**(예: \(f'\to\infty\))이 있더라도 적분이 수렴하면 길이는 유한할 수 있다.

---

## 3) \(x=g(y)\) 형식 (역함수형)

\(y\in[c,d]\)에서
\[
L \;=\; \int_{c}^{d} \sqrt{1+\big(g'(y)\big)^2}\,dy.
\]
> \(x\)를 \(y\)의 함수로 다루는 것이 자연스러운 곡선(예: 수직선에 가까운 그래프)에 유리.

---

## 4) 매개변수 곡선 \(\ (x(t),y(t))\)

매끄러운 곡선 \(t\in[\alpha,\beta]\)에서
\[
L \;=\; \int_{\alpha}^{\beta} \sqrt{\big(x'(t)\big)^2+\big(y'(t)\big)^2}\,dt.
\]

### 성질 (중요)
- **매개변수 불변성**: 동일한 기하 곡선이면 매개변수를 바꿔도 길이는 같다.  
  (변수치환 \(u=\phi(t)\), \(\phi\) 단조 ⇒ 같은 적분값)
- **속력/단위접선**: \(v(t)=\sqrt{x'^2+y'^2}\), \( \mathbf{T}=\dfrac{(x',y')}{v}\).
- **호의 길이 매개화**: \( s(t)=\int_{\alpha}^{t} v(\tau)\,d\tau\).  
  \(s\)를 새 변수로 쓰면 항상 \(\|d\mathbf{r}/ds\|=1\).

---

## 5) 극좌표 곡선 \(r=r(\theta)\)

평면 좌표 \(x=r\cos\theta,\ y=r\sin\theta\)로부터
\[
L \;=\; \int_{\theta_1}^{\theta_2} \sqrt{\,r(\theta)^2+\big(r'(\theta)\big)^2\,}\,d\theta.
\]
> 원 \(r=R\): \(L=\int_0^{2\pi}\sqrt{R^2+0}\,d\theta=2\pi R\).

---

## 6) 기본 예제

### 예제 A — 포물선 \(y=x^2\), \(x\in[0,1]\)
\[
L=\int_0^1 \sqrt{1+(2x)^2}\,dx
=\int_0^1 \sqrt{1+4x^2}\,dx.
\]
표준 원시함수로 정밀 계산 가능:
\[
\int \sqrt{1+4x^2}\,dx
=\frac{x}{2}\sqrt{1+4x^2}+\frac{1}{4}\,\operatorname{asinh}(2x)+C.
\]
따라서
\[
\boxed{\,L=\frac{\sqrt{5}}{2}+\frac{1}{4}\operatorname{asinh}(2)
=\frac{1}{4}\big(2\sqrt{5}+\ln(2+\sqrt{5})\big)\, }.
\]

> 주의: 많은 곡선(예: 타원)의 호길이는 **초등함수로 닫히지 않음**(타원적분 필요).

---

### 예제 B — \(x=y^3\), \(y\in[0,1]\)
\[
L=\int_0^1 \sqrt{1+(dx/dy)^2}\,dy
=\int_0^1 \sqrt{1+9y^4}\,dy.
\]
초등원시함수 없음 ⇒ 수치적분으로 근사(아래 코드 참고).

---

### 예제 C — 1/4 원호 (매개변수)
\[
x(t)=\cos t,\quad y(t)=\sin t,\quad t\in\Big[0,\tfrac{\pi}{2}\Big].
\]
\[
L=\int_0^{\pi/2}\sqrt{(-\sin t)^2+(\cos t)^2}\,dt
=\int_0^{\pi/2}1\,dt=\boxed{\tfrac{\pi}{2}}.
\]

---

### 예제 D — 싸이클로이드 한 아치
\[
x=r(t-\sin t),\ y=r(1-\cos t),\quad t\in[0,2\pi].
\]
\[
x'=r(1-\cos t),\ y'=r\sin t
\Rightarrow v=\sqrt{x'^2+y'^2}
=r\sqrt{(1-\cos t)^2+\sin^2 t}
=2r\sin\!\big(\tfrac{t}{2}\big).
\]
\[
L=\int_0^{2\pi} 2r\sin\!\big(\tfrac{t}{2}\big)dt
=\Big[-4r\cos\!\big(\tfrac{t}{2}\big)\Big]_0^{2\pi}
=\boxed{8r}.
\]

---

### 예제 E — 카테나리 \(y=a\cosh(x/a)\), \([-d,d]\)
\[
y'= \sinh(x/a),\quad \sqrt{1+y'^2}=\sqrt{1+\sinh^2}=\cosh(x/a).
\]
\[
L=\int_{-d}^{d}\cosh(x/a)\,dx
=a\sinh(x/a)\Big|_{-d}^{d}
=\boxed{2a\sinh(d/a)}.
\]

---

### 예제 F — 로그나선 \(r=a e^{b\theta}\), \([\theta_1,\theta_2]\)
\[
r'=ab e^{b\theta}=br,\quad \sqrt{r^2+r'^2}=r\sqrt{1+b^2}.
\]
\[
L=\sqrt{1+b^2}\int_{\theta_1}^{\theta_2} r(\theta)\,d\theta
=\frac{a}{b}\sqrt{1+b^2}\,\Big(e^{b\theta_2}-e^{b\theta_1}\Big).
\]

---

## 7) “왜 때로는 어렵나?” — 타원과 타원적분

타원 \(x=a\cos t,\ y=b\sin t\)의 전체 둘레:
\[
L= \int_0^{2\pi} \sqrt{a^2\sin^2 t+b^2\cos^2 t}\,dt
=4a\,E(e),
\]
여기서 \( e=\sqrt{1-b^2/a^2} \), \(E\)는 (제2종) **완전 타원적분**.  
즉, 많은 실전 곡선 길이는 **초등함수**로 표현되지 않는다(수치/특수함수 필요).

---

## 8) 수렴·정칙성(regularity) 메모

- **절대연속**(또는 부분별 \(C^1\))이면 길이 적분이 정의되기 쉽다.  
- **꼬임/스파이크**가 무한히 많은 프랙탈(예: Koch curve)은 길이가 **무한대**.  
- **매개변수 독립성**: 길이는 오직 기하 형태에만 의존.

---

## 9) 수치적분 실습 코드 (파이썬)

> 주: 아래 코드는 로컬/노트북에서 실행해 확인하세요.

### 9.1 함수형 \(y=f(x)\) 길이
```python
import numpy as np

def arc_length_f(fprime, a, b, n=20000):
    x = np.linspace(a, b, n+1)
    fx = fprime(x)
    integrand = np.sqrt(1.0 + fx*fx)
    h = (b - a)/n
    # 사다리꼴
    return h * (0.5*integrand[0] + integrand[1:-1].sum() + 0.5*integrand[-1])

# 예: y = x^2, f'(x)=2x, [0,1]
fprime = lambda x: 2.0*x
print("Length ~", arc_length_f(fprime, 0.0, 1.0))
# 이론값: (1/4)*(2*sqrt(5) + asinh(2))
```

### 9.2 매개변수형 길이
```python
def arc_length_param(xp, yp, t0, t1, n=20000):
    t = np.linspace(t0, t1, n+1)
    vx = xp(t); vy = yp(t)
    integrand = np.sqrt(vx*vx + vy*vy)
    h = (t1 - t0)/n
    return h * (0.5*integrand[0] + integrand[1:-1].sum() + 0.5*integrand[-1])

# 싸이클로이드: x' = r(1 - cos t), y' = r sin t
r = 1.0
xp = lambda t: r*(1-np.cos(t))
yp = lambda t: r*np.sin(t)
print("Cycloid one-arch length ~", arc_length_param(xp, yp, 0.0, 2*np.pi))  # ~ 8r
```

### 9.3 극좌표형 길이
```python
def arc_length_polar(r, rp, th0, th1, n=20000):
    th = np.linspace(th0, th1, n+1)
    rv = r(th); rpv = rp(th)
    integrand = np.sqrt(rv*rv + rpv*rpv)
    h = (th1 - th0)/n
    return h * (0.5*integrand[0] + integrand[1:-1].sum() + 0.5*integrand[-1])

# 로그나선 r = a e^{bθ}
a, b = 2.0, 0.3
r  = lambda th: a*np.exp(b*th)
rp = lambda th: b*r(th)
print("Log-spiral length ~", arc_length_polar(r, rp, 0.0, 2.0))
# 이론값: (a/b)*sqrt(1+b^2)*(e^{b*2}-1)
```

## 10) 시각화(선택)

아래는 포물선 길이 적분의 **적분핵** \(\sqrt{1+4x^2}\) 시각화 예시.
```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 1, 400)
integrand = np.sqrt(1 + 4*x*x)

plt.plot(x, integrand, label=r"$\sqrt{1+4x^2}$")
plt.title("Arc-length integrand for y=x^2 on [0,1]")
plt.xlabel("x"); plt.ylabel("integrand")
plt.grid(True); plt.legend()
plt.show()
```

## 11) 실전 체크리스트 & 함정

- [ ] **형식 선택**: \(y=f(x)\)? \(x=g(y)\)? 매개변수? 극좌표? (가장 쉬운 형식을 택하라)
- [ ] **분할 필요**: 코너/첨점(매끄럽지 않은 점)에서는 구간을 나눠 적분.
- [ ] **원시함수 집착 금지**: 초등원시함수 없으면 수치적분/특수함수 사용.
- [ ] **매개변수 독립성**: 속도 \(v=\sqrt{x'^2+y'^2}\)가 핵심. 재매개화해도 결과는 동일.
- [ ] **스케일/단위**: 좌표 스케일이 길이에 그대로 반영(단위 확인).

## 12) 추가 연습문제 (정답 스케치)

1. \(y=\tfrac{1}{2}x^{3/2}\), \(x\in[0,4]\). 길이?  
   \(f'(x)=\tfrac{3}{4}x^{1/2}\Rightarrow L=\int_0^4\sqrt{1+\tfrac{9}{16}x}\,dx\) (치환 \(u=1+\tfrac{9}{16}x\)).

2. 극좌표 \(r=2(1+\cos\theta)\) (카디오이드), \(\theta\in[0,2\pi]\). 전체 둘레?  
   \(L=\int_0^{2\pi}\sqrt{r^2+(r')^2}\,d\theta\) 계산(삼각항 정리).

3. 타원 \(x=3\cos t,\ y=2\sin t\) 반바퀴 길이?  
   \(L=\int_0^{\pi}\sqrt{9\sin^2 t+4\cos^2 t}\,dt=2\cdot 3\,E(e)\) (특수함수 사용), \(e=\sqrt{1-(2/3)^2}\).

4. \(x=t^3-3t,\ y=3t^2-1,\ t\in[0,2]\) (카시니/트로코이드류).  
   \(L=\int_0^2\sqrt{(3t^2-3)^2+(6t)^2}\,dt\) (수치적분 권장).

## 13) 요약 표

| 형식 | 길이 공식 | 비고 |
|---|---|---|
| \(y=f(x)\) | \(\displaystyle L=\int_a^b\sqrt{1+(f')^2}\,dx\) | 수직 접선 있어도 적분 수렴 가능 |
| \(x=g(y)\) | \(\displaystyle L=\int_c^d\sqrt{1+(g')^2}\,dy\) | \(y\)-기준이 유리할 때 |
| \((x(t),y(t))\) | \(\displaystyle L=\int_{\alpha}^{\beta}\sqrt{x'^2+y'^2}\,dt\) | 매개변수 불변성 |
| \(r(\theta)\) | \(\displaystyle L=\int_{\theta_1}^{\theta_2}\sqrt{r^2+(r')^2}\,d\theta\) | 극좌표형 |

---

### 끝맺음

- 모든 길이 공식은 **작은 선분 길이의 적분**으로 귀착된다.  
- “초등원시함수 없음”은 흔한 일 → **수치적분/특수함수**를 자연스럽게 사용하자.  
- 실전에서는 **형식 선택**과 **구간 분할**, 그리고 **수치 안정성**이 성패를 가른다.
