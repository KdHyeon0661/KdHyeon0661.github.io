# 면적 (Area)

면적은 **평면 도형이 차지하는 크기**입니다. 해석학 관점에서는 **리만 합 → 정적분**으로 정식화되고, 좌표 표현에 따라
- \(x\)-방향 적분 \( \int f(x)\,dx \),
- \(y\)-방향 적분 \( \int g(y)\,dy \),
- **매개변수 곡선**의 선적분 \( \int y\,dx \) (또는 그린 정리 \( \tfrac12\oint(x\,dy-y\,dx) \)),
- **극좌표**의 \( \tfrac12\int r(\theta)^2\,d\theta \),
- **다각형**의 슈레이스(신발끈) 공식
등으로 연결됩니다.

---

## 1) 가장 기본: 기초 도형의 면적

| 도형 | 면적 |
|---|---|
| 정사각형(한 변 \(a\)) | \(A=a^2\) |
| 직사각형(\(a\times b\)) | \(A=ab\) |
| 평행사변형(변 \(a,b\), 끼인각 \(\theta\)) | \(A=ab\sin\theta\) |
| 삼각형(밑변 \(b\), 높이 \(h\)) | \(A=\tfrac12 bh\) |
| 삼각형(두 변 \(a,b\), 끼인각 \(\theta\)) | \(A=\tfrac12 ab\sin\theta\) |
| 원(반지름 \(r\)) | \(A=\pi r^2\) |
| 부채꼴(반지름 \(r\), 중앙각 \(\theta\) 라디안) | \(A=\tfrac12 r^2\theta\) |

> **미세한 사각형으로 쪼개 합한다 → 정적분**으로 일반화됩니다.

---

## 2) 정적분으로 보는 면적

### 2.1 곡선 아래 면적 (비음수 함수)
연속이고 \(f(x)\ge 0\) 이면
$$
A=\int_a^b f(x)\,dx.
$$

**예제 2.1** \(f(x)=x,\ [0,2]\):  
\(\displaystyle A=\int_0^2 x\,dx=\frac12x^2\big|_0^2=2.\)

### 2.2 **기하학적 면적** vs **부호 있는 적분**
함수가 음수에도 내려가면 **면적**은
$$
\boxed{A=\int_a^b |f(x)|\,dx},
$$
반면 \(\int_a^b f(x)\,dx\) 는 **부호를 가진 면적(순면적)** 입니다.

**예제 2.2** \(f(x)=\sin x,\ [0,2\pi]\)  
- \(\int_0^{2\pi}\sin x\,dx=0\) (위아래가 상쇄)  
- 기하학적 면적: \(\int_0^{\pi}\sin x\,dx+\int_\pi^{2\pi}|\sin x|\,dx=2+2=4.\)

> **실전 팁**: 부호가 바뀌는 지점(영점)을 분할해 절댓값을 벗겨 적분합니다.

## 3) 두 곡선 사이의 면적 (상하 슬라이싱)

두 함수 \(f(x)\ge g(x)\) 가 \([a,b]\)에서 만날 때
$$
\boxed{A=\int_a^b\big(f(x)-g(x)\big)\,dx.}
$$

**예제 3.1** \(f(x)=x,\ g(x)=x^2\). 교점 \(x=0,1\).  
\(\displaystyle A=\int_0^1(x-x^2)\,dx=\frac12-\frac13=\frac16.\)

### 3.1 교점이 여러 개인 경우(위아래가 바뀌는 경우)
분할해 각 구간에서 “위-아래”를 다시 정합니다.

**예제 3.2** \(f(x)=\sin x,\ g(x)=x/2\) on \([0,2\pi]\).  
- 교점은 \(\sin x=x/2\) 수치해로 찾기 → 구간을 나눠 \(\sin x\)와 \(x/2\) 중 큰 함수를 위 함수로 선택 → 각 구간 적분 합.

> 아래 코드 §10.2에 **교점 탐색→면적 적분** 예제가 있습니다.

## 4) \(y\)-방향 슬라이싱 (좌우 적분)

수직 슬라이싱이 불편하면, **수평 슬라이스**로 바꿉니다.  
영역을 \(x = \text{오른쪽}(y) - \text{왼쪽}(y)\) 형태로 적분:

$$
\boxed{A=\int_{y=c}^{d}\Big(x_{\text{right}}(y)-x_{\text{left}}(y)\Big)\,dy.}
$$

**예제 4.1 (수평 슬라이스가 유리한 경우)**  
영역: 포물선 \(y=x^2\) 와 직선 \(x=1\) 사이.  
- \(y\in[0,1]\), 왼쪽 \(x_{\text{left}}(y)=-\sqrt{y}\) 가 아니라 **여기선 오른쪽 벽이 \(x=1\), 왼쪽 벽이 \(x=\sqrt{y}\)** (첫사분면만).  
- \(A=\int_0^1(1-\sqrt{y})\,dy=\big[y-\tfrac23 y^{3/2}\big]_0^1=\tfrac13.\)

> 동일 영역을 \(x\)-적분으로 잡으면 루트가 뒤섞여 구간분할이 번거로울 수 있습니다. **슬라이스 방향 선택**이 핵심.

## 5) 매개변수 곡선의 면적 (선적분/그린 정리)

### 5.1 \(x=x(t), y=y(t)\)가 **x-축 기준 면적**을 만들 때
“곡선 아래 면적”은
$$
A=\int y\,dx=\int_{t=a}^{b} y(t)\,x'(t)\,dt.
$$

### 5.2 **폐곡선**이 둘러싼 면적(그린 정리)
**단순 폐곡선** \(\mathcal{C}\) (반시계)라면
$$
\boxed{A=\frac12\oint_{\mathcal{C}}(x\,dy-y\,dx)
=\frac12\int_{t=a}^{b}\big(x(t)y'(t)-y(t)x'(t)\big)\,dt.}
$$

**예제 5.1 (원 \(x=\cos t, y=\sin t\), \(t\in[0,2\pi]\))**  
\(\displaystyle A=\tfrac12\!\int_0^{2\pi}(\cos t\cdot \cos t-\sin t\cdot(-\sin t))\,dt=\tfrac12\!\int_0^{2\pi}(\cos^2 t+\sin^2 t)\,dt=\pi.\)

**예제 5.2 (싸이클로이드 아치의 면적)**  
\(x=r(t-\sin t),\ y=r(1-\cos t)\), \(t\in[0,2\pi]\).  
아치 아래 면적:
\[
A=\int y\,dx=\int_0^{2\pi} r(1-\cos t)\cdot r(1-\cos t)\,dt
=r^2\!\int_0^{2\pi}(1-2\cos t+\cos^2 t)\,dt
=r^2(2\pi+0+\pi)=\boxed{3\pi r^2}.
\]

> 싸이클로이드 한 아치 아래 면적이 **굴린 원 3개**의 면적과 같다는 유명한 결과.

## 6) 극좌표 곡선 \(r=r(\theta)\) 의 면적

\(\theta\in[\alpha,\beta]\)에서 부채꼴 미소면적을 누적하면
$$
\boxed{A=\frac12\int_{\alpha}^{\beta} \big(r(\theta)\big)^2\,d\theta.}
$$

**예제 6.1 (원 \(r=R\), \([0,2\pi]\))**  
\(\displaystyle A=\tfrac12\int_0^{2\pi}R^2\,d\theta=\pi R^2.\)

**예제 6.2 (카디오이드 \(r=1+\cos\theta\), \([0,2\pi]\))**  
\[
A=\tfrac12\int_0^{2\pi}(1+\cos\theta)^2\,d\theta
=\tfrac12\int_0^{2\pi}(1+2\cos\theta+\cos^2\theta)\,d\theta
=\tfrac12\big(2\pi+0+\pi\big)=\boxed{\tfrac{3\pi}{2}}.
\]

## 7) 다각형의 면적 — 슈레이스(신발끈) 공식

꼭짓점 \(P_i=(x_i,y_i)\) 가 **시계/반시계**로 정렬된 단순 다각형의 면적(부호 포함)
$$
A_{\text{signed}}=\frac12\sum_{i=1}^{n} (x_i y_{i+1}-y_i x_{i+1}),\quad (x_{n+1},y_{n+1})=(x_1,y_1).
$$
**기하학적 면적**은 \(|A_{\text{signed}}|\).

**예제 7.1** \( (0,0)\to(2,0)\to(2,1)\to(0,1)\)  
\(\displaystyle A=\tfrac12[(0\cdot0-0\cdot2)+(2\cdot1-0\cdot2)+(2\cdot1-1\cdot0)+(0\cdot0-1\cdot0)]=\tfrac12(0+2+2+0)=2.\)

> **주의**: 정점 순서가 **뒤집히면 부호가 바뀝니다**(시계/반시계).  
> **자가교차** 다각형은 이 공식의 부호-면적이 “루프별”로 상쇄될 수 있습니다(해석 주의).

## 8) 면적 계산 전략 — 어떤 공식을 쓸까?

- **축과 나란한 슬라이싱이 쉬운가?**  
  → \(x\)-적분(상하) 또는 \(y\)-적분(좌우).
- **곡선이 매개변수/극좌표로 더 간단한가?**  
  → \(\int y\,dx\), \( \tfrac12\oint(x\,dy-y\,dx)\), \( \tfrac12\int r^2 d\theta \).
- **다각형인가?**  
  → 슈레이스.
- **식이 복잡해 닫힌형이 힘든가?**  
  → 수치적분(사다리꼴, 심프슨) 또는 **몬테카를로**.

## 9) 자주 하는 실수 & 체크리스트

- [ ] **절댓값** 필요 여부 확인(면적 vs 부호 있는 적분).  
- [ ] **교점**을 정확히 구해 **구간 분할**(위/아래가 바뀌는 곳).  
- [ ] 매개변수 **방향(시계/반시계)** 에 따른 부호 주의.  
- [ ] 극좌표 면적은 반드시 \( \tfrac12\int r^2 d\theta \) (종종 \(r\)를 그대로 넣는 실수).  
- [ ] 다각형: **정점 순서** 일관성(시계/반시계).  
- [ ] 수치법: 간격 \(h\)를 줄이며 **수렴 확인**(오차 추정).

### 10.1 기본: 함수 아래 면적 (사다리꼴/심프슨)

```python
import numpy as np

def trapz(f, a, b, n=10000):
    x = np.linspace(a, b, n+1)
    y = f(x)
    h = (b - a) / n
    return h*(0.5*y[0] + y[1:-1].sum() + 0.5*y[-1])

def simpson(f, a, b, n=10000):
    if n % 2 == 1: n += 1  # 짝수 분할
    x = np.linspace(a, b, n+1)
    y = f(x)
    h = (b - a) / n
    return (h/3)*(y[0] + y[-1] + 4*y[1:-1:2].sum() + 2*y[2:-2:2].sum())

# 예: |sin x|의 [0, 2π] 면적 → 4
f = np.sin
A_trap = trapz(lambda x: np.abs(f(x)), 0, 2*np.pi, n=20000)
A_simp = simpson(lambda x: np.abs(f(x)), 0, 2*np.pi, n=20000)
print("Trapz:", A_trap, "Simpson:", A_simp)
```

### 10.2 두 곡선 사이 면적: 교점 수치해+구간 분할

```python
import numpy as np

def bisect_root(h, a, b, iters=60):
    fa, fb = h(a), h(b)
    if fa*fb > 0:
        return None  # 부호변화 없음
    for _ in range(iters):
        m = 0.5*(a+b)
        fm = h(m)
        if fa*fm <= 0:
            b, fb = m, fm
        else:
            a, fa = m, fm
    return 0.5*(a+b)

def intersections(f, g, a, b, M=2000):
    xs = np.linspace(a, b, M+1)
    hs = f(xs) - g(xs)
    roots = []
    for i in range(M):
        if hs[i]==0: roots.append(xs[i])
        if hs[i]*hs[i+1] < 0:
            r = bisect_root(lambda x: f(x)-g(x), xs[i], xs[i+1])
            if r is not None: roots.append(r)
    roots = np.array(sorted(set(np.round(roots, 12))))
    return roots

def area_between(f, g, a, b, n=20000):
    # [a,b] 내 교점 분할
    pts = intersections(f, g, a, b)
    cut = np.concatenate(([a], pts[(pts>a)&(pts<b)], [b]))
    A = 0.0
    for u, v in zip(cut[:-1], cut[1:]):
        # 이 구간에서 위/아래 판단
        mid = 0.5*(u+v)
        top, bot = (f, g) if f(mid) >= g(mid) else (g, f)
        A += simpson(lambda x: top(x) - bot(x), u, v, n=max(200, int((v-u)*2000)))
    return A

# 예: f=sin x, g=x/2 in [0, 2π]
f = np.sin
g = lambda x: x/2
A = area_between(f, g, 0, 2*np.pi)
print("Area between sin x and x/2 on [0, 2π]:", A)
```

### 10.4 매개변수/폐곡선 면적 (∫y dx, 1/2∮(x dy - y dx))

```python
import numpy as np

def param_area_y_dx(x, y, a, b, n=20000):
    t = np.linspace(a, b, n+1)
    xt = x(t); yt = y(t)
    # y dx ~ sum y_i * (x_{i+1}-x_i)
    return np.sum(yt[:-1]*(xt[1:]-xt[:-1]))

def closed_curve_area(x, y, a, b, n=20000):
    # 1/2 ∮ (x dy - y dx)
    t = np.linspace(a, b, n+1)
    xt = x(t); yt = y(t)
    dx = xt[1:] - xt[:-1]
    dy = yt[1:] - yt[:-1]
    return 0.5*np.sum(xt[:-1]*dy - yt[:-1]*dx)

# 예: 원 x=cos t, y=sin t, t∈[0,2π]
x = np.cos; y = np.sin
A_closed = closed_curve_area(x, y, 0, 2*np.pi, n=200000)
print("Closed area (unit circle):", A_closed)  # ≈ π
```

### 10.5 극좌표 면적 (½∫ r^2 dθ)

```python
import numpy as np

def polar_area(r, a, b, n=20000):
    th = np.linspace(a, b, n+1)
    return 0.5 * simpson(lambda th: r(th)**2, a, b, n=n)

# 예: 카디오이드 r=1+cosθ on [0, 2π] → 3π/2
r = lambda th: 1 + np.cos(th)
A_polar = polar_area(r, 0, 2*np.pi, n=20000)
print("Polar area (cardioid):", A_polar)
```

### 10.6 다각형 슈레이스

```python
import numpy as np

def shoelace_area(points):
    # points: [(x1,y1),...,(xn,yn)] (폐합 필요X)
    P = np.asarray(points, dtype=float)
    x = P[:,0]; y = P[:,1]
    x1 = np.roll(x, -1); y1 = np.roll(y, -1)
    signed = 0.5*np.sum(x*y1 - y*x1)
    return abs(signed), signed

poly = [(0,0),(2,0),(2,1),(0,1)]
absA, signedA = shoelace_area(poly)
print("Polygon area:", absA, "signed:", signedA)
```

### 10.7 몬테카를로 면적(복잡 영역 근사)

```python
import numpy as np

def monte_carlo_area(indicator, bbox, N=200000):
    (xmin, xmax, ymin, ymax) = bbox
    xs = np.random.uniform(xmin, xmax, size=N)
    ys = np.random.uniform(ymin, ymax, size=N)
    inside = indicator(xs, ys)
    box_area = (xmax - xmin)*(ymax - ymin)
    return box_area * (inside.mean())

# 예: 단위원 x^2 + y^2 ≤ 1 의 면적 ≈ π
indicator = lambda x,y: (x**2 + y**2) <= 1.0
A_mc = monte_carlo_area(indicator, (-1,1,-1,1), N=300000)
print("MC area (unit disk):", A_mc)
```

## 11) 응용 스냅샷

- **물리학**: 힘–변위 그래프 면적 = **일(Work)**.  
- **확률/통계**: PDF 아래 면적 = **확률**(정규분포, 누적분포).  
- **컴퓨터 그래픽/게임**: 다각형 메쉬 면적, UV 맵 면적 보존, 픽셀 카운팅.  
- **GIS/측량**: 경계 다각형(GeoJSON) 면적 → 슈레이스 + 좌표계/투영 보정.  
- **경제학**: 수요–공급 곡선 사이 면적 = 소비자/생산자 잉여.

## 12) 연습문제 (핵심 아이디어 동봉)

1) \(f(x)=\cos x\) 의 \([-\pi,\pi]\) **기하학적 면적**을 구하라.  
   *힌트*: 영점 \(\pm\pi/2\) 로 분할, 절댓값 적분.

2) \(y=x^2\) 와 \(y=2x+3\) 사이 면적.  
   *힌트*: 교점 2개 → 분할 후 \(2x+3 - x^2\) 적분.

3) **수평 슬라이스**가 쉬운 영역: \(x^2\le y\le 2x+3\).  
   *힌트*: \(y\) 범위 계산, 좌/우 \(x(y)\) 로 적분.

4) **매개폐곡선** \(x=\cos t-\tfrac13\cos3t,\ y=\sin t-\tfrac13\sin3t\) (아스트로이드 변형) 면적.  
   *힌트*: \(\tfrac12\int(xy'-yx')\,dt\).

5) **극좌표** \(r=\cos(2\theta)\) 의 한 잎(lobe) 면적.  
   *힌트*: 해당 잎의 \(\theta\) 구간을 찾아 \( \tfrac12\int r^2 d\theta \).
