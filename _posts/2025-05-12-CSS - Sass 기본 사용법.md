# Sass 기본 사용법, SCSS 문법, 컴파일러 세팅법

## 한 장 요약 (TL;DR)

- **SCSS = 확장된 CSS**: 변수(`$`), 중첩, 믹스인/함수, 맵/리스트, 제어문(조건/반복), 모듈 시스템(@use/@forward).
- **모듈 시스템**: `@import`는 **이제 비권장**. **`@use`** 로 불러오고 **네임스페이스**로 접근. **`@forward`** 로 재노출.
- **컴파일**: 브라우저는 SCSS를 모름 → **Dart Sass**로 CSS로 빌드. CLI/NPM Script/Vite/Webpack/Gulp/VSCode 등.
- **실무 패턴**: 7-1 구조, 유틸리티 자동 생성(맵/반복), 색상 스케일, fluid 타이포(clamp), 반응형 믹스인, 컨테이너 쿼리, 다크모드.
- **품질/운영**: Source map, Autoprefixer(PostCSS), Stylelint/Prettier, 생산용(minify) 분리, 캐시 전략.

---

## Sass vs SCSS 문법

| 문법 | 특징 | 확장자 |
|---|---|---|
| Sass | 들여쓰기 기반, `{}`/`;` 없음 | `.sass` |
| **SCSS** | **CSS와 거의 동일**(기존 CSS 그대로 유효) | `.scss` |

> 이 문서 전체는 **SCSS** 기준입니다.

## SCSS 핵심 문법

### 변수(Variables)

```scss
$primary: #3498db;
$spacing: 1rem;
$font-stack: system-ui, -apple-system, "Segoe UI", Roboto, "Noto Sans KR", sans-serif;

.button {
  background: $primary;
  padding: $spacing $spacing * 1.5;
  font-family: $font-stack;
}
```

- 산술 가능, 문자열/리스트/맵 등 복합 타입 지원.

### 중첩(Nesting)

```scss
nav {
  ul { margin: 0; padding: 0; list-style: none; }
  li { display: inline-block; }
  a {
    color: $primary;
    &:hover { text-decoration: underline; }
    &.active { font-weight: 700; }
  }
}
```

- **주의**: 과도한 중첩은 **특이도 폭증** → 3~4단계 이내 권장.

### 부모 참조(&)와 BEM

```scss
.card {
  &__title { font-weight: 700; }
  &__meta  { color: #6b7280; }
  &--highlight { border: 2px solid $primary; }
}
```

### 믹스인(Mixin) & 인클루드

```scss
@mixin center-inline {
  display: inline-flex;
  align-items: center;
  justify-content: center;
}

.badge {
  @include center-inline;
  padding: .25rem .5rem;
  border-radius: .5rem;
}
```

- 인자/기본값, `@content` 슬랏 지원.

```scss
@mixin mq($min) {
  @media (min-width: $min) { @content; }
}

.title {
  font-size: 1.25rem;
  @include mq(768px) { font-size: 1.5rem; }
}
```

### 함수(Function)

```scss
@function px-to-rem($px, $base: 16px) {
  @return ($px / $base) * 1rem;
}

h1 { font-size: px-to-rem(24px); } // 1.5rem
```

### 조건/반복(제어문)

```scss
$theme: light;

@if $theme == dark {
  body { background: #0b0b0c; color: #f1f1f1; }
} @else {
  body { background: #fff; color: #111; }
}

@for $i from 1 through 3 {
  .col-#{$i} { width: (100% / 3) * $i; }
}

$spacings: 0 4px 8px 12px 16px;
@each $s in $spacings {
  .m-#{$s} { margin: $s; }
}
```

### Placeholder 와 @extend

```scss
%btn-base {
  display: inline-flex; align-items: center; justify-content: center;
  border: 0; border-radius: .5rem; padding: .6rem 1rem;
}
.btn        { @extend %btn-base; background: $primary; color: #fff; }
.btn--ghost { @extend %btn-base; background: transparent; border: 1px solid $primary; color: $primary; }
```

> **주의**: `@extend`는 **선택자 병합**을 일으켜 예기치 않은 CSS 팽창/결합을 유발할 수 있음. 컴포넌트 경계가 명확할 때만 사용. 일반적으로 **믹스인**이 더 안전.

## 모듈 시스템: @use / @forward (필수)

> `@import`는 **Deprecated**. 새 프로젝트는 **`@use`/`@forward`** 사용.

### `@use` 기본

```scss
/* scss/_tokens.scss */
$brand: #2563eb;
$gap: 1rem;

@function shadow($elev: 1) {
  @return 0 6px (12px * $elev) rgba(0,0,0,.1);
}

/* scss/style.scss */
@use "tokens"; // 파일명 앞의 '_'와 뒤의 확장자 생략 가능

.button {
  background: tokens.$brand;
  box-shadow: tokens.shadow(1);
  padding: tokens.$gap tokens.$gap * 1.5;
}
```

- **네임스페이스**가 기본(충돌 방지).
- 별칭:

```scss
@use "tokens" as t;
.card { box-shadow: t.shadow(2); }
```

- 전역(import처럼) 쓰고 싶다면(권장X):

```scss
@use "tokens" as *; // 충돌 위험
```

### `@forward`로 재노출(집합 모듈)

```scss
/* scss/_colors.scss */  $brand: #2563eb;
/* scss/_spaces.scss */  $gap: 1rem; @function grid-gap($x) { @return $x * 1rem; }

/* scss/_design.scss */
@forward "colors";
@forward "spaces" show $gap, grid-gap;

/* scss/style.scss */
@use "design" as d;
.box { padding: d.grid-gap(2); background: d.$brand; }
```

- `show`/`hide`로 **공개 API** 제어.

## 맵(Map)·리스트(List)·유틸리티 자동 생성

### 맵 선언/접근/병합

```scss
$colors: (
  "primary": #2563eb,
  "danger":  #ef4444,
  "muted":   #6b7280
);

.button {
  background: map-get($colors, "primary");
}

$colors: map-merge($colors, ("success": #16a34a));
```

### 맵 반복으로 유틸리티 클래스 생성

```scss
@each $name, $value in $colors {
  .u-text-#{$name} { color: $value; }
  .u-bg-#{$name}   { background-color: $value; }
}
```

### 브레이크포인트 맵 + 믹스인

```scss
$breakpoints: (
  "sm": 640px,
  "md": 768px,
  "lg": 1024px,
  "xl": 1280px
);

@mixin up($bp) {
  $val: map-get($breakpoints, $bp);
  @if $val { @media (min-width: $val) { @content; } }
}

.grid {
  display: grid;
  grid-template-columns: 1fr;
  @include up(md) { grid-template-columns: repeat(2, 1fr); }
  @include up(lg) { grid-template-columns: repeat(3, 1fr); }
}
```

## 색상/타이포/레이아웃 고급 패턴

### 색상 함수

```scss
$brand: #2563eb;

.button {
  background: $brand;
  &:hover { background: scale-color($brand, $lightness: 8%); }
  &:active { background: darken($brand, 6%); }
  &.ghost {
    color: $brand;
    border: 1px solid mix($brand, #000, 35%); // 구 Sass
    // 또는 color.mix($brand, #000, $weight: 35%) with dart-sass:sass:color 모듈
  }
}
```

> Dart Sass 모듈(권장): `@use "sass:color";` `color.mix(...)`, `color.adjust(...)` 등.
> 구형 `lighten/darken/mix`는 단순 HSL 기반이라 과도/비선형 결과가 나올 수 있음.

### Fluid 타이포그래피(clamp)

```scss
@function fluid($min, $max, $vw-min: 360, $vw-max: 1280) {
  // 선형 보간: clamp(min, calc(...vw + ...), max)
  $slope: ($max - $min) / ($vw-max - $vw-min) * 100;
  $intercept: $min - ($slope * $vw-min / 100);
  @return clamp(#{$min}, #{$intercept}rem + #{$slope}vw, #{$max});
}

h1 { font-size: fluid(1.5rem, 3rem); }
h2 { font-size: fluid(1.25rem, 2.25rem); }
body { line-height: 1.6; }
```

> 직관 수식:
> \( f(w) = \mathrm{clamp}\big(f_{\min}, a \cdot w + b, f_{\max}\big) \) with \( a=\frac{f_{max}-f_{min}}{vw_{max}-vw_{min}} \).

### 레이아웃 토큰 + 컨테이너 쿼리

```scss
:root { --gap: clamp(.75rem, 1vw, 1.25rem); }

.card-grid {
  display: grid;
  gap: var(--gap);
  grid-template-columns: repeat(auto-fit, minmax(min(22rem, 100%), 1fr));
  container-type: inline-size;
}

@container (inline-size > 60rem) {
  .card { display: grid; grid-template-columns: 1fr auto; align-items: center; }
}
```

> SCSS는 **CSS 변수와 함께** 쓰면 테마/반응형을 **런타임** 제어로 확장 가능.

## 빌드/컴파일러 세팅

### Dart Sass (CLI)

```bash
# 전역 설치

npm i -g sass

# 단발 컴파일

sass scss/main.scss public/assets/main.css

# 감시(Watch)

sass --watch scss:public/assets

# Source Map 포함

sass scss/main.scss public/assets/main.css --source-map

# 압축(minify)

sass scss/main.scss public/assets/main.css --style=compressed
```

**NPM Scripts**:

```json
{
  "scripts": {
    "sass:dev": "sass scss:public/assets --source-map --style=expanded",
    "sass:watch": "sass --watch scss:public/assets --source-map",
    "sass:prod": "sass scss:public/assets --no-source-map --style=compressed"
  }
}
```

> Windows PowerShell에서 경로에 공백이 있으면 `"\"C:\\My Project\\scss\""` 처럼 감싸주세요.

### Gulp

```bash
npm i -D gulp sass gulp-sass postcss autoprefixer gulp-sourcemaps gulp-clean-css
```

`gulpfile.js`:

```js
const gulp = require('gulp');
const dartSass = require('sass');
const gulpSass = require('gulp-sass')(dartSass);
const sourcemaps = require('gulp-sourcemaps');
const postcss = require('gulp-postcss');
const autoprefixer = require('autoprefixer');
const cleanCSS = require('gulp-clean-css');

function styles() {
  return gulp.src('scss/main.scss')
    .pipe(sourcemaps.init())
    .pipe(gulpSass.sync({ outputStyle: 'expanded' }).on('error', gulpSass.logError))
    .pipe(postcss([autoprefixer()]))
    .pipe(sourcemaps.write('.'))
    .pipe(gulp.dest('public/assets'));
}

function stylesProd() {
  return gulp.src('scss/main.scss')
    .pipe(gulpSass.sync({ outputStyle: 'expanded' }))
    .pipe(postcss([autoprefixer()]))
    .pipe(cleanCSS())
    .pipe(gulp.dest('public/assets'));
}

exports.dev = styles;
exports.prod = stylesProd;
exports.watch = function() { gulp.watch('scss/**/*.scss', styles); };
```

## CSS 변수와 SCSS의 협업

- **SCSS 변수**: **컴파일 타임**에 치환 → **런타임 변경 불가**.
- **CSS 변수(`var(--x)`)**: **런타임 변경 가능**, JS/테마 전환/미디어쿼리와 궁합 최강.

### 하이브리드 전략

```scss
// SCSS로 기본 테마 토큰 정의
$brand: #2563eb;
$radius: 12px;

// CSS 커스텀 속성으로 노출(런타임 오버라이드 허용)
:root {
  --brand: #{$brand};
  --radius: #{$radius};
}
.button {
  background: var(--brand);
  border-radius: var(--radius);
}
```

### 미디어쿼리에서 변수만 변경

```scss
:root { --gap: 1rem; }
@media (min-width: 768px) {
  :root { --gap: 1.5rem; }
}
.wrapper { gap: var(--gap); }
```

@each $k, $v in $spaces {

## 성능·배포

- **코드 스플리팅**: 페이지별 엔트리 SCSS → 필요한 CSS만 로드.
- **미니파이**: `--style=compressed` 또는 CleanCSS/cssnano.
- **중복 제거**: `@extend` 남용 금지, 유틸리티 자동 생성은 **범위 제한**.
- **크리티컬 CSS**: above-the-fold 인라인 + 나머지 지연 로드.
- **소스맵**: 개발만 활성화, 배포시 제거.

---

## 마이그레이션: `@import` → `@use/@forward`

- `@import`는 **중복 평가/순환/전역 오염** 유발.
- 대체 규칙:
  - 선언 파일을 **모듈**로 만들고 `@forward`로 모으기.
  - 소비자는 **`@use`로 네임스페이스 접근**.
- 예시:

```scss
/* before */
@import "variables";
@import "mixins";
.button { @include flex-center; color: $brand; }

/* after */
@use "variables" as v;
@use "mixins" as m;
.button { @include m.flex-center; color: v.$brand; }
```

## React/Vue와 함께 쓰기 (요약)

- **CSS Modules**: `Component.module.scss` → 클래스 **로컬 스코프**.
- Vite/CRA/Next/Nuxt: `sass` 설치 후 곧바로 사용 가능.
- 전역 토큰 주입은 Vite `preprocessorOptions.scss.additionalData` 추천.

## 트러블슈팅

- **슬래시 나눗셈 경고**: `10px/2`는 **문맥 모호** → `@use "sass:math"; math.div(10px, 2)` 사용.
- **경로 문제**: 빌드 도구의 **Sass includePaths** 설정 또는 절대 경로 별칭 사용.
- **특이도 폭주**: 중첩 과다/태그+ID 혼합 지양, BEM/유틸리티화.
- **@extend 부작용**: 예기치 않은 선택자 병합 → 필요 최소만 사용.
- **소스맵 실패**: 로더 순서/경로 확인(`sass-loader`→`postcss-loader`→`css-loader`).

## “끝판왕” 예제: 유틸/컴포넌트/테마/반응형 한 번에

```scss
/* abstracts/_tokens.scss */
$colors: (
  "brand": #2563eb,
  "accent": #8b5cf6,
  "danger": #ef4444,
  "muted":  #6b7280
);
$spaces: (0: 0, 1: .25rem, 2: .5rem, 3: .75rem, 4: 1rem, 6: 1.5rem, 8: 2rem);
$bp: ("sm": 640px, "md": 768px, "lg": 1024px);

/* abstracts/_mixins.scss */
@use "sass:color";
@use "sass:math";

@mixin up($key) {
  @media (min-width: map-get($bp, $key)) { @content; }
}

@function fluid($min, $max, $vw-min: 360, $vw-max: 1280) {
  $slope: ($max - $min) / ($vw-max - $vw-min) * 100;
  $intercept: $min - ($slope * $vw-min / 100);
  @return clamp(#{$min}, #{$intercept}rem + #{$slope}vw, #{$max});
}

/* utilities/_spacing.scss */
@use "../abstracts/tokens" as t;
@each $k, $v in t.$spaces {
  .m-#{$k}  { margin: $v; }
  .p-#{$k}  { padding: $v; }
  .mx-#{$k} { margin-inline: $v; }
  .py-#{$k} { padding-block: $v; }
}

/* utilities/_color.scss */
@use "../abstracts/tokens" as t;
@each $n, $c in t.$colors {
  .text-#{$n} { color: $c; }
  .bg-#{$n}   { background: $c; }
  .bd-#{$n}   { border-color: $c; }
}

/* components/_button.scss */
@use "../abstracts/tokens" as t;
@use "../abstracts/mixins" as m;

.button {
  display: inline-flex; align-items: center; justify-content: center;
  padding: .6rem 1rem; border-radius: .75rem; border: 1px solid transparent;
  background: map-get(t.$colors, "brand"); color: #fff; font-weight: 600;
  transition: filter .2s ease, transform .15s ease;
  &:hover { filter: brightness(1.05); }
  &:active { transform: translateY(1px); }
  &--ghost {
    background: transparent;
    color: map-get(t.$colors, "brand");
    border-color: color.mix(map-get(t.$colors, "brand"), #000, $weight: 30%);
  }
  &--danger { background: map-get(t.$colors, "danger"); }
  &--xl { font-size: m.fluid(1rem, 1.25rem); padding: .8rem 1.25rem; }
}

/* components/_card.scss */
@use "../abstracts/mixins" as m;

.card {
  background: #fff; border-radius: .75rem; padding: 1rem;
  box-shadow: 0 10px 24px rgba(0,0,0,.08);
  h3 { margin: 0 0 .5rem; font-size: m.fluid(1.125rem, 1.5rem); }
  p  { margin: 0; color: #6b7280; }
  @include m.up(md) { padding: 1.25rem; }
}

/* themes/_dark.scss */
:root[data-theme="dark"] {
  --bg: #0c0d10; --fg: #eaeaea; --surface: #14161a;
}
body {
  background: var(--bg, #fff);
  color: var(--fg, #111);
}
.card { background: var(--surface, #fff); }

/* main.scss */
@use "abstracts/tokens";
@use "abstracts/mixins";
@use "utilities/spacing";
@use "utilities/color";
@use "components/button";
@use "components/card";
```

## 부록 A) 수학 섹션(간단)

- **Fluid 보간**: \( f(w) = a\cdot w + b \), \( a=\frac{f_{max}-f_{min}}{vw_{max}-vw_{min}} \), \( b=f_{min}-a\cdot vw_{min} \).
- CSS에서 `clamp(min, ideal, max)`로 구현하면 미디어쿼리 없이 **유동 타이포/간격** 가능.
