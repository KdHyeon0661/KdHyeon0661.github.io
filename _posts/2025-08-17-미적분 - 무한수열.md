# 무한수열(Infinite Sequences)

## 1. 수열의 본질과 표기

수열은 자연수에서 실수(또는 복소수)로 가는 함수다.
$$
a:\mathbb{N}\to\mathbb{R},\quad n\mapsto a_n
$$
필요 시 부분수열(subsequence)을 사용한다. 지수열 \(\{a_{n_k}\}\)에서
$$
1\le n_1<n_2<\cdots,\qquad b_k:=a_{n_k}
$$
로 정의하면 \(\{b_k\}\)는 \(\{a_n\}\)의 부분수열이다.

---

## 2. 수렴과 발산: 정의·필요조건·유일성

수열이 \(L\)에 수렴한다는 것은 다음을 의미한다.
$$
\forall\,\varepsilon>0,\ \exists\,N\in\mathbb{N}:\ n>N\Rightarrow |a_n-L|<\varepsilon
$$
기본 성질:
- 수렴 수열의 극한은 유일하다.
- 수열이 수렴하면 유계이다.
- 수렴은 연산과 궁합이 좋다(합·차·곱·몫(분모 극한 \(\neq0\))·연속함수 통과).

필요조건(테스트 0):
$$
\text{만약 } \sum_{n} a_n \text{이 아니라 } \{a_n\} \text{가 수렴한다면 } \lim_{n\to\infty} a_n \text{이 존재한다.}
$$
특히, 급수 수렴의 필요조건으로 \(\lim a_n=0\)이 자주 쓰이지만, 여기서는 수열 자체의 극한을 직접 다룬다.

---

## 3. 단조수열 수렴정리·유계성

단조수열 수렴정리:
- \(\{a_n\}\)이 단조증가이고 상계가 있으면 수렴한다.
- \(\{a_n\}\)이 단조감소이고 하계가 있으면 수렴한다.

유계·단조 점검은 실무에서 가장 빠른 1차 체크다.

---

## 4. 끼인값 정리(샌드위치, Squeeze)

수열 \(\{a_n\},\{b_n\},\{c_n\}\)에 대하여
$$
a_n\le b_n\le c_n\quad\text{및}\quad \lim a_n=\lim c_n=L
$$
이면
$$
\lim b_n=L
$$
이다. 대표 예: \(\lim_{n\to\infty} n\sin\frac{1}{n}=1\cdot 0=0\) 유도에 쓰인다.

---

## 5. 대표적 극한과 직관

- \(\displaystyle \lim_{n\to\infty}\frac{1}{n}=0\)  
- \(\displaystyle \lim_{n\to\infty}\sqrt[n]{n}=1\)  
- \(\displaystyle \lim_{n\to\infty}\left(1+\frac{1}{n}\right)^{n}=e\)  
- \(\displaystyle \lim_{n\to\infty}\left(1+\frac{x}{n}\right)^{n}=e^{x}\)  
- \(\displaystyle \lim_{n\to\infty}\frac{\ln n}{n}=0\), \(\displaystyle \lim_{n\to\infty}\frac{n^\alpha}{c^n}=0\) \((c>1,\ \alpha>0)\)  
- 진동 예: \(\{\sin n\}\)은 수렴하지 않는다(치밀한 값들로 진동).  
- 교대 감쇠 예: \(\displaystyle \frac{(-1)^n}{n}\to 0\).

---

## 6. 수열 수렴 판정 도구(자주 쓰는 것)

### 6.1 비교(상·하한 비교)
두 수열 \(0\le a_n\le b_n\)이고 \(\{b_n\}\to 0\)이면 \(\{a_n\}\to 0\).  
또는 크기 비교로 유계/단조를 만든다.

### 6.2 비율·근 형태의 추정
수열에 직접 비율/근을 쓰는 일은 급수만큼 체계적이진 않지만, 다음 직관은 유효하다.
- \(n!\) vs \(c^n\): \(\sqrt[n]{n!}\sim \frac{n}{e}\)이므로 다항·로그보다 팩토리얼이 압도적.
- \(n^\alpha\) vs \(c^n\) (\(c>1\)): 지수형이 다항형을 이긴다.
- \(\sqrt[n]{|a_n|}\to L<1\)이면 \(a_n\to 0\).

### 6.3 스트촐츠–체사로(Stolz–Cesàro)
\(\{b_n\}\)이 증가하고 \(b_n\to\infty\)일 때,
$$
\lim_{n\to\infty}\frac{a_n}{b_n}=L\quad\text{(존재)}\ \Longleftrightarrow\ \lim_{n\to\infty}\frac{a_{n+1}-a_n}{b_{n+1}-b_n}=L
$$
대표 응용: \(\displaystyle \lim_{n\to\infty}\frac{\ln n}{n}=0\)에서
$$
\frac{a_{n+1}-a_n}{b_{n+1}-b_n}=\frac{\ln(n+1)-\ln n}{(n+1)-n}=\ln\!\left(1+\frac{1}{n}\right)\to 0
$$

---

## 7. 코시(Cauchy) 수열과 완비성

코시 수열 정의:
$$
\forall\,\varepsilon>0,\ \exists\,N:\ m,n>N\Rightarrow |a_m-a_n|<\varepsilon
$$
\(\mathbb{R}\)의 완비성 때문에
$$
\{a_n\}\text{ 수렴}\ \Longleftrightarrow\ \{a_n\}\text{ 코시}
$$
이다(실수의 근본 성질). 수치 계산에선 코시 조건이 실질적인 종료 조건으로 쓰인다.

---

## 8. 상극한/하극한 — limsup, liminf

정의(꼬리 상한/하한의 극한):
$$
\limsup_{n\to\infty} a_n=\lim_{n\to\infty}\Big(\sup_{k\ge n} a_k\Big),\qquad
\liminf_{n\to\infty} a_n=\lim_{n\to\infty}\Big(\inf_{k\ge n} a_k\Big)
$$
성질:
- 항상 \(\displaystyle \liminf a_n \le \limsup a_n\).
- \(\displaystyle \lim a_n\)이 존재하면 둘이 같고 그 값이 극한이다.
- \(\{a_n\}\)가 유계면 부분수열의 극한들 집합의 상/하한과 일치한다.
- 예: \(a_n=(-1)^n\Rightarrow \liminf a_n=-1,\ \limsup a_n=1\) (수렴하지 않음).

---

## 9. 부분수열과 볼차노–바이어슈트라스(BW)

유계 수열은 항상 수렴하는 부분수열을 가진다(BW 정리).  
활용: 수렴성 증명에 실패하면 부분수열의 거동을 분석해 모순을 이끌어내거나 진동을 보인다.

---

## 10. 점화식(재귀) 수열: 고정점·수렴성·수치예

### 10.1 고정점 반복
함수 \(g\)에 대하여
$$
a_{n+1}=g(a_n)
$$
고정점 \(x^\*\)는 \(g(x^\*)=x^\*\).  
수렴 충분조건(근방 수축):
$$
|g'(x^\*)|<1 \ \text{이고 초기값이 충분히 가까우면 } a_n\to x^\*
$$

### 10.2 헤론(바빌로니아) 방법 — \(\sqrt{\alpha}\)
$$
a_{n+1}=\frac{1}{2}\left(a_n+\frac{\alpha}{a_n}\right),\quad \alpha>0,\ a_1>0
$$
고정점 방정식 \(x=\frac{1}{2}(x+\alpha/x)\Rightarrow x^2=\alpha\Rightarrow x=\sqrt{\alpha}\).  
\(\alpha=2\)일 때 단조감소·하계 \(\sqrt2\)·수렴 증명이 전형적이다.

### 10.3 뉴턴 방법(함수근)
$$
a_{n+1}=a_n-\frac{f(a_n)}{f'(a_n)}
$$
적절한 초기값과 정칙성 하에 2차 수렴. 예: \(f(x)=x^2-2\)면 헤론과 동일 형태.

## 11. 실전 예제 풀이(요청 항목 포함)

### 11.1 수렴 여부 판단
(1) \(a_n=\dfrac{n+1}{2n}\)  
분자/분모 최고차 비교로
$$
\lim_{n\to\infty} a_n=\frac{1}{2}
$$

(2) \(a_n=(-1)^n\cdot\dfrac{1}{n}\)  
절댓값은 \(1/n\to 0\), 부호는 교대지만 감쇠 → 
$$
\lim_{n\to\infty} a_n=0
$$
(진동하되 0으로 수렴)

(3) \(a_n=\sin n\)  
\(\{n\ \mathrm{mod}\ 2\pi\}\)가 \([0,2\pi)\)에 조밀 → \(\{\sin n\}\)은 서로 다른 점들 근처를 무한히 방문하므로 극한 불존재(발산).

### 11.2 재귀 수열 극한
\(a_1=1,\ a_{n+1}=\dfrac{1}{2}\Big(a_n+\dfrac{2}{a_n}\Big)\)  
헤론 방법으로 \(\sqrt{2}\)를 수렴 타겟으로 갖는다.

정당화 개요:
- \(a_1=1<\sqrt2\)이지만 \(a_2=\frac{1}{2}(1+2)\)=\(1.5>\sqrt2\).
- \(a_n>\sqrt2\Rightarrow a_{n+1}-\sqrt2=\dfrac{(a_n-\sqrt2)^2}{2a_n}>0\)이므로 \(\{a_n\}\)은 \(\sqrt2\)보다 크고 감소.
- 하계 \(\sqrt2\)로 단조·유계 → 수렴. 극한 \(L\)에 대해 \(L=\dfrac{1}{2}(L+2/L)\Rightarrow L^2=2\Rightarrow L=\sqrt2\).

## 12. 추가 예제(상황별)

1) \(a_n=\sqrt[n]{n}\to 1\)  
로그 취해 \(\ln a_n=\dfrac{\ln n}{n}\to 0\Rightarrow a_n\to e^0=1\).

2) \(a_n=\dfrac{n^\alpha}{c^n}\to 0\) (\(c>1\))  
스트촐츠·비교로 가능. 직관적으로 지수 성장은 다항을 이긴다.

3) \(a_n=\ln(1+\tfrac{1}{n})\to 0\)  
테일러 또는 \(\ln(1+x)\le x\) 사용.

4) \(a_n=\dfrac{\sin n}{n}\to 0\)  
\(|\sin n|\le 1\)과 비교.

5) \(a_n=\left(1+\dfrac{x}{n}\right)^{n}\to e^{x}\)  
표준 극한.

## 13. 수치 검증용 파이썬 코드

### 13.1 기본 유틸(부분수열, 단조·유계 체크, limsup/liminf 근사)
```python
import math

def seq_limit_preview(seq, N=20):
    return [seq(n) for n in range(1, N+1)]

def is_monotone_nonincreasing(vals, tol=0.0):
    return all(vals[i+1] <= vals[i] + tol for i in range(len(vals)-1))

def is_bounded(vals, low=-math.inf, high=math.inf):
    return all(low <= v <= high for v in vals)

def approx_limsup_liminf(seq, N=10000, tail=500):
    # 마지막 tail개에서 상/하극한 근사
    vals = [seq(n) for n in range(1, N+1)]
    tail_vals = vals[-tail:]
    return max(tail_vals), min(tail_vals)
```

### 13.2 요청 예제의 수치 확인
```python
# (1) a_n = (n+1)/(2n) -> 1/2
a = lambda n: (n+1)/(2*n)
vals = seq_limit_preview(a, N=10)
print("a_n first 10:", vals)
print("monotone nonincreasing? ", is_monotone_nonincreasing(vals))
print("approx tail (limsup, liminf):", approx_limsup_liminf(a))

# (2) a_n = (-1)^n / n -> 0
b = lambda n: ((-1)**n)/n
vals = seq_limit_preview(b, N=10)
print("b_n first 10:", vals)
print("bounded? ", is_bounded(vals, -1, 1))
print("approx tail (limsup, liminf):", approx_limsup_liminf(b))

# (3) a_n = sin(n) -> no limit (oscillatory)
c = lambda n: math.sin(n)
vals = seq_limit_preview(c, N=20)
print("c_n first 20:", [round(v, 6) for v in vals])
print("approx tail (limsup, liminf):", approx_limsup_liminf(c))
```

### 13.3 헤론 방법으로 \(\sqrt{2}\) 근사
```python
def heron_sqrt(alpha=2.0, a1=1.0, steps=10):
    a = a1
    seq = [a]
    for _ in range(steps):
        a = 0.5*(a + alpha/a)
        seq.append(a)
    return seq

hs = heron_sqrt(alpha=2.0, a1=1.0, steps=8)
print("Heron sequence for sqrt(2):")
for i, v in enumerate(hs, start=0):
    print(f"n={i:2d}, a_n={v:.12f}, error={abs(v - math.sqrt(2)):.3e}")
```

| 항목 | 내용 |

## 15. 추가 연습문제(해설 아이디어 포함)

1) \(a_n=\sqrt[n]{n!}/n\)의 극한: 스털링 근사 \(n!\sim \sqrt{2\pi n}(n/e)^n\)로 \(a_n\to 1/e\).  
2) \(a_n=\dfrac{n}{\ln n}\)의 증가성(충분히 큰 \(n\)에서 증가)과 \(\dfrac{\ln n}{n}\to 0\).  
3) \(a_n=\dfrac{1}{n}\sum_{k=1}^n \ln\left(1+\dfrac{k}{n}\right)\) 극한: 리만합으로 \(\displaystyle \int_0^1 \ln(1+x)\,dx=2\ln 2-1\).  
4) \(a_{n+1}=\cos a_n,\ a_1\in[0,1]\) 수렴성: 연속·콤팩트·단조/수축성 분석 후 고정점 방정식 \(x=\cos x\)로 귀결.

## 16. 마무리 체크리스트

- 유계·단조 여부를 가장 먼저 본다.  
- 끼인값·비교·스트촐츠–체사로를 상황에 맞춰 쓴다.  
- 진동은 limsup/liminf로 정리하며 부분수열 극한을 본다.  
- 점화식은 고정점 방정식과 수축 여부 확인, 단조/유계 유도.  
- 수치 검증 코드는 코시형 종료 기준과 오차 로그 출력까지 준비한다.
