# 멱급수의 수렴

## 1. 멱급수의 정의와 표기 재정리

- **멱급수(power series)**:
$$
\sum_{n=0}^{\infty} a_n\, (x-c)^n
$$

- \(a_n\): 계수, \(c\): 중심(전개점), \(x\): 변수.  
- 중심이 0이면 **맥클로린형**: $$\displaystyle \sum_{n=0}^{\infty} a_n x^n$$.

---

## 2. 수렴반경 \(R\) — 코시–아다마르 공식과 해석

### 2.1 코시–아다마르(Cauchy–Hadamard) 공식
- 상극한(루트 판정의 정밀 버전):
$$
\frac{1}{R}
=\limsup_{n\to\infty} \sqrt[n]{|a_n|}
\quad(\text{즉 } R=\big(\limsup \sqrt[n]{|a_n|}\big)^{-1})
$$
- 존재 시 **비판정법의 역수**도 사용 가능:
$$
R=\lim_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right| \quad(\text{극한이 존재할 때})
$$

### 2.2 의미
- \(|x-c|<R\): 절대수렴, \(|x-c|>R\): 발산, \(|x-c|=R\): 별도 판정 필요.
- \(\limsup\)이 등장하는 이유: 계수의 미시적 진동에도 **최대 성장률의 상한**으로 반경이 안정적으로 정의되도록 하기 위해서다.

---

## 3. 수렴구간과 경계점 분석 로드맵

### 3.1 내부
- 모든 \(r<R\)에 대해 \(|x-c|\le r\)에서 **절대·균등수렴**.  
- 결과: 항별 연산(미분/적분/합) 정당화, 합 함수의 해석성(analyticity) 보장.

### 3.2 경계 \(|x-c|=R\)
- 멱급수는 **경계에서 각 점을 개별 판정**해야 한다.
- 도구: 교대급수 판정(Leibniz), 비교/극한비교, 적분판정, (때로) 아벨(Abel)류 정리.

---

## 4. 항별 미분/적분과 분석적 성질

### 4.1 항별 미분/적분
- \(|x-c|<R\)에서
$$
\frac{d}{dx}\left(\sum_{n=0}^{\infty} a_n(x-c)^n\right)
=\sum_{n=1}^{\infty} n a_n (x-c)^{n-1},
$$
$$
\int \sum_{n=0}^{\infty} a_n(x-c)^n\,dx
=\sum_{n=0}^{\infty} \frac{a_n}{n+1}(x-c)^{n+1}+C.
$$
- 새 급수의 수렴반경은 **같거나 더 크다**(동일 \(R\) 유지가 표준적 결과).

### 4.2 해석함수
- \(|x-c|<R\)에서 합 \(S(x)=\sum a_n(x-c)^n\)는 무한번 미분 가능하고 각 미분을 항별로 교환 가능:
$$
a_n=\frac{S^{(n)}(c)}{n!}.
$$

---

## 5. Abel 류 정리와 경계 접근

### 5.1 Abel의 정리(전형적 형태)
- \(\sum_{n=0}^{\infty} a_n\)가 수렴하면, \(\displaystyle S(x)=\sum_{n=0}^{\infty} a_n x^n\)는 \(x\in[0,1)\)에서 수렴하며
$$
\lim_{x\to 1^-} S(x)=\sum_{n=0}^{\infty} a_n.
$$
- 해석: **경계점의 수렴이 주어지면 안쪽에서 경계로의 연속적 접근이 가능**.

### 5.2 응용 포인트
- \(\sum a_n R^n\)의 수렴/발산이 판정되면, \(x\to R^-\) 방향에서의 극한 거동을 제어할 수 있다.

---

## 6. 이론적 배경과의 연결 고도화

### 6.1 실수의 완비성 공리
- “위로 유계인 부분집합은 최소상계 \(\sup\)를 갖는다.”  
- 효과:
  - \(\limsup\) 존재(실수 체계에서 상극한은 항상 정의).
  - 코시 수열은 실수 내에서 **필연적으로 수렴**(한계가 존재).

### 6.2 코시 수열 정리
- 급수의 부분합이나 멱급수의 부분합은 **수렴 ⇔ 코시**.  
- \(|x-c|<R\)에서 절대수렴은 코시성을 즉각 보장.

### 6.3 볼차노–바이어슈트라스 정리
- 유계 수열은 수렴 부분수열을 가진다.  
- \(\limsup\)·\(\liminf\) 정의와 계수 성장률의 상한/하한 분석에 필수.

---

## 7. 실전 예제 — 수렴반경/경계 판정 패턴

### 예제 1: 계승과 지수
1) $$\displaystyle \sum_{n\ge0}\frac{x^n}{n!}$$  
- 비판정:
$$
\left|\frac{a_{n+1}}{a_n}\right|=\frac{1}{n+1}\to 0\quad\Rightarrow\quad R=\infty.
$$
- 전 구간에서 수렴(해는 \(e^x\)).

2) $$\displaystyle \sum_{n\ge0} n! \, x^n$$  
- 비판정:
$$
\left|\frac{a_{n+1}}{a_n}\right|=n+1\to\infty\quad\Rightarrow\quad R=0.
$$
- \(x=0\)에서만 수렴.

### 예제 2: 다항·지수 혼합
$$
\sum_{n\ge0}\frac{2^n}{n}\,(x-1)^n\quad(n\ge1\ \text{부터})
$$
- \(a_n=\frac{2^n}{n}\). 비판정:
$$
\left|\frac{a_{n+1}}{a_n}\right|=\frac{2^{n+1}/(n+1)}{2^n/n}=\frac{2n}{n+1}\to 2
\Rightarrow R=\frac{1}{2}.
$$
- 경계 \(|x-1|=1/2\)에서 별도 판정:  
  - \(x=1+\frac{1}{2}\): 항 \(\sim\frac{2^n}{n}(\frac{1}{2})^n=\frac{1}{n}\) ⇒ 조화급수 발산.  
  - \(x=1-\frac{1}{2}\): 항 \(\sim\frac{2^n}{n}(-\frac{1}{2})^n=\frac{(-1)^n}{n}\) ⇒ 교대조화급수 **조건수렴**.

### 예제 3: 조합 계수
$$
\sum_{n\ge0}\binom{2n}{n}\,x^n.
$$
- 아심프: \(\displaystyle \binom{2n}{n}\sim \frac{4^n}{\sqrt{\pi n}}\). 루트판정:
$$
\sqrt[n]{a_n}\sim 4 \quad\Rightarrow\quad R=\frac{1}{4}.
$$
- 경계 \(x=\pm\frac{1}{4}\):  
  - \(x=1/4\): 항 \(\sim \frac{1}{\sqrt{\pi n}}\) ⇒ p-급수 \(p=\tfrac12<1\) 발산.  
  - \(x=-1/4\): 교대형 \(\sim \frac{(-1)^n}{\sqrt{\pi n}}\) ⇒ 교대판정으로 **수렴(조건)**.

### 예제 4: 로그/아크탄
1) $$-\ln(1-x)=\sum_{n\ge1}\frac{x^n}{n},\ |x|<1$$  
- \(R=1\); \(x=1\) 발산(조화급수), \(x=-1\) 수렴(\(-\ln 2\)).

2) $$\arctan x=\sum_{n\ge0}\frac{(-1)^n}{2n+1}\,x^{2n+1},\ |x|<1$$  
- \(R=1\); \(x=\pm1\)에서 각각 조건수렴(\(\arctan 1=\pi/4\)).

### 예제 5: 중심 이동
$$
\sum_{n\ge0}\frac{(x-3)^n}{3^n\sqrt{n+1}}
$$
- 루트판정: \(\sqrt[n]{|a_n|}\approx \frac{|x-3|}{3}\) ⇒ \(R=3\).  
- 경계 \(|x-3|=3\): 항 \(\sim \frac{1}{\sqrt{n}}\) 또는 교대 \(\frac{(-1)^n}{\sqrt{n}}\).  
  - 전자는 발산, 후자는 교대판정으로 **수렴(조건)** 가능. 점별 판정 필요.

## 8. 균등수렴과 항별 연산의 정당화

### 8.1 Weierstrass M-test로 본 균등수렴
- \(|x-c|\le r<R\)라면
$$
|a_n(x-c)^n|\le |a_n|\,r^n,
$$
그리고 \(\sum |a_n|r^n\) 수렴 ⇒ 원래 급수는 \([c-r,c+r]\)에서 **절대·균등수렴**.  
- 결과: **항별 연산(미분/적분/극한 교환) 안전**.

### 8.2 실전 체크
- 수렴반경 내부의 **컴팩트 구간**(경계 제외)에서만 균등수렴을 주장하라.  
- 경계점에선 균등수렴 실패가 흔하다(항별 교환 금지).

## 9. 자주 나오는 질문/함정

1) **비판정 \(L=1\)**  
   판정불가. 다른 판정(교대, 비교/극한비교, 적분)을 동원하라.  
2) **경계점 일괄취급 금지**  
   \(|x-c|=R\)에서 점별로 다르다. 예제 2,3 참조.  
3) **항별 미분의 반경**  
   미분/적분 후에도 같은 \(R\)이나, 경계값의 수렴성은 이전과 달라질 수 있다.  
4) **조건수렴과 재배열**  
   반경 내부는 절대수렴이므로 재배열 안전. 경계에서 **조건수렴**이면 재배열 위험.  
5) **\(\limsup\)과 직감**  
   \(\sqrt[n]{|a_n|}\)이 진동해도 **상극한**이 반경을 안정적으로 규정한다.

## 10. 스니펫 — SymPy로 반경/경계 자동 점검

> 아이디어: 주어진 \(a_n\) 패턴으로 \(R\)을 추정(비판정·루트판정), 경계 후보에서 표준 판정 실행.

```python
import sympy as sp

n = sp.symbols('n', positive=True, integer=True)
x, c = sp.symbols('x c', real=True)

# 예: a_n = 2**n / n  (예제 2)
a_n = 2**n / n
# 중심 c=1, 변수 x
R_ratio = sp.limit_abs(sp.simplify(a_n / sp.simplify(a_n.subs(n, n+1))), n, sp.oo)
R_root  = 1 / sp.limit_sup(sp.root(sp.Abs(a_n), n), n, sp.oo)
R_ratio, R_root  # 각각 1/2, 1/2

# 경계점 검사: x = c ± R
R = sp.Rational(1, 2)
x1 = 1 + R
x2 = 1 - R

# 경계에서의 일반항
term_x1 = a_n * (x1-1)**n
term_x2 = a_n * (x2-1)**n

# 수렴성 판정용: 비교/교대/적분 테스트는 케이스 바이에 케이스로 해석적 판단을 병행
sp.simplify(term_x1), sp.simplify(term_x2)  # 1/n, (-1)**n / n
```

## 11. 스니펫 — 수치 부분합으로 경계 거동 관측

```python
import numpy as np

def partial_sums_series(an, x, c=0.0, N=200000):
    # an: callable a(n), n >= 0
    n = np.arange(1, N+1, dtype=np.float64)
    terms = an(n) * (x - c)**n
    return np.cumsum(terms)

# 예제 2: a_n = 2^n / n, c=1
def a(n): return (2.0**n)/n

for x in [1.0+0.5, 1.0-0.5, 1.0, 0.9]:
    s = partial_sums_series(a, x, c=1.0, N=200000)
    print(x, s[-1])
# 1.5: 발산 경향, 0.5: 교대조화형 수렴 경향, 1.0: 발산, 0.9: 내부 수렴
```

## 12. 더 나아가기: 고급 주제 단서

- **무한반경(\(R=\infty\))**: 지수형/계승 분모형이 흔함(\(x^n/n!\), 베셀/지수형 특수함수).  
- **라쿤 시리즈(간극 멱급수, lacunary series)**: \(n\)이 희소(예: \(n=2^k\)). 반경은 동일하나 경계 거동이 비전형적.  
- **해석적 연장(analytic continuation)**: 반경 밖에서 함수를 확장하는 기술. 반경은 **특이점까지의 거리**와 직결.

## 13. 요약 표

| 항목 | 핵심식/포인트 | 실전 코멘트 |
|---|---|---|
| 수렴반경 \(R\) | $$\displaystyle \frac{1}{R}=\limsup_{n\to\infty}\sqrt[n]{|a_n|}$$ | 비판정 극한이 흔하면 \(R=\lim|a_n/a_{n+1}|\) |
| 내부 \(|x-c|<R\) | 절대·균등수렴 | 항별 미·적분, 합의 해석성 보장 |
| 경계 \(|x-c|=R\) | 점별 판정 | 교대/비교/적분/Abel 테스트 |
| Abel 정리 | $$\lim_{x\to 1^-}\sum a_n x^n=\sum a_n$$ (수렴 가정下) | 경계 접근 연속성 확보 |
| 균등수렴 | M-test로 컴팩트에서 보장 | 연산 교환의 안전장치 |
| 이론 배경 | 완비성·코시·B–W | \(\limsup\) 정의, 수렴 보장 기반 |

## 14. 연습문제(직접 판정 루틴)

1) $$\displaystyle \sum_{n\ge0}\frac{(3x)^n}{\sqrt{n+1}}$$. \(R\)과 경계 \(|x|=1/3\) 판정.  
2) $$\displaystyle \sum_{n\ge1}\frac{(-1)^n}{n}(x-2)^n$$. \(R\)과 \(x=1,3\)에서의 거동.  
3) $$\displaystyle \sum_{n\ge0}\binom{3n}{n}x^n$$. 스털링 근사로 \(R\) 추정, 경계 판정.  
4) $$\displaystyle \sum_{n\ge0}\frac{n!}{(2n)!}(x-1)^n$$. 비판정 vs 루트판정으로 \(R\) 비교 검산.

## 15. 마무리

- 기존 정리(정의/수렴반경/근·비판정법/완비성·코시·B–W 연계)를 **코시–아다마르, 균등수렴·항별연산, Abel 정리, 경계판정 패턴**까지 확장했다.  
- 실전에서는 \(R\)을 먼저 잡고, **경계는 각 점별로** 교대·비교·적분판정 등으로 정교하게 본다.  
- 내부 컴팩트 구간의 균등수렴을 항상 염두에 두어 **미·적분/극한 교환을 안전하게** 수행하라.
