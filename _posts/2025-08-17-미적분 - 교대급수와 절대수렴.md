# 교대급수와 절대수렴

---

## 1. 정의

**교대급수**란 항의 부호가 번갈아 바뀌는 급수로, 표준형은 다음과 같다.
$$
\sum_{n=1}^{\infty}(-1)^{n+1}a_n
\quad \text{또는}\quad
\sum_{n=1}^{\infty}(-1)^{n}a_n,
\qquad a_n>0.
$$
여기서 \(a_n\)은 보통 “크기”만을 담당하고, 부호는 \((-1)^n\)이 번갈아 부여한다.

---

## 2. 라이프니츠(Leibniz) 교대급수 판정법

다음 세 조건을 만족하면 교대급수는 수렴한다(충분조건).

1) \(a_n>0\)  
2) 단조감소: \(a_{n+1}\le a_n\) (혹은 **결국(eventually) 단조감소**이면 충분)  
3) 극한 영: \(\displaystyle \lim_{n\to\infty}a_n=0\)

이때 **부분합의 오차 추정**(Alternating Series Estimation)도 자동으로 주어진다(§3).

### 간결 증명 스케치
부분합 \(S_N=\sum_{n=1}^N(-1)^{n+1}a_n\)을 짝수/홀수로 나누면
- 짝수 부분합 \((S_{2k})\)는 증가 수열,
- 홀수 부분합 \((S_{2k+1})\)는 감소 수열,
- 그리고 \(S_{2k}\le S_{2k+2}\le S_{2k+1}\le S_{2k-1}\).
서로 끼워맞춤(샌드위치)으로 공통 극한에 수렴한다.

> 주의: 단조감소는 **충분조건**이지 **필요조건**은 아니다. 단조감소가 아니어도 다른 방법(Dirichlet/Abel 등)으로 수렴을 증명할 수 있다. 그러나 실전 판정에선 “(결국) 단조감소 + 극한 0” 체크가 가장 빠르다.

---

## 3. 오차 추정(Alternating Series Estimation Theorem)

라이프니츠 조건 하에서, 급수의 합을 \(S\), 부분합을 \(S_N\)이라 할 때
$$
|S-S_N|\le a_{N+1}\quad\text{그리고}\quad S_N\ \text{은 }S\ \text{을 번갈아 가며 과대/과소로 끼운다}.
$$
즉, **다음 항의 크기**가 곧 절대 오차의 상계이다.  
실무적으로 “목표 오차 \(\varepsilon\)”를 주면, **\(a_{N+1}\le\varepsilon\)**가 되는 \(N\)을 고르면 된다.

### 예
교대 조화급수
$$
\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}=\ln 2
$$
에서 \(|S-S_N|\le 1/(N+1)\).  
\(\varepsilon=10^{-6}\)이면 \(N\ge 10^6-1\)을 택하면 된다.

## 4. 절대수렴과 조건수렴

- **절대수렴**: \(\displaystyle \sum |a_n|\)이 수렴하면 \(\sum a_n\)도 반드시 수렴.  
  (절대수렴 ⇒ 무조건 수렴, 재배열해도 합이 변하지 않음.)
- **조건수렴**: \(\sum a_n\)은 수렴하지만 \(\sum|a_n|\)은 발산.

### 대표 예
- \(\displaystyle \sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}\) 은 수렴(라이프니츠)하지만 \(\sum 1/n\) 발산 ⇒ **조건수렴**.
- \(\displaystyle \sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n^2}\) 은 \(\sum 1/n^2\) 수렴 ⇒ **절대수렴**.

---

## 5. 리만 재배열 정리(조건수렴의 위험성)

\(\sum a_n\)이 **조건수렴**일 때, 항들의 **순서를 재배열**하면 합을 **임의의 실수**로 만들거나 **\(\pm\infty\)**로 발산시킬 수 있다.  
따라서 조건수렴 급수는 **순서에 민감**하다(비교: 절대수렴은 순서에 둔감).

실전 요령:
- 먼저 \(\sum|a_n|\)에 대해 비교/비율/근/적분 판정으로 절대수렴을 확인한다.
- 절대수렴이 아니면 교대·진동 판정(라이프니츠, Dirichlet, Abel 등)으로 “조건수렴”을 점검한다.
- 조건수렴이면 **오차 추정과 함께 순서 의존성**까지 설명한다.

---

## 6. 더 일반적인 수렴 판정(단조감소가 아닐 때)

### Dirichlet 판정
\(\sum a_n b_n\)에서 부분합 \(A_N=\sum_{n=1}^N a_n\)이 **유계**이고, \(b_n\downarrow 0\)이면 수렴.  
교대급수는 \(a_n=(-1)^n\), \(b_n\downarrow 0\)로 해석 가능(부분합 유계: \(|\sum_{k=1}^N(-1)^k|\le 1\)).

### Abel 판정
\(\sum a_n b_n\)에서 \(\sum a_n\)이 수렴, \(b_n\)이 단조이고 유계이면 수렴.

> 활용: \(a_n=\sin n\), \(b_n=1/n\) 같은 **진동×감쇠** 조합, 또는 \(a_n=(-1)^n\), \(b_n=\frac{\log n}{n}\)(단조감소는 결국 성립) 등.

---

## 7. 대표 예제와 해설

### 7.1 교대 조화급수(조건수렴, 합은 \(\ln 2\))
$$
\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}=\ln 2.
$$
라이프니츠 조건 성립(단조감소, 극한 0). 오차 \(|S-S_N|\le 1/(N+1)\).

### 7.2 \(\displaystyle \sum_{n=2}^\infty \frac{(-1)^{n}}{n(\ln n)^p}\)
- \(p>0\): \(a_n=\frac{1}{n(\ln n)^p}\downarrow 0\) (충분히 큰 \(n\)에서 단조감소) ⇒ **교대급수 수렴**.
- 절대수렴은 \(p>1\)에서만 성립(로그 경계 테스트).  
  ⇒ \(0<p\le 1\)은 **조건수렴**, \(p>1\)은 **절대수렴**.

### 7.3 \(\displaystyle \sum_{n=1}^{\infty}\frac{(-1)^{n+1}\log n}{n}\)
\(\frac{\log n}{n}\)은 \(n\ge 3\)에서 단조감소, 극한 0 ⇒ **교대급수 수렴**.  
하지만 \(\sum \frac{\log n}{n}\)은 발산 ⇒ **조건수렴**.

### 7.4 절대수렴 예
\(\displaystyle \sum_{n=1}^{\infty}\frac{(-1)^{n}}{n^2}\)은 \(\sum 1/n^2\) 수렴 ⇒ **절대수렴**.  
재배열해도 합 불변, 수치 안정성 좋음.

---

## 8. 테일러 급수와 교대급수(경계점에서의 행동)

### \(\log(1+x)\) (맥클로린)
$$
\log(1+x)=\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}\,x^n,\qquad |x|<1,\ x\neq -1.
$$
경계 \(x=1\)에서 \(\sum(-1)^{n+1}/n=\log 2\)로 **조건수렴**,  
\(x=-1\)에선 발산. 수렴 반경의 경계점은 **교대·조건수렴**이 자주 등장한다.

### \(\arctan x\)
$$
\arctan x=\sum_{n=0}^{\infty}\frac{(-1)^n}{2n+1}x^{2n+1},\qquad |x|\le 1,\ x\neq \pm 1.
$$
\(x=1\)에서 \(\sum(-1)^n/(2n+1)=\pi/4\) (조건수렴).  
이런 경계 점검은 실전에서 매우 중요하다.

---

## 9. 실무 레시피(문제 만나면 이렇게)

1) \(\sum|a_n|\)에 대해 **비교/비율/근/적분 판정**으로 절대수렴을 먼저 본다.  
2) 아니면 \((-1)^n a_n\) 꼴로 보고 **라이프니츠**(결국 단조감소+극한 0) 적용.  
3) 단조감소가 애매하면 **Dirichlet/Abel**로 진동×감쇠 조합을 처리.  
4) 수치 근삿값이 필요하면 **오차 \(\le a_{N+1}\)** 규칙으로 필요한 \(N\)을 바로 산출.  
5) 조건수렴이면 **재배열 위험(리만 정리)**을 반드시 주석으로 명시.

## 10. 코드 예제

### 10.1 오차 목표에 맞춘 교대급수 근사(일반 루틴)
다음은 “\(a_n>0\), 단조감소, \(a_n\to 0\)”라는 전형적 상황에 맞춘 근사 루틴이다.  
오차가 \(\varepsilon\) 이하가 되면 정지한다(이론상 \(|\text{오차}|\le a_{N+1}\)).

```python
# 교대급수 S = sum_{n=1}^\infty (-1)^{n+1} a(n)
# 가정: a(n) > 0, 단조감소, a(n) -> 0
def alternating_sum(a, eps=1e-8, Nmax=10_000_000):
    s = 0.0
    n = 1
    sign = 1.0  # (-1)^{n+1}
    while n <= Nmax:
        an = a(n)
        s += sign * an
        if an <= eps:  # |S - S_n| <= a_{n+1} 이므로 n+1 항 점검해도 충분
            break
        sign = -sign
        n += 1
    return s, n

# 예1: ln 2 = sum (-1)^{n+1}/n
def a1(n): return 1.0/n

s, n = alternating_sum(a1, eps=1e-6)
print("approx ln2 =", s, " terms =", n)

# 예2: 조건수렴(로그 약감쇠) sum (-1)^n / (n*(log n)^0.5), n>=2
import math
def a2(n): 
    if n < 2: return 1.0  # 더미
    return 1.0/(n*(math.log(n)**0.5))

s2, n2 = alternating_sum(a2, eps=1e-6)
print("example with n*(log n)^0.5 denominator:", s2, " terms =", n2)
```

### 10.2 목표 오차에서 필요한 항 개수 즉시 계산
교대 조화급수처럼 \(a_n\)의 폐형이 간단하면 **직접 계산**이 가능하다.

```python
# ln 2 근사: |S - S_N| <= 1/(N+1) <= eps  =>  N >= 1/eps - 1
def N_for_ln2(eps):
    return int(math.ceil(1.0/eps - 1.0))

import math
for eps in [1e-2, 1e-4, 1e-6]:
    print(eps, "=> N >=", N_for_ln2(eps))
```

### 10.3 재배열의 민감성(조건수렴) 시뮬레이션
간단한 휴리스틱으로 “양의 항을 몇 개 더해 목표를 초과하면 음의 항을 더해 내리고…”를 반복하면,  
교대 조화급수(조건수렴)가 **어떤 값**에라도 근접할 수 있음을 볼 수 있다(리만 정리의 직관적 데모).

```python
# 재배열 데모: alternating harmonic series를 목표값 T로 근사
def rearrange_to_target(T, Kmax=200000):
    pos = []  # +1/k (k=1,3,5,...)
    neg = []  # -1/k (k=2,4,6,...)
    for k in range(1, Kmax+1):
        term = 1.0/k
        (pos if k%2==1 else neg).append(term)
    i = j = 0
    s = 0.0
    seq = []
    while i < len(pos) or j < len(neg):
        while i < len(pos) and s < T:
            s += pos[i]; seq.append(+pos[i]); i += 1
        while j < len(neg) and s > T:
            s -= neg[j]; seq.append(-neg[j]); j += 1
        if (i >= len(pos) or j >= len(neg)):
            break
    return s, seq

val, seq = rearrange_to_target(T=1.0)  # 예: 1.0에 가깝게
print("rearranged partial sum ~", val, " (length =", len(seq), ")")
```

> 이 코드는 이론 데모일 뿐이며, 실제로는 항 선택 정책/정지 조건 등에 따라 수렴 속도가 크게 달라진다. 핵심은 **조건수렴 급수는 순서에 민감**하다는 점이다.

## 11. 자주 묻는 질문(FAQ)

- Q1. \(a_n\to 0\)이면 교대급수는 항상 수렴하는가?  
  A. 아니다. \(a_n\to 0\)만으로는 부족하다. 보통 **(결국) 단조감소**까지 있어야 라이프니츠 판정으로 즉시 결론을 낼 수 있다. 단조감소가 아니더라도 Dirichlet/Abel 등 다른 판정으로 수렴을 보일 수는 있다.

- Q2. 절대수렴과 조건수렴의 실무적 차이는?  
  A. 절대수렴이면 재배열해도 합이 변하지 않아 수치적으로 안정적이다. 조건수렴이면 **순서에 따라 합이 달라질 수 있으므로** 수치 알고리즘 설계에서 주의가 필요하다.

- Q3. 오차가 \(a_{N+1}\)보다 작다는 사실은 언제 쓰나?  
  A. 교대 테일러 급수(예: \(\log(1+x),\ \arctan x\) @ 경계) 근사에 매우 유용하며, 목표 정밀도로 필요한 항 수를 **즉시** 산출할 수 있다.

## 12. 요약

- **교대급수 수렴(Leibniz)**: \(a_n>0\), (결국) 단조감소, \(a_n\to 0\).  
- **오차 추정**: \(|S-S_N|\le a_{N+1}\). 목표 오차에서 필요한 항 수를 바로 구할 수 있다.  
- **절대수렴 ⇒ 수렴**(재배열 불변), **조건수렴**은 재배열에 민감(리만 정리).  
- 단조감소가 애매하면 **Dirichlet/Abel**로 진동×감쇠 조합을 판정.  
- 테일러 급수의 경계점에서는 교대·조건수렴이 자주 등장하니 오차 추정과 함께 다룰 것.
