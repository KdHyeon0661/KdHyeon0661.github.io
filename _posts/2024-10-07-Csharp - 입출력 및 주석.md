# C# - 기본 입출력과 주석 활용

C#에서 콘솔 입출력과 주석 작성은 프로그래밍의 기본이 되는 중요한 요소입니다. 이 가이드는 간결하면서도 실용적인 예제를 통해 기본 개념을 설명합니다.

## 콘솔 입출력

### 기본 출력

```csharp
// 기본적인 콘솔 출력
Console.Write("Hello ");          // 줄바꿈 없이 출력
Console.WriteLine("World!");      // 줄바꿈 포함 출력

// 문자열 보간법 (String Interpolation)
string name = "김철수";
int age = 25;
Console.WriteLine($"이름: {name}, 나이: {age}");

// 서식 문자열 사용
Console.WriteLine("이름: {0}, 나이: {1}", name, age);
```

### 다양한 형식의 출력

```csharp
// 숫자 서식화
double price = 12345.6789;
Console.WriteLine($"가격: {price:C}");        // 통화 형식
Console.WriteLine($"가격: {price:N2}");       // 숫자 형식 (소수점 2자리)

// 날짜 서식화
DateTime now = DateTime.Now;
Console.WriteLine($"현재: {now:yyyy-MM-dd}");
Console.WriteLine($"현재: {now:HH:mm:ss}");
```

### 콘솔 입력

```csharp
// 기본 입력
Console.Write("이름을 입력하세요: ");
string? inputName = Console.ReadLine();  // nullable string
Console.WriteLine($"안녕하세요, {inputName}님!");

// 숫자 입력 처리
Console.Write("나이를 입력하세요: ");
string? ageInput = Console.ReadLine();

if (int.TryParse(ageInput, out int age))
{
    Console.WriteLine($"입력된 나이: {age}세");
}
else
{
    Console.WriteLine("올바른 숫자를 입력해주세요.");
}
```

### 여러 값 입력받기

```csharp
Console.WriteLine("학생 정보 입력");
Console.Write("이름: ");
string? studentName = Console.ReadLine();

Console.Write("학번: ");
string? studentId = Console.ReadLine();

Console.Write("학점: ");
if (double.TryParse(Console.ReadLine(), out double grade))
{
    Console.WriteLine($"\n입력된 정보:");
    Console.WriteLine($"이름: {studentName}");
    Console.WriteLine($"학번: {studentId}");
    Console.WriteLine($"학점: {grade:F2}");
}
```

### XML 문서 주석 (API 문서화)

```csharp
/// <summary>
/// 학생 정보를 나타내는 클래스입니다.
/// </summary>
public class Student
{
    /// <summary>
    /// 학생의 이름을 가져오거나 설정합니다.
    /// </summary>
    public string Name { get; set; }
    
    /// <summary>
    /// 학생의 나이를 가져오거나 설정합니다.
    /// </summary>
    /// <value>0에서 100 사이의 정수 값입니다.</value>
    public int Age { get; set; }
    
    /// <summary>
    /// 새로운 Student 인스턴스를 생성합니다.
    /// </summary>
    /// <param name="name">학생 이름</param>
    /// <param name="age">학생 나이</param>
    public Student(string name, int age)
    {
        Name = name;
        Age = age;
    }
    
    /// <summary>
    /// 학생 정보를 문자열로 반환합니다.
    /// </summary>
    /// <returns>포맷된 학생 정보 문자열</returns>
    /// <example>
    /// <code>
    /// var student = new Student("홍길동", 20);
    /// Console.WriteLine(student.GetInfo()); // "이름: 홍길동, 나이: 20"
    /// </code>
    /// </example>
    public string GetInfo()
    {
        return $"이름: {Name}, 나이: {Age}";
    }
}
```

## 실용적인 예제

```csharp
/// <summary>
/// 간단한 계산기 프로그램
/// </summary>
class Calculator
{
    /// <summary>
    /// 두 수의 합을 계산합니다.
    /// </summary>
    /// <param name="a">첫 번째 숫자</param>
    /// <param name="b">두 번째 숫자</param>
    /// <returns>두 수의 합</returns>
    public int Add(int a, int b)
    {
        return a + b;
    }
    
    /// <summary>
    /// 사용자로부터 계산을 입력받고 결과를 출력합니다.
    /// </summary>
    public void Run()
    {
        Console.WriteLine("간단 계산기");
        Console.WriteLine("============");
        
        // 첫 번째 숫자 입력
        Console.Write("첫 번째 숫자를 입력하세요: ");
        string? input1 = Console.ReadLine();
        
        // 두 번째 숫자 입력  
        Console.Write("두 번째 숫자를 입력하세요: ");
        string? input2 = Console.ReadLine();
        
        // 입력값 검증 및 계산
        if (int.TryParse(input1, out int num1) && int.TryParse(input2, out int num2))
        {
            int result = Add(num1, num2);
            Console.WriteLine($"결과: {num1} + {num2} = {result}");
        }
        else
        {
            Console.WriteLine("올바른 숫자를 입력해주세요!");
        }
        
        Console.WriteLine("\n계산이 완료되었습니다.");
    }
}

// 프로그램 실행
class Program
{
    static void Main()
    {
        // 계산기 인스턴스 생성 및 실행
        Calculator calc = new Calculator();
        calc.Run();
        
        // 사용자 응답 대기
        Console.Write("\n계속하려면 아무 키나 누르세요...");
        Console.ReadKey();
    }
}
```

## 핵심 요약

1. **콘솔 출력**: `Console.Write()`, `Console.WriteLine()`으로 사용자에게 정보 표시
2. **콘솔 입력**: `Console.ReadLine()`으로 사용자 입력 받기
3. **형식화 출력**: 문자열 보간법(`$"..."`)으로 가독성 높은 출력 생성
4. **안전한 입력**: `TryParse()` 메서드로 사용자 입력 검증
5. **한 줄 주석**: `//`로 간단한 설명 추가
6. **여러 줄 주석**: `/* */`로 긴 설명 작성
7. **XML 문서 주석**: `///`로 API 문서화 - IDE에서 IntelliSense 지원

이러한 기본적인 입출력과 주석 활용법을 익히면 C# 프로그래밍의 기초를 다질 수 있으며, 코드의 가독성과 유지보수성을 크게 향상시킬 수 있습니다.
