# 회전체의 표면적 (Surface Area of a Solid of Revolution)

> 한 줄 요약: 곡선의 작은 길이 요소 \(ds\)를 축 둘레 길이(=원둘레)와 곱해 적분하면 표면적이 된다.  
> \[
> \boxed{\,S=2\pi\int (\text{회전 반지름})\cdot ds\,}\qquad
> ds=\sqrt{1+\big(f'(x)\big)^2}\,dx.
> \]

---

## 1) 개념과 미세 원대(원통띠) 근사

곡선 \(y=f(x)\)를 \(x\in[a,b]\)에서 축 주위로 회전시키면, 곡선의 아주 작은 조각 길이 \(ds\)가 회전해 **가느다란 원대(frustum)**를 만든다.
- 원대의 **둘레**: \(2\pi \times (\text{반지름})\)
- 원대의 **높이**: \(ds\)

작은 조각들의 면적을 더해 극한을 취하면 전체 표면적:
\[
S=\int 2\pi(\text{반지름})\;ds.
\]

---

## 2) 공식 총정리

### (A) \(x\)-축 회전 — \(y=f(x)\)
반지름 \(=|f(x)|\), \(ds=\sqrt{1+f'(x)^2}\,dx\)
\[
\boxed{\,S_{x\text{-axis}}=2\pi\int_{a}^{b} |f(x)|\sqrt{1+\big(f'(x)\big)^2}\,dx\,}.
\]
(대개 \(f\ge 0\)인 상황에서 절댓값 없이 표기)

### (B) \(y\)-축 회전 — \(y=f(x)\) (또는 \(x=g(y)\))
- \(y=f(x)\)를 \(y\)-축 회전(반지름 \(=|x|\)):
\[
\boxed{\,S_{y\text{-axis}}=2\pi\int_{a}^{b} |x|\sqrt{1+\big(f'(x)\big)^2}\,dx\,}.
\]
- \(x=g(y)\)를 \(y\)-축 회전(반지름 \(=|g(y)|\), \(ds=\sqrt{1+g'(y)^2}\,dy\)):
\[
\boxed{\,S_{y\text{-axis}}=2\pi\int_{c}^{d} |g(y)|\sqrt{1+\big(g'(y)\big)^2}\,dy\,}.
\]

### (C) 매개변수형 \((x(t),y(t))\), \(t\in[\alpha,\beta]\)
속력 \(v(t)=\sqrt{x'(t)^2+y'(t)^2}\).
\[
\boxed{\,S_{x\text{-axis}}=2\pi\int_{\alpha}^{\beta} |y(t)|\,v(t)\,dt,\qquad
S_{y\text{-axis}}=2\pi\int_{\alpha}^{\beta} |x(t)|\,v(t)\,dt\,}.
\]

### (D) 극좌표 \(r=r(\theta)\) (참고)
극좌표 자체는 축회전 공식과 바로 직결되진 않지만, \((x,y)=(r\cos\theta,r\sin\theta)\)로 바꿔 (C)를 쓰면 된다.

---

## 3) 기초 예제

### 예제 1 — \(y=x^2\) (\(x\)-축 회전, \(x\in[0,1]\))
\[
S=2\pi\int_0^1 x^2\sqrt{1+4x^2}\,dx.
\]
표준 원시함수(치환 \(u=2x\))를 이용하면
\[
\int x^2\sqrt{1+4x^2}\,dx
=\frac{1}{64}\Big[u\sqrt{1+u^2}\,(2u^2+1)-\operatorname{asinh}(u)\Big]_{u=2x}.
\]
따라서
\[
S
=2\pi\cdot\frac{1}{64}\left[2\sqrt{5}\cdot 9-\operatorname{asinh}(2)\right]
\approx 3.81\ (\text{단위}^2).
\]
> 수치검산: 사다리꼴/심슨 규칙으로도 쉽게 확인 가능.

---

### 예제 2 — 반원 \(y=\sqrt{r^2-x^2}\), \(x\in[-r,r]\)의 \(x\)-축 회전
미분 \(y'=-x/\sqrt{r^2-x^2}\).  
핵심 단순화:
\[
\sqrt{1+\left(\frac{y'}{}\right)^2}
=\sqrt{1+\frac{x^2}{r^2-x^2}}
=\sqrt{\frac{r^2}{r^2-x^2}}
=\frac{r}{\sqrt{r^2-x^2}}
=\frac{r}{y}.
\]
따라서 적분핵이 \(y\cdot\frac{r}{y}=r\)로 상수화:
\[
S=2\pi\int_{-r}^{r} r\,dx
=4\pi r^2.
\]
> **구의 표면적**이 자연스럽게 산출됨.

---

### 예제 3 — 직선 \(y=\frac{R}{L}x\), \(x\in[0,L]\)의 \(x\)-축 회전 (원뿔 옆면)
\[
y'=\frac{R}{L},\quad \sqrt{1+y'^2}=\frac{\sqrt{L^2+R^2}}{L}=\frac{\ell}{L}
\ (\ell=\text{모선}).
\]
\[
S=2\pi\int_0^L \frac{R}{L}x\cdot\frac{\ell}{L}\,dx
=2\pi\frac{R\ell}{L^2}\cdot\frac{L^2}{2}
=\boxed{\pi R\ell}.
\]
> 교과서 공식과 일치: **원뿔 옆면적** \(=\pi r \times \text{모선}\).

---

### 예제 4 — \(y=f(x)=\dfrac{1}{x}\), \(x\in[1,\infty)\)의 \(x\)-축 회전 (**가브리엘의 뿔**)
- **체적**: \(\displaystyle V=\pi\int_1^\infty \frac{1}{x^2}\,dx=\pi\) (유한)
- **표면적**:
\[
S=2\pi\int_1^\infty \frac{1}{x}\sqrt{1+\frac{1}{x^4}}\,dx
>2\pi\int_1^\infty \frac{1}{x}\,dx=\infty.
\]
> **유한 체적이지만 표면적은 무한대**인 고전적 역설적 예(수렴·발산 판정의 중요성).

---

## 4) 정칙성·수렴 체크리스트

- (C¹이면 충분) \(f\)가 구간에서 **부분별 미분 가능**이고  
  \(\int |f|\sqrt{1+f'^2}\)이 수렴하면 표면적 유한.
- 끝점에서 축에 닿는 경우(\(f\to 0\))엔 **임프로퍼 적분** 여부 확인:
  \(\int_a^{a+\varepsilon} f\sqrt{1+f'^2}\,dx\)의 수렴성 점검.
- 회전축과 곡선이 교차/접촉하는 구간은 **구간 분할**하여 적분.

---

## 5) 빠른 도구: 파이썬 수치적분 & 시각화 (단일 플롯)

> 노트북/로컬에서 실행해 보세요. (matplotlib만 사용, 색상 미지정)

### (A) 공통 유틸
```python
import numpy as np

def trapz(y, x):
    return np.trapz(y, x)  # 사다리꼴 규칙
```

### (B) 예제 1: \(y=x^2\), \(x\in[0,1]\)
```python
import numpy as np
import matplotlib.pyplot as plt

f  = lambda x: x**2
fp = lambda x: 2*x

x = np.linspace(0, 1, 20001)
integrand = 2*np.pi * f(x) * np.sqrt(1 + fp(x)**2)
S = trapz(integrand, x)
print("Surface area ~", S)  # ≈ 3.81

# 간단 시각화(함수와 회전 반지름)
plt.figure()
plt.plot(x, f(x), label='y = x^2')
plt.title('Surface-of-Revolution Integrand Preview')
plt.xlabel('x'); plt.ylabel('y')
plt.grid(True); plt.legend()
plt.show()
```

### (C) 예제 3: 원뿔 옆면(검산)
```python
R, L = 3.0, 5.0
f  = lambda x: (R/L)*x
fp = lambda x: R/L

x = np.linspace(0, L, 20001)
integrand = 2*np.pi * f(x) * np.sqrt(1 + fp(x)**2)
S_num = trapz(integrand, x)

ell = (L**2 + R**2)**0.5
S_ref = np.pi * R * ell

print("Numeric ~", S_num, " | Closed-form =", S_ref)
```

## 6) (고급) 파푸스의 중심정리(Pappus)로 빠르게 보기

- **곡선** 길이 \(L\)를 축 주위로 회전할 때, 축이 곡선을 가르지 않으면  
  \[
  \boxed{S = (2\pi \cdot d)\, L}
  \]
  여기서 \(d\)는 곡선 **무게중심(centroid)**의 회전 반경(축까지 거리).
- 예: **원**의 둘레 생성: 반원호(길이 \(L=\pi r\))의 중심 거리 \(=\frac{2r}{\pi}\) 가 아니므로(곡선/영역 혼동 주의) 이 정리를 적용하려면 정확한 **곡선의 무게중심**이 필요. 실전에서는 무게중심 구하기가 번거로우므로 기본 적분 공식을 더 자주 사용한다.

---

## 7) 자주 틀리는 포인트

1) \(x\)-축 회전인데 반지름을 \(x\)로 쓰거나, \(y\)-축 회전인데 반지름을 \(y\)로 쓰는 실수  
→ **반지름은 회전축으로부터의 수직거리**다.  
2) 절댓값을 잊는 경우  
→ 함수가 음수가 될 수 있으면 \( |f(x)| \) 필요.  
3) 임프로퍼 적분 점검 누락  
→ 끝점/무한구간에서 수렴성 확인 필수.  
4) 매개변수화 오류  
→ 속력 \(v=\sqrt{x'^2+y'^2}\)를 반드시 곱한다.

---

## 8) 추가 연습문제 (스케치 해설)

1. \(y=\sqrt{x}\), \(x\in[0,4]\)를 \(x\)-축 회전.  
   \(f'=1/(2\sqrt{x})\Rightarrow S=2\pi\int_0^4 \sqrt{x}\sqrt{1+\frac{1}{4x}}\,dx\) (끝점 \(x=0\) 임프로퍼 처리).

2. \(x=y^2\), \(y\in[0,2]\)를 \(y\)-축 회전.  
   \(g(y)=y^2,\ g'=2y\Rightarrow S=2\pi\int_0^2 y^2\sqrt{1+4y^2}\,dy\).

3. 매개변수 \(x=t^3-3t,\ y=2t^2+1,\ t\in[0,2]\)를 \(x\)-축 회전.  
   \(v=\sqrt{(3t^2-3)^2+(4t)^2}\), \(S=2\pi\int_0^2 |y(t)|\,v\,dt\) (수치적분 권장).

---

## 9) 핵심 요약

- \(\displaystyle S=2\pi\int (\text{반지름})\cdot\sqrt{1+f'^2}\,dx\) (또는 해당 변수/매개로 변환)  
- **원뿔 옆면** \(\pi r\ell\), **구 표면** \(4\pi r^2\)가 자연스럽게 도출된다.  
- **수렴성**과 **반지름 선택**만 틀리지 않으면 대부분의 표면적 문제는 일관된 레시피로 해결된다.
