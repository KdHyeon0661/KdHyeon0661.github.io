# 무한급수(Infinite Series)

## 1. 무한급수의 기본 개념

### 1.1 정의와 표기
무한급수는 무한히 많은 항을 더한 것:
$$
\sum_{n=1}^{\infty} a_n = a_1 + a_2 + a_3 + \cdots
$$
여기서 $a_n$은 일반항입니다.

### 1.2 부분합과 수렴
$n$번째 **부분합**을
$$
S_n=\sum_{k=1}^n a_k
$$
로 두면, 무한급수의 수렴은
$$
\sum_{n=1}^{\infty} a_n \text{ 이 수렴} \iff \lim_{n\to\infty} S_n=S \text{(유한)}
$$
으로 정의합니다. 이때 $S$를 급수의 합이라 합니다.

### 1.3 Cauchy 수렴 기준(본질)
임의의 $\varepsilon>0$에 대해 $\exists N$ s.t. $\forall m>n\ge N$이면
$$
\biggl|\sum_{k=n+1}^m a_k\biggr|<\varepsilon.
$$
즉 꼬리합이 임의로 작아질 수 있으면 수렴합니다(완비성에 기반).

---

## 2. 가장 기초적인 사실

- (필요조건) $$\sum a_n \text{ 수렴 } \Rightarrow a_n\to 0.$$
- 역은 거짓: $a_n\to 0$이더라도 조화급수 $\sum 1/n$처럼 발산할 수 있습니다.

**예)** 등비급수:
$$
\sum_{n=0}^{\infty} ar^n=
\begin{cases}
\dfrac{a}{1-r},& |r|<1,\\[4pt]
\text{발산},& |r|\ge 1.
\end{cases}
$$

**예)** 조화급수:
$$
\sum_{n=1}^{\infty}\frac1n \quad\text{발산.}
$$

---

## 3. 양항급수(항이 $\ge 0$)의 판정법

양의 항 급수는 단조성/비교가 특히 잘 통합니다.

### 3.1 비교판정법(Direct Comparison)
$a_n,b_n\ge 0$, 그리고 $0\le a_n\le b_n$이면
- $\sum b_n$ 수렴 $\Rightarrow$ $\sum a_n$ 수렴
- $\sum a_n$ 발산 $\Rightarrow$ $\sum b_n$ 발산

**예)** $0\le \dfrac{\ln n}{n^2}\le \dfrac{n}{n^2}=\dfrac1n$은 불가(반례!) — 비교는 **더 큰 쪽**을 잘 골라야 합니다.  
정확히는, $\ln n \le n^{1/2}$ (충분히 큰 $n$에서)로부터
$$
\frac{\ln n}{n^2}\le \frac{n^{1/2}}{n^2} = \frac1{n^{3/2}}\ \Rightarrow\ \sum \frac{\ln n}{n^2}\ \text{수렴.}
$$

### 3.2 극한비교판정법(Limit Comparison)
$a_n,b_n>0$이고
$$
\lim_{n\to\infty}\frac{a_n}{b_n}=c\in(0,\infty)
$$
이면 $\sum a_n$과 $\sum b_n$은 **동시 수렴/발산**.

**예)** $a_n=\dfrac{3n+5}{n^2+7},\ b_n=\dfrac1n$:
$$
\lim_{n\to\infty}\frac{a_n}{b_n}
=\lim_{n\to\infty}\frac{3n+5}{n^2+7}\cdot n
=\lim_{n\to\infty}\frac{3n^2+5n}{n^2+7}=3\ (\neq 0,\infty).
$$
$\sum b_n=\sum 1/n$가 발산이므로 $\sum a_n$도 발산.

### 3.3 적분판정법(Integral Test)
$f(x)\ge 0$, 연속, 단조감소이고 $f(n)=a_n$이면
$$
\sum_{n=1}^{\infty} a_n \text{ 수렴 } \iff \int_1^{\infty} f(x)\,dx \text{ 수렴}.
$$
**p-급수**: $\sum 1/n^p$는 $p>1$일 때 수렴, $p\le 1$일 때 발산.

**예)** $\sum \dfrac{1}{n(\ln n)^p}$ ($n\ge 2$부터).
- $p>1$ 수렴, $p\le 1$ 발산.

### 3.4 Cauchy 응축판정(Cauchy Condensation)
$a_n\downarrow 0$ (감소, 양의항)일 때
$$
\sum_{n=1}^{\infty} a_n \text{ 수렴 } \iff \sum_{k=0}^{\infty} 2^k a_{2^k} \text{ 수렴}.
$$
**예)** $a_n=1/n$이면 $2^k a_{2^k}=2^k/2^k=1$ → 응축급수 $\sum 1$ 발산 → 원급수 발산.

### 3.5 Raabe(라베) 판정(조금 진보)
$a_n>0$에서
$$
\lim_{n\to\infty} n\biggl(1-\frac{a_{n+1}}{a_n}\biggr)=R.
$$
$R>1$이면 수렴, $R<1$이면 발산, $R=1$은 불명.

## 4. 교대급수와 조건수렴

### 4.1 Leibniz(라이프니츠) 판정(교대급수)
$(-1)^{n+1}a_n$ 꼴, $a_n\downarrow 0$이면 $\sum (-1)^{n+1} a_n$ 수렴.

**나머지(오차) 경계**: 부분합 $S_N$과 합 $S$의 차는
$$
|S-S_N|\le a_{N+1}.
$$

**예)** 교대조화급수 $\sum_{n=1}^{\infty} (-1)^{n+1}\frac1n=\ln 2$ (수렴).  
$N$번째까지 더했을 때 오차 $\le 1/(N+1)$.

### 4.2 절대수렴 vs 조건수렴
- **절대수렴**: $\sum |a_n|$ 수렴 ⇒ $\sum a_n$도 수렴(어떤 순열로 재배열해도 합이 동일).
- **조건수렴**: $\sum a_n$ 수렴하지만 $\sum |a_n|$ 발산.  
  (예: $\sum (-1)^{n+1}\frac1n$) — **재배열**에 따라 합이 변할 수 있음(리만 재배열 정리).

### 4.3 Dirichlet/Abel 판정(교대 또는 진동성)
- **Dirichlet**: $\{A_n=\sum_{k=1}^n a_k\}$이 유계, $b_n\downarrow 0$이면 $\sum a_n b_n$ 수렴.  
  **예)** $\sum \dfrac{\sin n}{n}$ 수렴(조건수렴).
- **Abel**: $\sum a_n$이 부분합 유계, $b_n$ 단조·유계이면 $\sum a_n b_n$ 수렴.

## 5. 비율/근 판정법과 전형적 패턴

### 5.1 비율판정(Ratio Test)
$$
L=\lim_{n\to\infty}\left|\frac{a_{n+1}}{a_n}\right|.
$$
- $L<1$: 수렴, $L>1$ 또는 $=\infty$: 발산, $L=1$: 불명.

**예)** $a_n=\dfrac{n!}{n^n}$:
$$
\frac{a_{n+1}}{a_n}=\frac{(n+1)!}{(n+1)^{n+1}} \cdot \frac{n^n}{n!}
= \Bigl(\frac{n}{n+1}\Bigr)^n \to e^{-1}<1 \Rightarrow \text{수렴.}
$$

### 5.2 근판정(Root Test)
$$
L=\lim_{n\to\infty}\sqrt[n]{|a_n|}.
$$
조건은 비율판정과 동일.

**예)** $a_n=\dfrac{3^n}{n!}$: $\sqrt[n]{a_n}\sim \dfrac{3}{n/e}= \dfrac{3e}{n}\to 0$ ⇒ 수렴.

**주의**: $L=1$이면 둘 다 **판단 불가**(예: $a_n=1/n$).

## 6. 망원(텔레스코핑)과 부분분수

### 6.1 망원급수
$$
\sum_{n=1}^{\infty}(b_n-b_{n+1})=b_1-\lim_{n\to\infty}b_{n+1}.
$$
**예)** $\sum_{n=1}^{\infty}\left(\frac1n-\frac1{n+1}\right)=1$.

### 6.2 부분분수 분해
$$
\frac{1}{n(n+1)}=\frac{1}{n}-\frac{1}{n+1}\ \Rightarrow\ \sum\frac{1}{n(n+1)}=1.
$$
**예)** $\dfrac{1}{n(n+2)}=\dfrac12\Bigl(\dfrac1n-\dfrac1{n+2}\Bigr)$ 등.

---

## 7. 급수와 함수 — 테일러/멱급수

### 7.1 멱급수와 수렴반경
$$
\sum_{n=0}^{\infty} c_n (x-a)^n.
$$
비율 또는 근판정으로 **수렴반경 $R$** 결정:
$$
\frac{1}{R}=\limsup_{n\to\infty}\sqrt[n]{|c_n|}=\lim_{n\to\infty}\left|\frac{c_{n+1}}{c_n}\right|
\ (\text{존재 시}).
$$

### 7.2 대표 예
- $e^x=\sum_{n=0}^{\infty}\dfrac{x^n}{n!}$ (모든 $x$에서 수렴)
- $\sin x=\sum_{n=0}^{\infty}\dfrac{(-1)^n x^{2n+1}}{(2n+1)!}$  
- $\cos x=\sum_{n=0}^{\infty}\dfrac{(-1)^n x^{2n}}{(2n)!}$  
- $\ln(1+x)=\sum_{n=1}^{\infty}\dfrac{(-1)^{n+1}x^n}{n}$, $|x|<1$ (끝점 $x=1$에서 조건수렴, $x=-1$에서 발산)

---

## 8. 나머지(오차) 평가

### 8.1 교대급수(Leibniz) 나머지
$$
|S-S_N|\le a_{N+1}.
$$
**예)** $\sum(-1)^{n+1}\dfrac1n$에서 $N=999$면 오차 $\le 1/1000$.

### 8.2 적분판정 나머지(양의 단조감소)
$f(n)=a_n$이면
$$
\int_{N+1}^{\infty} f(x)\,dx \le S-S_N \le \int_{N}^{\infty} f(x)\,dx.
$$
**예)** $\sum \dfrac1{n^2}$의 꼬리오차는 $\approx \dfrac1{N}$ 수준보다 더 작게 (정확히는 $\int_N^\infty 1/x^2dx=1/N$).

## 9. 대표 예제 모음 (풀이 포함)

### 예제 1. $\displaystyle \sum_{n=1}^{\infty}\frac{1}{n^2}$ 수렴 여부와 합
- $p=2>1$이므로 수렴. 합은 고전적 결과로
$$
\sum_{n=1}^{\infty}\frac{1}{n^2}=\frac{\pi^2}{6}.
$$
(푸리에 급수/오일러 해법 등으로 증명 가능)

**수치 근사 코드**
```python
import math
def partial_sum_N2(N):
    return sum(1.0/(n*n) for n in range(1, N+1))
for N in [10, 100, 1000, 10000]:
    s = partial_sum_N2(N)
    print(N, s, "error≈", abs((math.pi**2)/6 - s))
```

### 예제 2. $\displaystyle \sum_{n=2}^{\infty}\frac{1}{n(\ln n)^p}$ 의 수렴성
- 적분판정으로 $p>1$ 수렴, $p\le 1$ 발산.

```python
# 간단한 부분합 관찰(발산/수렴 경향)
import math
def S(N,p):
    return sum(1.0/(n*(math.log(n)**p)) for n in range(2, N+1))
for p in [0.8,1.0,1.2]:
    for N in [10**3, 10**4, 10**5]:
        print("p=",p,"N=",N,"S≈",S(N,p))
```

### 예제 3. $\displaystyle \sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}$ (교대조화급수)
- Leibniz 판정: 수렴, 합은 $\ln 2$.
- $N$항까지의 오차 $\le 1/(N+1)$.

```python
import math
def alt_harmonic(N):
    return sum((1 if n%2==1 else -1)/n for n in range(1,N+1))
for N in [10,100,1000,10000]:
    s=alt_harmonic(N)
    print(N, s, "error≤", 1.0/(N+1), "target≈", math.log(2))
```

### 예제 4. 극한비교: $\displaystyle \sum_{n=1}^{\infty}\frac{\ln n}{n^p}$
- $p>1$이면 $\ln n\le n^{\varepsilon}$를 이용해 $p-\varepsilon>1$로 비교 ⇒ 수렴.
- $p=1$이면 $\sum \dfrac{\ln n}{n}$은 적분판정으로 발산.
- $0<p<1$는 당연히 발산.

### 예제 5. 비율판정: $\displaystyle \sum_{n=1}^{\infty}\frac{n!}{n^n}$
- 앞서 계산한 대로 수렴.

```python
def a(n): return math.factorial(n)/(n**n)
def ratio(n): return a(n+1)/a(n)
for n in [5,10,20,50]:
    print(n, ratio(n), "(→ e^{-1}≈", 1/math.e, ")")
```

### 예제 6. 망원: $\displaystyle \sum_{n=1}^{\infty}\left(\frac{1}{n}-\frac{1}{n+1}\right)$
- 부분합 $S_N=1-\dfrac{1}{N+1}\to 1$.

### 예제 7. Dirichlet: $\displaystyle \sum_{n=1}^{\infty}\frac{\sin n}{n}$
- $\sum \sin n$의 부분합은 유계, $1/n\downarrow 0$ ⇒ 조건수렴.

### 예제 8. 조건수렴과 재배열(개념)
- $\sum (-1)^{n+1}\dfrac{1}{n}$은 절대수렴이 아니므로, 항의 순서를 바꾸면 합이 변할 수 있습니다(리만).

## 10. 실전 체크리스트(함정 포함)

- **항의 극한**: $a_n\not\to 0$이면 즉시 발산.
- **비율/근 판정의 $L=1$**: 판단 불가. 다른 판정(비교/적분/교대 등)로 전환.
- **단조감소 조건**: 적분판정/Leibniz에 필요(형식적으로라도 큰 $n$에서 성립 확인).
- **비교 대상 선택**: 비교판정은 상·하한을 올바르게. (예: $\ln n/n^2$는 $1/n^{3/2}$와 비교)
- **양의항 vs 부호교대**: 양의항은 비교·응축·적분, 교대는 Leibniz/Dirichlet가 강력.
- **절대수렴 우선**: 가능하면 $|a_n|$로 평가 → 안전.
- **끝점 점검**: 멱급수/테일러는 **수렴반경 경계점**에서 따로 판정.

## 11. 마무리 표(요약)

| 주제 | 요지 | 대표 예 |
|---|---|---|
| 필요조건 | 수렴 ⇒ $a_n\to 0$ | 반례: $a_n=1/n$ (발산) |
| 비교/극한비교 | 상하 비교 또는 비율 극한 | $\ln n/n^2$ vs $1/n^{3/2}$ |
| 적분판정 | $\sum a_n$ ↔ $\int f$ | $1/n^p$, $1/[n(\ln n)^p]$ |
| 응축판정 | 감소 양의항에 강력 | $1/n$ 발산 판정 |
| 비율/근 | 팩토리얼/지수형에 적합 | $n!/n^n$ 수렴 |
| 교대(Leibniz) | 단조↓, $a_n\to 0$ | $\sum (-1)^{n+1}/n$ |
| Dirichlet/Abel | 진동 × 감소 | $\sum \sin n/n$ |
| 망원/부분분수 | 구조 분해로 합 도출 | $\sum(1/n-1/(n+1))$ |
| 멱급수 | 수렴반경 $R$ | $e^x,\ \ln(1+x)$ |
| 오차 | 교대: $\le a_{N+1}$ | 교대조화 오차 |

## 12. 연습문제(요청하신 3문항 상세 해설)

### (1) $\displaystyle \sum_{n=1}^{\infty}\frac{1}{n^2}$ 수렴성 및 합 추정
- $p=2>1$ → 수렴. 합은 $\pi^2/6$.
- 수치 추정: 부분합 $S_N$ 계산 후 오차는 꼬리적분으로 근사
  $$
  0<\sum_{n=N+1}^{\infty}\frac{1}{n^2}\le \int_{N}^{\infty}\frac{dx}{x^2}=\frac1N.
  $$
- 예: $N=1000$이면 오차 $\le 0.001$.

```python
import math
def approx_zeta2(N=10000):
    S = sum(1.0/(n*n) for n in range(1, N+1))
    err_upper = 1.0/N
    return S, err_upper, (math.pi**2)/6
print(approx_zeta2(10000))
```

### (2) $\displaystyle \sum_{n=1}^{\infty}\frac{n!}{n^n}$ 수렴 여부(비율)
$$
\frac{a_{n+1}}{a_n}=\Bigl(\frac{n}{n+1}\Bigr)^n\to e^{-1}<1
\Rightarrow \text{수렴.}
$$

```python
import math
def series_val(N=20):
    return sum(math.factorial(n)/(n**n) for n in range(1, N+1))
print(series_val(10), series_val(20))
```

### (3) $\displaystyle \sum_{n=1}^{\infty}(-1)^n\cdot\frac{n}{n+1}$ 조건수렴 여부
- 일반항 $a_n=(-1)^n\frac{n}{n+1}$의 절댓값은 $\frac{n}{n+1}\to 1\neq 0$.
- **필요조건 위배** $\Rightarrow$ 급수는 발산(조건수렴 이전에 수렴 자체가 아님).

## 13. 보너스: 빠른 진단 가이드(실무 감각)

1) $a_n\not\to 0$? → 즉시 발산 종료.  
2) 양의항인가? → 비교/응축/적분/p-형으로.  
3) 팩토리얼·지수 포함? → 비율/근 판정.  
4) 교대·진동형? → Leibniz/Dirichlet/Abel.  
5) 멱급수? → $R$ 먼저, 끝점 따로.  
6) 망원형 구조? → 차분/부분분수 분해.

## 14. 미니 유틸 코드(학습용)

```python
import math

def partial_sum(a, N):           # a: 함수 a(n)
    return sum(a(n) for n in range(1, N+1))

def ratio_limit(a, N0=1000, K=50):
    # 대략적 비율한계 추정: n=N0..N0+K에서 a(n+1)/a(n)
    vals=[]
    for n in range(N0, N0+K):
        an, an1 = a(n), a(n+1)
        if an==0: continue
        vals.append(abs(an1/an))
    return sum(vals)/len(vals) if vals else float('nan')

def root_limit(a, N0=1000, K=50):
    # 대략적 근한계 추정: (|a(n)|)^(1/n)
    vals=[]
    for n in range(N0, N0+K):
        an=abs(a(n))
        if an<=0: continue
        vals.append(an**(1.0/n))
    return sum(vals)/len(vals) if vals else float('nan')

# 예: a_n = n!/n^n
def a(n): return math.factorial(n)/(n**n)

print("ratio≈", ratio_limit(a))
print("root≈", root_limit(a))
print("S_20≈", partial_sum(a, 20))
```
