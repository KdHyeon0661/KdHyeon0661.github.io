# C 언어 소개 및 개발 환경 설정

## C 언어란?

- 1972년 Dennis M. Ritchie가 UNIX 개발 과정에서 설계
- **절차지향 · 정적 타입 · 컴파일 언어**
- 시스템 프로그래밍의 표준 언어 (운영체제/드라이버/임베디드)
- 하드웨어 친화적 제어 가능 (비트/바이트 레벨 접근)
- 높은 성능과 이식성
- **메모리 직접 제어** + **포인터** + **수동 메모리 관리**

## C 언어 특징 요약

| 특징 | 설명 |
|------|------|
| 절차지향 | 함수 중심 프로그래밍 |
| 컴파일 | `.c` → 오브젝트 → 실행파일 |
| 저수준 접근 | 포인터, 비트 연산, 메모리 직접 제어 |
| 이식성 | 다양한 플랫폼에서 동작 |
| 속도 | 얇은 추상화로 인한 낮은 오버헤드 |

## 표준 버전

- **C89/C90**: 초기 표준 (가장 널리 호환)
- **C99**: `//` 주석, `inline`, `stdint.h` 추가
- **C11**: 원자성/스레딩 지원
- **C17**: 결함 정정판
- **C23**: 최신 표준 (`typeof`, `nullptr` 등)

## 첫 프로그램: Hello, World!

```c
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```

**컴파일 및 실행:**
```bash
gcc hello.c -o hello
./hello
```

## 기본 프로그램 구조

```c
#include <stdio.h>   // 표준 입출력 헤더

int main(void) {     // 프로그램 시작점
    // 실행 코드
    return 0;        // 정상 종료
}
```

## 기본 자료형

| 자료형 | 설명 | 예시 |
|--------|------|------|
| 정수형 | `int`, `long`, `short`, `char` | `int age = 25;` |
| 부호 없는 | `unsigned` | `unsigned count = 100u;` |
| 실수형 | `float`, `double` | `float pi = 3.14f;` |
| 불리언 | `_Bool` (C99) | `_Bool flag = 1;` |

## 함수 기본

```c
// 함수 선언 및 정의
int add(int a, int b) {
    return a + b;
}

int main(void) {
    int result = add(3, 4);
    printf("결과: %d\n", result);
    return 0;
}
```

## 기본 입출력

```c
#include <stdio.h>

int main(void) {
    int number;
    char name[50];
    
    printf("이름을 입력하세요: ");
    scanf("%s", name);
    
    printf("나이를 입력하세요: ");
    scanf("%d", &number);
    
    printf("안녕하세요, %s님! 나이: %d\n", name, number);
    return 0;
}
```

## 디버깅 기본

```bash
# 디버그 정보 포함 컴파일
gcc -g -O0 program.c -o program

# GDB로 디버깅
gdb ./program
# (gdb) run
# (gdb) break main
# (gdb) next
```

## 컴파일 옵션 (기본)

```bash
# 기본 빌드
gcc program.c -o program

# 경고 최대화
gcc -Wall -Wextra program.c -o program

# C11 표준 사용
gcc -std=c11 program.c -o program

# 최적화
gcc -O2 program.c -o program
```
