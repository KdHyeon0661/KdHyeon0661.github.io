# 수학적 증명이란? (Introduction to Proof)

## 0. 학습 목표

- $$\text{정의}$$, $$\text{공리}$$, $$\text{정리}$$, $$\text{보조정리}$$, $$\text{계}$$, $$\text{가설}$$, $$\text{추론 규칙}$$ 의 역할을 정확히 구분한다.
- 직접증명, 대우증명, 모순증명, 반례, 경우분할, 귀납법(약한·강한·구조적), 존재·유일성·필요충분 증명 등 **핵심 레퍼토리**를 자유롭게 선택해 쓴다.
- 간결하고 재현 가능한 **증명 작성 습관**을 체득한다.
- 파이썬 보조 실험으로 작은 범위에서 **직관 점검**을 수행하되, 실험이 증명이 아님을 분명히 한다.

---

## 1. 증명이란 무엇인가

**정의**
$$\text{증명}$$은 주어진 $$\text{공리}$$, $$\text{정의}$$, 그리고 허용된 $$\text{추론 규칙}$$을 사용하여 어떤 명제가 **참**임을 단계적으로 보이는 **유한한 논리적 서술**이다.

핵심 질문은 다음과 같다.
“이 명제는 **왜**, **무엇에 근거해**, **어떤 단계**를 거쳐 참이 되는가?”

---

## 2. 왜 증명이 필요한가

- **신뢰성**: 실험이나 많은 사례는 **가능성**을 시사할 뿐, **보편성**을 보장하지 못한다.
- **설계 검증**: 알고리즘의 정확성, 복잡도 경계, 암호 안전성, 컴파일러 최적화의 의미 보존, 회로의 기능 안전 등.
- **지식 축적**: 증명 가능한 사실만을 쌓아 올린 체계가 수학과 이론 컴퓨터 과학의 기반을 제공한다.
- **사고력**: 추상화, 귀납, 반례 구성, 불변량 탐색 등 문제해결의 핵심 기술을 훈련한다.

---

## 3. 용어 정리

| 용어 | 설명 |
|---|---|
| $$\text{정리 (Theorem)}$$ | 중요한 명제. 반드시 증명을 통해 참을 확립한다. |
| $$\text{주장 (Proposition)}$$ | 정리보다 비중이 낮지만 참임을 증명하는 명제. |
| $$\text{보조정리 (Lemma)}$$ | 큰 정리를 위해 먼저 증명하는 중간 명제. |
| $$\text{계 (Corollary)}$$ | 정리에서 즉시 따라오는 결과. |
| $$\text{공리 (Axiom)}$$ | 논리 체계에서 증명 없이 받아들이는 출발점. |
| $$\text{정의 (Definition)}$$ | 기호·개념을 도입하는 약속. 증명이 아니라 **의미 부여**다. |
| $$\text{가설 (Hypothesis)}$$ | 특정 정리의 전제 조건. |
| $$\text{추론 규칙 (Inference)}$$ | 참에서 참으로만 이동하게 하는 형식 규칙(예: Modus Ponens). |

요점: 정리·주장·보조정리는 모두 **명제**이며, **증명**을 요구한다. 정의는 **참/거짓**의 대상이 아니다.

---

## 4. 증명의 일반 구조

1) **가설 명시**: 문제의 조건과 전제를 정확히 쓴다.
2) **논리 전개**: 알려진 성질, 불변량, 등호·부등식·대수 규칙, 그래프·집합 성질 등을 사용해 단계적으로 이끈다.
3) **결론**: 목표 명제가 도출되었음을 분명히 표시한다.

좋은 증명은 **필요 최소한**의 가정으로 **명확한 단계**를 통해 **재현 가능**하게 도달한다.

---

## 5. 핵심 증명 방법들

### 5.1 직접 증명
전제에서 출발해 등식 변형, 성질 적용, 정의 전개로 곧장 결론을 연역한다.

예시(짝수의 제곱은 짝수):
$$x \text{ 가 짝수} \Rightarrow x=2k \Rightarrow x^2=4k^2=2(2k^2) \Rightarrow x^2 \text{ 도 짝수}.$$

### 5.2 대우에 의한 증명
$$p\to q \equiv \lnot q \to \lnot p$$ 를 이용해 **증명 목표를 쉽게 바꾼다**.

예시( $$x^2$$ 가 홀수이면 $$x$$ 도 홀수 ):
대우로, $$x$$ 가 짝수이면 $$x^2$$ 도 짝수. 위 5.1로 끝.

### 5.3 모순에 의한 증명
목표의 부정을 가정해 **모순**을 이끌어 원명을 참으로 결론낸다.

예시( $$\sqrt{2}$$ 는 무리수 ):
$$\sqrt{2}=\frac{a}{b}$$ 를 기약분수라 가정하면 $$2b^2=a^2$$ 이고 결국 $$a,b$$ 모두 짝수라는 모순.

### 5.4 반례로 거짓 증명
보편 명제 $$\forall x\,P(x)$$ 를 깨려면 **단 하나의 반례**면 충분하다.

예시: “모든 소수는 홀수”는 $$2$$ 가 반례.

### 5.5 경우 분할
서로 배타적으로 전체를 덮는 경우를 나누어 각각에서 결론을 얻는다.

예시: 정수 $$n$$ 에 대해 $$n^2 \equiv 0 \text{ 또는 } 1 \pmod{4}$$
짝수인 경우와 홀수인 경우로 나누어 바로 확인.

### 5.6 수학적 귀납법

**약한 귀납**
- 기초: $$n=1$$ 에서 성립.
- 가정: $$n=k$$ 에서 성립.
- 단계: $$n=k+1$$ 에서 성립을 보임.

**강한 귀납**
- 단계에서 $$\le k$$ 모두를 가정해 $$k+1$$ 을 보인다.
- 분해·재귀 구조에서 유용.

**구조적 귀납**
- 문자열·트리 등 **구성 규칙**에 따라 정의된 객체에 대해, 생성 규칙을 따라 명제를 보인다.

예시(연속 합 공식):
$$\sum_{i=1}^n i=\frac{n(n+1)}{2}$$ 은 기초와 귀납 단계 계산으로 바로 성립.

### 5.7 존재·유일성·필요충분

- **존재 증명**:
  - 구성적: 객체를 직접 만든다.
  - 비구성적: 모순, 피존홀, 극한 논증 등으로 **존재만** 보인다.
- **유일성 증명**: 존재를 보인 뒤, 임의의 두 해가 같음을 증명한다.
- **필요충분(쌍방향)**: $$P \Leftrightarrow Q$$ 는 $$P\Rightarrow Q$$ 와 $$Q\Rightarrow P$$ 를 각각 증명한다.

## 6. 대표 예제 모음

### 6.1 정수론

**예제 A.** $$n^3-n$$ 은 항상 $$6$$ 으로 나누어떨어진다.
증명: $$n(n-1)(n+1)$$ 은 연속한 세 정수의 곱이므로 **짝수**가 하나 이상, **3의 배수**가 하나 이상 포함된다. 따라서 $$6\mid n(n-1)(n+1)=n^3-n.$$

**예제 B.** $$a\mid b$$, $$a\mid c$$ 이면 임의의 정수 $$x,y$$ 에 대해 $$a\mid (bx+cy)$$.
증명: $$b=as, c=at$$ 이라 쓰면 $$bx+cy=a(sx+ty)$$ 이다.

**예제 C.** 소수는 무한히 많다(유클리드).
증명: 유한하다고 가정하여 곱에 1을 더해 모순을 낸다.

### 6.2 집합론

**예제 D.** $$A\triangle B=(A\cup B)\setminus(A\cap B)$$.
증명: 포함관계 두 방향을 원소기반으로 보이면 된다.

**예제 E.** $$A\subseteq B \Leftrightarrow A\cup B=B \Leftrightarrow A\cap B=A$$.
증명: 양방향으로 원소 기준 equivalence를 전개한다.

### 6.3 부등식

**예제 F.** $$0<c$$ 이면 $$a<b \Rightarrow \frac{a}{c}<\frac{b}{c}$$.
증명: $$b-a>0$$ 에 $$1/c>0$$ 을 곱한다.

**예제 G.** $$a,b\ge 0$$ 에서 $$\sqrt{ab}\le \frac{a+b}{2}$$ (산술-기하평균 부등식의 기본꼴).
증명: $$(\sqrt{a}-\sqrt{b})^2\ge 0 \Rightarrow a+b-2\sqrt{ab}\ge 0.$$

### 6.4 그래프 이론

**예제 H.** 트리 $$T$$ 의 간선 수는 정점 수 $$n$$ 에 대해 $$n-1$$.
증명(귀납):
- 기초 $$n=1$$: 간선 0.
- 가정 $$n=k$$: 간선 $$k-1$$.
- 단계 $$n=k+1$$: 잎 하나 제거하면 $$k$$ 정점 트리이므로 간선 $$k-1$$, 다시 잎을 붙이면 간선 $$k$$.

### 6.5 함수·단사성

**예제 I.** $$f$$ 가 실수 구간에서 **엄격 증가**이면 **단사**이다.
증명: $$f(x)=f(y)$$ 이면 증가성에 의해 $$x=y$$.

### 6.6 고전 예시

**예제 J.** $$\sqrt{2}$$ 는 무리수(모순증명). 위 5.3 참조.

## 7. 귀납법을 깊게: 약한·강한·구조적

### 7.1 동치 관점
자연수에 대한 약한 귀납, 강한 귀납, **최소 원리**는 서로 동치이다. 어떤 가설이든 하나를 가정하면 나머지를 유도할 수 있다.

### 7.2 강한 귀납 예시
모든 $$n\ge 2$$ 는 소수들의 곱으로 표현된다.
증명: $$n$$ 이 소수가 아니면 $$n=ab$$, $$2\le a,b\le n-1$$. 귀납가정으로 $$a,b$$ 를 소수 곱으로 전개.

### 7.3 구조적 귀납 예시(문자열)
알파벳 $$\Sigma$$ 로부터 생성된 괄호열의 균형 성질 같은 특성은 **생성 규칙**을 따라 증명한다.

## 8. 필요충분, 존재, 유일성

### 8.1 필요충분
$$P \Leftrightarrow Q$$ 를 보이려면
1) $$P\Rightarrow Q$$, 2) $$Q\Rightarrow P$$ 두 방향을 **독립적으로** 증명한다.

### 8.2 존재
- **구성적**: 예를 직접 제시.
- **비구성적**: 피존홀 원리, 극한, 모순 등으로 존재만 보임.

### 8.3 유일성
$$\exists!x\,P(x)$$ 를 보이려면
1) $$\exists x\,P(x)$$, 2) $$P(a),P(b)\Rightarrow a=b$$.

---

## 9. 알고리즘 정확성: 루프 불변식과 종료성

**정확성**은 보통 두 축으로 나뉜다.
- **부분 정확성**: 종료한다면 결과가 옳다.
- **종료성**: 항상 끝난다.

### 9.1 삽입정렬의 루프 불변식
- 불변식: 외부 루프의 각 단계에서, **왼쪽 구간은 정렬되어 있음**.
- 초기 성립: 길이 1 구간은 정렬.
- 유지: 한 원소를 삽입하면 정렬 상태 유지.
- 종료: 모든 원소를 삽입하면 전체 정렬.
- 종료성: 인덱스를 1씩 증가시키므로 유한 단계 후 종료.

---

## 10. 증명 작성 요령

1) **가설과 목표를 분명히**: 기호, 도메인, 상수, 변수의 의미를 먼저 고정한다.
2) **정의 전개**: 짝수, 나눗셈 가능, 단사 등은 정의를 즉시 식으로 바꾸어 사용한다.
3) **불변량·대우·모순을 빠르게 점검**: 어느 길이 가장 짧은지 선택한다.
4) **원소기반 vs 대수기반**: 집합 등식은 원소기반이 안정적이다.
5) **과정과 결론의 구분**: 계산은 메모, 최종 서술은 단정하게.
6) **가독성**: 문장형 서술 + 필요한 곳만 수식. 각 단계에 근거를 덧붙인다.
7) **WLOG 사용 자제**: 대칭성 등 분명한 이유가 있을 때만 쓰고 근거를 말한다.

## 11. 흔한 오류

- **결론 긍정**: $$p\to q,\, q \Rightarrow p$$ 는 일반적으로 성립하지 않는다.
- **전건 부정**: $$p\to q,\, \lnot p \Rightarrow \lnot q$$ 도 성립하지 않는다.
- **정의 오용**: 예를 들어 $$a\mid b$$ 는 $$\exists k\, (b=ak)$$ 을 뜻한다.
- **귀납 실수**: 기초 단계 누락, 귀납 단계에서 $$k\to k+2$$ 처럼 잘못 점프.
- **도메인 불명확**: 자연수의 시작을 0으로 볼지 1로 볼지 합의 없이 사용.
- **모순증명에서 모순 확인 실패**: 진정한 모순(동시에 참일 수 없는 두 명제)을 확보해야 한다.

## 12. 예제로 익히는 증명

### 12.1 대우증명 예제
명제: $$xy$$ 가 홀수이면 $$x$$, $$y$$ 모두 홀수.
대우: $$x$$ 또는 $$y$$ 가 짝수이면 $$xy$$ 가 짝수. 자명.

### 12.2 모순증명 예제
명제: $$\sqrt{3}$$ 은 무리수.
가정: $$\sqrt{3}=\frac{a}{b}$$ 기약분수. $$3b^2=a^2$$ 이므로 $$a$$ 가 3의 배수, 이어서 $$b$$ 도 3의 배수. 기약성에 모순.

### 12.3 강한 귀납 예제
명제: 모든 $$n\ge 2$$ 는 소수곱. 7.2 참조.

### 12.4 집합 등식 예제
명제: $$A\setminus(B\cup C)=(A\setminus B)\cap(A\setminus C)$$.
증명: 두 포함관계 각자 원소기반으로 보이면 끝.

## 13. 파이썬으로 직관 점검(증명은 아님)

작은 범위에서 패턴을 확인해 **아이디어를 검증**할 수 있다.

### 13.1 $$n^3-n$$ 의 배수성 실험
```python
def divisible_by_6(n):
    return (n**3 - n) % 6 == 0

print(all(divisible_by_6(n) for n in range(-1000, 1001)))
```

### 13.2 집합 항등식 퍼저 테스트
```python
import random
U = list(range(20))

def rset():
    return {x for x in U if random.random() < 0.5}

for _ in range(200):
    A, B, C = rset(), rset(), rset()
    left  = A - (B | C)
    right = (A - B) & (A - C)
    assert left == right
print("identity holds on random trials")
```

### 13.3 간단한 귀납 확인(수식 자체는 증명으로 처리)
```python
def s(n): return n*(n+1)//2
for n in range(1, 1000):
    assert sum(range(1, n+1)) == s(n)
print("checked n up to 999")
```

주의: **유한 검사는 증명이 아니다**. 반례 탐색과 직관 보조로만 사용한다.

## 14. 연습문제(힌트 포함)

1) 증명 방식 제안:
명제 $$\forall n\in\mathbb{N},\, n^2+n \text{ 는 짝수}$$.
힌트: 직접증명 또는 대우증명. $$n(n+1)$$ 은 연속 두 정수의 곱.

2) 경우 분할:
정수 $$n$$ 에 대해 $$n^2 \equiv 0 \text{ 또는 } 1 \pmod{4}$$.
힌트: 짝수·홀수로 분할.

3) 유일성:
방정식 $$x^2=2$$ 의 실수해는 정확히 두 개.
힌트: 존재는 연속성, 유일성은 단조성 또는 대수 전개로.

4) 필요충분:
정수 $$n$$ 에 대해 $$3\mid n \Leftrightarrow 3\mid (n^2)$$ 이 성립하는가.
힌트: 한 방향은 자명, 역방향은 모듈로 관찰.

5) 강한 귀납:
$$n\ge 2$$ 에 대해 $$n$$ 을 2의 거듭제곱과 홀수의 곱으로 표현.
힌트: 나눗셈 알고리즘으로 2를 반복 소거.

6) 집합 등식:
$$(A\cap B)\cup(A\cap C)=A\cap(B\cup C)$$.
힌트: 원소기반 두 포함.

## 15. 마무리

- 증명은 **참을 재현 가능하게 전달하는 기술**이다.
- 방법의 선택(직접·대우·모순·귀납·경우분할·존재·유일성·필요충분)이 길이를 가른다.
- 집합, 수론, 그래프, 알고리즘 등 영역마다 **정의 전개**와 **불변량**이 핵심 도구다.
