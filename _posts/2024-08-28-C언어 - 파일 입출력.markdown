# 📂 C 언어 9편 – 파일 입출력 (File I/O)

이번 편에서는 C 언어에서 **파일을 읽고 쓰는 방법**을 배웁니다. 구조체, 배열 등 다양한 데이터를 파일에 저장하고 다시 불러오는 데 필수적인 기능입니다.

---

## 📄 파일 입출력 개요

C에서는 `stdio.h`에 정의된 파일 관련 함수들을 사용합니다.

```c
FILE* 파일포인터;
```

- `FILE`은 C에서 파일 정보를 담는 구조체
- `파일포인터`는 파일을 조작할 수 있는 핸들

---

## 📖 파일 열기: `fopen()`

```c
FILE* fopen(const char* filename, const char* mode);
```

| 모드 | 의미 |
|------|------|
| `"r"` | 읽기 전용 (파일 없으면 실패) |
| `"w"` | 쓰기 전용 (파일 새로 생성, 기존 내용 삭제) |
| `"a"` | 추가 전용 (끝에 덧붙임) |
| `"r+"` | 읽기/쓰기 겸용 |
| `"w+"` | 읽기/쓰기 (기존 내용 삭제) |
| `"a+"` | 읽기/추가 |

```c
FILE* fp = fopen("data.txt", "w");
if (fp == NULL) {
    printf("파일 열기 실패\n");
}
```

---

## ✏️ 파일 쓰기

### `fprintf()`

```c
fprintf(fp, "이름: %s, 나이: %d\n", name, age);
```

### `fputs()` / `fputc()`

```c
fputs("Hello\n", fp);
fputc('A', fp);
```

---

## 📘 파일 읽기

### `fscanf()`

```c
char name[20];
int age;
fscanf(fp, "%s %d", name, &age);
```

### `fgets()` / `fgetc()`

```c
char line[100];
fgets(line, sizeof(line), fp);

char ch = fgetc(fp);
```

> ⚠️ `fgets()`는 줄 단위로 읽고 `fscanf()`는 공백 전까지 읽음

---

## ❌ 파일 닫기: `fclose()`

파일을 모두 사용한 후에는 반드시 닫아야 함

```c
fclose(fp);
```

---

## 🔁 전체 예제 – 파일에 구조체 저장

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct {
    char name[20];
    int age;
} Person;

int main() {
    FILE* fp = fopen("people.txt", "w");
    if (fp == NULL) {
        printf("파일 열기 실패\n");
        return 1;
    }

    Person p1 = {"Alice", 30};
    Person p2 = {"Bob", 25};

    fprintf(fp, "%s %d\n", p1.name, p1.age);
    fprintf(fp, "%s %d\n", p2.name, p2.age);

    fclose(fp);
    return 0;
}
```

---

## 🧾 이진 파일 (Binary File)

```c
fwrite(const void* ptr, size_t size, size_t count, FILE* stream);
fread(void* ptr, size_t size, size_t count, FILE* stream);
```

```c
fwrite(&p1, sizeof(Person), 1, fp);
fread(&p1, sizeof(Person), 1, fp);
```

> 텍스트보다 빠르고 효율적이나, 파일을 사람이 직접 읽기 어려움

---

## ⚠️ 파일 입출력 시 주의사항

- `fopen()` 반환값 확인 필수 (`NULL` 체크)
- 파일 포인터 사용 전 열기 성공 여부 확인
- 파일은 반드시 `fclose()` 호출하여 닫기
- 읽기와 쓰기 모드를 정확히 구분할 것

| 함수 | 설명 |

| `fopen()` | 파일 열기 |

| `fprintf()` | 텍스트 형식 쓰기 |
| `fscanf()` | 텍스트 형식 읽기 |
| `fputs()` / `fputc()` | 문자열/문자 쓰기 |
| `fgets()` / `fgetc()` | 문자열/문자 읽기 |
| `fwrite()` / `fread()` | 이진 형식 쓰기/읽기 |
