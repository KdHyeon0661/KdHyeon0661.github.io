# 곡선 사이의 면적과 거리

두 그래프 사이의 “면적”은 **한 축 방향(수직 or 수평)으로의 차이**를 적분해 얻는 기하량이고,  
“거리”는 문맥에 따라 **수직 거리(평균·RMS)** 또는 **진짜 유클리드 최단 거리**를 뜻한다.  

---

## 1) 곡선 사이의 면적 — 기본 원리

두 함수 $$y=f(x),\ y=g(x)$$ 가 구간 $$[a,b]$$ 에서 정의되고 **항상** $$f(x)\ge g(x)$$ 이면

\[
\boxed{\,A=\int_a^b\big(f(x)-g(x)\big)\,dx\,}.
\]

구간 내에서 위아래가 바뀌면(교점 존재) **분할**하거나 **절댓값**으로 처리한다:

\[
\boxed{\,A=\int_a^b \big|f(x)-g(x)\big|\,dx\,}.
\]

**해석 포인트**
- $$\int (f-g)$$ 는 **부호 있는 면적**(signed area).  
- 기하학적 면적은 **항상 양수** → 절댓값 또는 구간 분할 필수.  
- 교점은 $$f-g=0$$ 의 해. 닫힌형이 힘들면 수치적으로 찾는다(이분법/뉴턴법 등).

---

## 2) x-적분 vs y-적분 — 축 선택 요령

- **세로 띠(수직 슬라이스)** 가 단순 → $$x$$ 로 적분:  \(A=\int (상-하)\,dx\).
- **가로 띠(수평 슬라이스)** 가 단순 → $$y$$ 로 적분:  \(A=\int (오른쪽-왼쪽)\,dy\).

**y-적분 공식**  (그래프 $$x=f(y)\ge g(y)$$, $$y\in[c,d]$$)
\[
\boxed{\,A=\int_{c}^{d}\big(f(y)-g(y)\big)\,dy\,}.
\]

> **축 선택 체크리스트**  
> - 한 $$x$$ 값에서 영역이 **위/아래**로 깔끔히 갈리면 \(x\)-적분.  
> - 한 $$y$$ 값에서 **오른쪽/왼쪽**이 깔끔히 갈리면 \(y\)-적분.  
> - 복잡하면 교점 기준으로 **영역 분해** 후 각 조각에 최적 축을 쓴다.

---

## 3) 교점과 구간 분할 — 절차

1) **교점**: $$f(x)-g(x)=0$$ 을 풀어 \([a,b]\) 안의 해를 구한다.  
2) **구간 분할**: 교점들을 오름차순으로 정렬해 분할 구간 \([x_i,x_{i+1}]\) 생성.  
3) **각 구간의 상·하 판정**: 해당 구간에서 \(f\)와 \(g\)의 대소를 결정.  
4) **적분**: \(A=\sum \int (상-하)\,dx\) (또는 절댓값).

**수치 교점 찾기(예시 코드)**

```python
# f(x) - g(x) = 0의 근을 이분법으로 찾기 (구간 목록 주어짐)
def bisect_root(h, a, b, tol=1e-8, maxit=100):
    fa, fb = h(a), h(b)
    if fa*fb > 0:
        raise ValueError("부호 변화 없음: 교점 보장이 안 됨")
    for _ in range(maxit):
        m = 0.5*(a+b)
        fm = h(m)
        if abs(fm) < tol or 0.5*(b-a) < tol:
            return m
        if fa*fm <= 0:
            b, fb = m, fm
        else:
            a, fa = m, fm
    return 0.5*(a+b)
```

## 4) 대표 예제 (x-적분, y-적분, 교차 포함)

### 예 1 — \(y=x\) (위), \(y=x^2\) (아래), 교점 \(x=0,1\)
\[
A=\int_{0}^{1}(x-x^2)\,dx
=\Big[\tfrac12x^2-\tfrac13x^3\Big]_0^1
=\tfrac16.
\]

### 예 2 — \(y=\sin x\) vs. x축 on \([0,2\pi]\): 부호 상쇄 방지(절댓값)
\[
\int_0^{2\pi}|\sin x|\,dx
=4.
\]

### 예 3 — y-적분이 유리한 경우: \(x=\sqrt{y}\) (오른쪽), \(x=y\) (왼쪽)
교점 \(y=0,1\),
\[
A=\int_0^1\big(\sqrt{y}-y\big)\,dy
=\Big[\tfrac23 y^{3/2}-\tfrac12 y^2\Big]_0^1
=\tfrac16.
\]

### 예 4 — 교차(상·하 교대): \(f(x)=\sin x,\ g(x)=\cos x\) on \([0,\pi]\)
교점 \(x=\tfrac{\pi}{4}\),
\[
\begin{aligned}
A&=\int_{0}^{\pi/4}\!(\cos x-\sin x)\,dx \;+\; \int_{\pi/4}^{\pi}\!(\sin x-\cos x)\,dx \\
 &=\big[\sin x+\cos x\big]_0^{\pi/4} + \big[-\cos x-\sin x\big]_{\pi/4}^{\pi}
 =2.
\end{aligned}
\]
(부호 있는 면적은 0이지만, 기하학적 면적은 2가 아님에 유의 — 위 적분은 **부호 유지**.  
기하학적 면적을 원하면 절댓값 또는 조각별 “상–하”를 적용.)

### 예 5 — 여러 교차: \(f(x)=x^3-x,\ g(x)=0\) on \([-1,1]\)
교점 \(-1,0,1\). 기하학적 면적
\[
A=2\int_{0}^{1}\!(x-x^3)\,dx
=2\Big[\tfrac12x^2-\tfrac14x^4\Big]_0^1
=\tfrac12.
\]

## ✅ 5) 극좌표·매개변수 — 다른 좌표계에서의 면적

### (A) 극좌표 \(r=R(\theta)\)
원점과 곡선 사이(부채꼴) 면적
\[
\boxed{\,A=\frac12\int_{\alpha}^{\beta} R(\theta)^2\,d\theta\,}.
\]
두 극곡선 \(R_1\ge R_2\) 사이
\[
\boxed{\,A=\frac12\int_{\alpha}^{\beta}\!\big(R_1^2-R_2^2\big)\,d\theta\,}.
\]

**예 6 —** 원 \(r=2\cos\theta\) 의 오른쪽 반원 (\(\theta\in[-\tfrac{\pi}{2},\tfrac{\pi}{2}]\))
\[
A=\tfrac12\int_{-\pi/2}^{\pi/2} 4\cos^2\theta\,d\theta
=2\cdot\tfrac{\pi}{2}=\pi.
\]

### (B) 매개변수 곡선 \(\gamma:\ (x(t),y(t))\), \(t\in[\alpha,\beta]\)
닫힌 곡선의 내부 면적(양의 방향):
\[
\boxed{\,A=\int y\,dx=\int_{\alpha}^{\beta} y(t)\,x'(t)\,dt
=\frac12\int_{\alpha}^{\beta}\big(xy'-yx'\big)\,dt\,}.
\]

**예 7 —** 사이클로이드 한 아치 \(x=a(t-\sin t),\ y=a(1-\cos t),\ t\in[0,2\pi]\)
\[
A=\int y\,dx=\int_0^{2\pi} a(1-\cos t)\cdot a(1-\cos t)\,dt
=a^2\int_0^{2\pi}(1-\cos t)^2\,dt
=3\pi a^2.
\]

## ✅ 6) “거리”의 여러 정의 — 무엇을 의미하나?

### (1) 수직(또는 수평) 거리 함수
같은 \(x\) 에서의 간격:
\[
D_{\perp}(x)=|f(x)-g(x)|.
\]
- **평균 수직 거리** on \([a,b]\):
\[
\boxed{\,\overline{D}_{\perp}=\frac{1}{b-a}\int_a^b |f-g|\,dx\,}.
\]
- **RMS(제곱평균제곱근) 거리**:
\[
D_{\text{RMS}}=\sqrt{\frac{1}{b-a}\int_a^b (f-g)^2\,dx}.
\]

**예 8 —** \(f=\sin x,\ g=\cos x\) on \([0,\pi]\)
\[
\int_0^\pi |\sin-\cos|\,dx
=\big[ -\cos-\sin\big]_0^{\pi/4}
+\big[-\cos+\sin\big]_{\pi/4}^{\pi}
=2\sqrt{2},
\]
\[
\boxed{\,\overline{D}_{\perp}=\dfrac{2\sqrt{2}}{\pi}\,}.
\]
또,
\[
(f-g)^2=1-\sin(2x)\ \Rightarrow\ \frac{1}{\pi}\int_0^\pi (f-g)^2\,dx=1
\ \Rightarrow\ \boxed{\,D_{\text{RMS}}=1\,}.
\]

### (2) 유클리드 **최단 거리** (진짜 거리)
두 곡선 \(C_1:\ y=f(x)\), \(C_2:\ y=g(x)\) 사이
\[
\min_{x_1\in I,\ x_2\in J}\sqrt{(x_1-x_2)^2+\big(f(x_1)-g(x_2)\big)^2}.
\]
매끄러울 때 **필요조건**(법선 정렬):
\[
\begin{cases}
(x_1-x_2,\, f(x_1)-g(x_2)) \ \perp\ (1, f'(x_1)),\\[4pt]
(x_1-x_2,\, f(x_1)-g(x_2)) \ \perp\ (1, g'(x_2)).
\end{cases}
\]
즉,
\[
\begin{cases}
(x_1-x_2)+f'(x_1)\,(f(x_1)-g(x_2))=0,\\
(x_1-x_2)+g'(x_2)\,(f(x_1)-g(x_2))=0.
\end{cases}
\]
이 비선형 연립을 풀어 후보점을 얻고, 거리 최소를 고른다(경계 포함).

**예 9 —** 두 포물선 \(y=x^2+1\) 과 \(y=-x^2\) 의 최단거리  
대칭으로 \(x_1=x_2=0\) 가 후보. 거리 \(=|1-0|=1\).  
실제로 \(D^2=(x_1-x_2)^2+\big(x_1^2+1+x_2^2\big)^2\) 의 전역 최소가 \((0,0)\)이며 최단거리 \(1\).

> 실전에서 “거리”라 하면 문맥상 (1) **수직 평균/ RMS 거리**를 가리키는 경우가 많다.  
> “최단 거리”는 별도 **최적화 문제**로 푼다.

## ✅ 7) 실전 절차(면적 & 거리) — 체크리스트

1) **교점** 찾기: \(f-g=0\) (필요시 수치).  
2) **축 선택**: 수직/수평 띠 중 단순한 쪽.  
3) **분할**: 위아래(또는 좌우)가 바뀌는 지점에서.  
4) **부호 처리**: 기하학적 면적이면 절댓값 또는 각 분할에서 “상–하(우–좌)”.  
5) **좌표변환**: 극좌표(부채꼴), 매개변수(\(\int y\,dx\)).  
6) **거리 정의** 명확히: 평균/ RMS 수직 거리 vs 유클리드 최단 거리.  
7) **수치 검증**: 복잡한 구간·교점·비선형 최적화는 수치 체크로 교차 확인.

## ✅ 8) 추가 예제 모음 (풀이)

### 예 10 — 삼각함수 교차의 기하학적 면적 (절댓값으로)
\(f=\sin x,\ g=\cos x\) on \([0,\pi]\). 교점 \(x=\tfrac{\pi}{4}\).
\[
A=\int_0^{\pi/4}(\cos-\sin)\,dx+\int_{\pi/4}^{\pi}(\sin-\cos)\,dx
=2\sqrt{2}.
\]
(이 적분은 절댓값을 반영한 “기하학적 면적”.)

### 예 11 — y-적분으로 더 쉬운 경우
\(x=y^3\) 와 \(x=y+2\) 사이의 면적(교점 \(y_1<y_2\)):
\[
A=\int_{y_1}^{y_2}\big((y+2)-y^3\big)\,dy
=\Big[\tfrac12 y^2+2y-\tfrac14 y^4\Big]_{y_1}^{y_2}.
\]
(교점은 \(y^3-y-2=0\) 의 해. 실수근 1개, 수치로 구한다.)

### 예 12 — 극좌표: 두 곡선 사이
\(r=\sin(2\theta)\) 와 \(r=\tfrac12\) 사이(교차각 \(\theta_1,\theta_2\)):
\[
A=\frac12\int_{\theta_1}^{\theta_2}\big(\sin^2(2\theta)-\tfrac14\big)\,d\theta.
\]

### 예 13 — 평균 수직 거리 = (면적)/(폭)
\[
\overline{D}_{\perp}=\frac{1}{b-a}\int_a^b |f-g|\,dx.
\]
예 1에서 \(f=x,\ g=x^2,\ [0,1]\):
\[
\overline{D}_{\perp}=\int_0^1 (x-x^2)\,dx=\tfrac16 \quad(\text{폭 }=1).
\]

## ✅ 9) 파이썬 예제 — 평균 수직 거리 & 교점–분할–적분(수치)

```python
import numpy as np

# 평균 수직 거리: f(x)=sin x, g(x)=cos x on [0, π]
f = np.sin; g = np.cos
xs = np.linspace(0, np.pi, 200_001)
avg_vert = (1/np.pi) * np.trapz(np.abs(f(xs)-g(xs)), xs)
print("평균 수직 거리:", avg_vert)  # → 2*sqrt(2)/π

# 일반적인 "교점→분할→각 구간 면적" 루틴(수치 근사)
def area_between(f, g, a, b, roots, N=20000):
    # roots: (a,b) 내부 교점 리스트 (a<b 가정)
    pts = [a] + sorted([r for r in roots if a < r < b]) + [b]
    total = 0.0
    for L, R in zip(pts[:-1], pts[1:]):
        xs = np.linspace(L, R, N+1)
        diff = f(xs) - g(xs)
        # 이 조각에서의 '상-하'를 강제(기하학적 면적)
        total += np.trapz(np.abs(diff), xs)
    return total

# 예: f=x, g=x**2, [0,1], 교점: 0,1 (내부 교점 없음)
f = lambda x: x
g = lambda x: x*x
A = area_between(f, g, 0.0, 1.0, roots=[])
print("면적(≈):", A, " — 이론값 1/6")
```

> 수치 적분/근사에서는 **분할 수**(해상도)와 **교점 탐색 정확도**가 핵심.  
> 교점이 다수·근접하면 세밀한 분할이 필요하고, 특이점/급변 구간은 **적응적 분할**이 유리.

## ✅ 10) 함정 & 체크리스트

- [ ] **교점 누락** → 상·하가 바뀌는 구간에서 부호 상쇄/면적 오류.  
- [ ] \(x\)-적분 vs \(y\)-적분 **축 선택 오류** → 수식 복잡/오류.  
- [ ] **절댓값/분할** 미적용 → 기하학적 면적 대신 signed area.  
- [ ] 극좌표 적분에서 **범위를 한 번만** 훑는지 확인(중복 그리기 주의).  
- [ ] 매개변수 면적은 \(\int y\,dx\) (또는 \(\tfrac12\!\int(xy'-yx')dt\)).  
- [ ] 최단 거리 문제는 **별도 최적화**이며, 경계/복수 해/국소해에 주의.  
- [ ] 수치법 사용 시 **해상도·안정성**(진동/특이점) 점검.

## ✅ 11) 한눈 요약

- **면적**: \(\int (상-하)\,dx\) 또는 \(\int (오른쪽-왼쪽)\,dy\).  
  교차 시 **분할/절댓값**으로 기하학적 면적을 구한다.  
- **좌표계**: 문제 구조에 맞춰 \(x\)/\(y\)-적분, 극좌표(\(\tfrac12\int r^2 d\theta\)), 매개변수(\(\int y\,dx\)).  
- **거리**: 수직 평균/RMS vs **유클리드 최단 거리**(법선 정렬 조건으로 최적화).  
- **루틴**: 교점 → 축 선택 → 분할 → 부호 처리 → (필요 시) 좌표/표현 변환 → 수치 검증.
