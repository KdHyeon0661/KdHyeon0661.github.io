# 삼각함수의 도함수(Trigonometric Derivatives)

## 0. 준비: 반드시 라디안(radian)

아래 모든 미분 공식은 **x가 라디안**일 때만 성립합니다.  
만약 각도를 “도(degree)”로 쓴다면  
\[
\frac{d}{dx}\sin(x^\circ) = \cos(x^\circ)\cdot \frac{\pi}{180},\quad
\frac{d}{dx}\cos(x^\circ) = -\sin(x^\circ)\cdot \frac{\pi}{180}.
\]

---

## 1. 핵심 표 (라디안 기준)

| \(f(x)\) | \(f'(x)\) | 정의역 주의 |
|---|---|---|
| \(\sin x\) | \(\cos x\) | 전체 \(\mathbb{R}\) |
| \(\cos x\) | \(-\sin x\) | 전체 \(\mathbb{R}\) |
| \(\tan x\) | \(\sec^2 x\) | \(x\neq \tfrac{\pi}{2}+k\pi\) |
| \(\cot x\) | \(-\csc^2 x\) | \(x\neq k\pi\) |
| \(\sec x\) | \(\sec x\,\tan x\) | \(x\neq \tfrac{\pi}{2}+k\pi\) |
| \(\csc x\) | \(-\csc x\,\cot x\) | \(x\neq k\pi\) |

---

## 2. 왜 \(\frac{d}{dx}\sin x=\cos x\), \(\frac{d}{dx}\cos x=-\sin x\) 인가?

### 2.1 극한 기반 증명 스케치
핵심 두 극한
\[
\lim_{h\to0}\frac{\sin h}{h}=1,\qquad
\lim_{h\to0}\frac{\cos h-1}{h}=0
\]
을 이용해
\[
\frac{d}{dx}\sin x=\lim_{h\to0}\frac{\sin(x+h)-\sin x}{h}
=\lim_{h\to0}\frac{\sin x\cos h+\cos x\sin h-\sin x}{h}
=\cos x.
\]
동일하게
\[
\frac{d}{dx}\cos x=\lim_{h\to0}\frac{\cos(x+h)-\cos x}{h}
=\lim_{h\to0}\frac{\cos x(\cos h-1)-\sin x\sin h}{h}
=-\sin x.
\]

> **라디안이 핵심**: 위 두 극한 값이 1과 0이 되도록 정의된 각 단위가 바로 라디안입니다.

### 2.2 기하/회전의 직관
단위원(반지름 1)에서 각 \(x\)의 점은 \((\cos x,\sin x)\).  
각을 미세하게 \(x\mapsto x+h\) 회전하면 \(y\)-성분(=\(\sin x\))의 변화율이 바로 \(x\)-성분(=\(\cos x\))가 됩니다.  
회전을 90° 더하면 사인↔코사인이 부호와 함께 교대합니다.

## 3. 나머지 기본 4형: 몫/합성으로 유도

- \(\tan x=\dfrac{\sin x}{\cos x}\Rightarrow\)
\[
(\tan x)'=\frac{\cos x\cos x-\sin x(-\sin x)}{\cos^2 x}
=\frac{\cos^2 x+\sin^2 x}{\cos^2 x}=\sec^2 x.
\]
- \(\cot x=\dfrac{\cos x}{\sin x}\Rightarrow (\cot x)'=-\csc^2 x\).
- \(\sec x=\dfrac{1}{\cos x}\Rightarrow (\sec x)'=\sec x\tan x\).
- \(\csc x=\dfrac{1}{\sin x}\Rightarrow (\csc x)'=-\csc x\cot x\).

> **정의역 주의**: 분모가 0인 곳(수직점근선)은 **미분 불가**.

---

## 4. 연쇄법칙과 합성 함수

- \(\dfrac{d}{dx}\sin(g(x))=\cos(g(x))\cdot g'(x)\)  
- \(\dfrac{d}{dx}\cos(g(x))=-\sin(g(x))\cdot g'(x)\)  
- \(\dfrac{d}{dx}\tan(g(x))=\sec^2(g(x))\cdot g'(x)\)

### 예제 4.1
\[
f(x)=\sin(5x^2)\;\Rightarrow\; f'(x)=\cos(5x^2)\cdot 10x.
\]

### 예제 4.2
\[
f(x)=\cos(3x+1)\;\Rightarrow\; f'(x)=-\sin(3x+1)\cdot 3.
\]

---

## 5. 고계 도함수(패턴 암기 꿀팁)

\[
\frac{d^n}{dx^n}\sin x=\sin\!\Big(x+\frac{n\pi}{2}\Big),\qquad
\frac{d^n}{dx^n}\cos x=\cos\!\Big(x+\frac{n\pi}{2}\Big).
\]
즉, 미분할 때마다 **위상이 \(\pi/2\)씩** 증가합니다.

| \(n\) mod 4 | \(\sin^{(n)}x\) | \(\cos^{(n)}x\) |
|---|---|---|
| 0 | \(\sin x\) | \(\cos x\) |
| 1 | \(\cos x\) | \(-\sin x\) |
| 2 | \(-\sin x\) | \(-\cos x\) |
| 3 | \(-\cos x\) | \(\sin x\) |

### 예제 5.1
\(\dfrac{d^7}{dx^7}\sin x=\dfrac{d^3}{dx^3}\sin x\) (주기 4) \(= -\cos x\).

---

## 6. 곱/몫과 삼각 — 실전형

### 6.1 곱: \(x\sin x\)
\[
\frac{d}{dx}(x\sin x)=\sin x+x\cos x.
\]

### 6.2 지수×삼각: \(e^{ax}\sin(bx+c)\)
\[
\frac{d}{dx}\big(e^{ax}\sin(bx+c)\big)
=ae^{ax}\sin(\cdot)+be^{ax}\cos(\cdot)
=e^{ax}\big(a\sin(bx+c)+b\cos(bx+c)\big).
\]
**상대 최대/최소** 탐색, 공진 문제, 해석해의 기본 빌딩블록입니다.

### 6.3 몫: \(\dfrac{\sin x}{x}\) (x≠0)
\[
\left(\frac{\sin x}{x}\right)'=\frac{x\cos x-\sin x}{x^2}.
\]
\(x=0\)에서는 도함수 정의로 극한을 따로 보면 미분가능(연장)합니다.

---

## 7. 역삼각함수(보너스) — 암기/증명 한 번에

- \(\displaystyle \frac{d}{dx}\arcsin x=\frac{1}{\sqrt{1-x^2}}\quad(|x|<1)\)  
  (증명) \(y=\arcsin x\Rightarrow \sin y=x\Rightarrow \cos y\;dy=dx\Rightarrow dy=\frac{dx}{\cos y}=\frac{dx}{\sqrt{1-\sin^2 y}}=\frac{dx}{\sqrt{1-x^2}}\).
- \(\displaystyle \frac{d}{dx}\arccos x=-\frac{1}{\sqrt{1-x^2}}\)  
- \(\displaystyle \frac{d}{dx}\arctan x=\frac{1}{1+x^2}\).

> 구간/부호 주의: \(\arcsin,\arccos\)은 분모가 0되는 \(|x|=1\) 근처에서 경사가 무한대.

---

## 8. 급수·복소지수로 보는 미분 (깊이 이해)

### 8.1 테일러 급수 접근
\[
\sin x=x-\frac{x^3}{3!}+\frac{x^5}{5!}-\cdots
\;\Rightarrow\;
(\sin x)'=1-\frac{x^2}{2!}+\frac{x^4}{4!}-\cdots=\cos x.
\]
\[
\cos x=1-\frac{x^2}{2!}+\frac{x^4}{4!}-\cdots
\;\Rightarrow\;
(\cos x)'=-\sin x.
\]

### 8.2 오일러 공식
\(e^{ix}=\cos x+i\sin x\).  
미분하면 \(i e^{ix}=(-\sin x)+i\cos x\) → 실수부/허수부 비교로 같은 결론.

## 9. 성질·응용 팩트

- **주기 보존**: \(f\)가 주기 \(T\)면 \(f'\)도 같은 주기.  
- **평균 0**: \(\displaystyle \int_{x_0}^{x_0+T} f'(x)\,dx=f(x_0+T)-f(x_0)=0\).  
- **SHM(단순 조화운동)**: \(x(t)=A\sin(\omega t+\phi)\)  
  \(\dot x=A\omega\cos(\cdot)\), \(\ddot x=-A\omega^2\sin(\cdot)=-\omega^2 x\).  
- **파동** \(y(x,t)=A\sin(kx-\omega t)\)  
  \(\partial_t y=-A\omega\cos(\cdot)\), \(\partial_x y=A k\cos(\cdot)\).

## 10. 흔한 실수 & 체크리스트

- [ ] **라디안**인지 확인(도 단위면 \(\pi/180\) 배 필요).  
- [ ] \(\tan,\sec,\cot,\csc\)의 **정의역**(분모 0)에서 미분 불가.  
- [ ] \(\sin^2 x\)는 \((\sin x)^2\) → 연쇄법칙과 곱미분 섞기.  
- [ ] \(\dfrac{d}{dx}\tan x=\sec^2 x\) (가끔 \(\csc^2 x\)로 착각 금지).  
- [ ] 고계 도함수는 “\(+\pi/2\) 위상 이동” 규칙으로 빠르게 점검.

## 11. 예제 세트 (풀이 포함)

### 예제 11.1 (합성)
\[
f(x)=\cos^3(2x-1)=\big(\cos(2x-1)\big)^3.
\]
\[
f'(x)=3\cos^2(2x-1)\cdot(-\sin(2x-1))\cdot 2
=-6\cos^2(2x-1)\sin(2x-1).
\]

### 예제 11.2 (곱)
\[
f(x)=x^2\sin x \;\Rightarrow\; f'(x)=2x\sin x+x^2\cos x.
\]

### 예제 11.3 (몫)
\[
f(x)=\frac{\cos x}{1+\sin x}.
\]
\[
f'=\frac{(-\sin x)(1+\sin x)-\cos x(\cos x)}{(1+\sin x)^2}
=\frac{-\sin x-\sin^2 x-\cos^2 x}{(1+\sin x)^2}
=\frac{-\sin x-1}{(1+\sin x)^2}=-\frac{1}{1+\sin x}.
\]
(깔끔!)

### 예제 11.4 (고계)
\[
\frac{d^6}{dx^6}\cos x=\cos\Big(x+\frac{6\pi}{2}\Big)=\cos(x+3\pi)=-\cos x.
\]

### 예제 11.5 (역삼각)
\[
y=\arctan(3x)\Rightarrow y'=\frac{3}{1+9x^2}.
\]

## 12. 심화 예제: 극값/증가감소/오목볼록

\(f(x)=\sin x-\tfrac12\cos(2x)\).  
- \(f'(x)=\cos x+2\sin x\cos x=\cos x(1+2\sin x)\).  
- 임계점: \(\cos x=0\) 또는 \(\sin x=-\frac12\).  
- \(f''(x)=-\sin x+2\cos(2x)\).  
임계점에 \(f''\) 부호로 극대/극소 판정 → 파동 합성 문제에서 위상관계 해석에 유용.

## 13. 수치·상징 검증 코드(학습용)

> 아래 코드는 **그냥 붙여 실행**하면 됩니다(NumPy/Matplotlib/SymPy).  
> (테마/색 지정 없이 작성)

### 13.1 중앙차분으로 \(\frac{d}{dx}\sin x \approx \cos x\) 확인

```python
import numpy as np

def numerical_derivative(f, x, h=1e-6):
    return (f(x+h) - f(x-h)) / (2*h)

xs = np.linspace(-2*np.pi, 2*np.pi, 9)
approx = numerical_derivative(np.sin, xs)
truth  = np.cos(xs)
for x, a, t in zip(xs, approx, truth):
    print(f"x={x:.3f}, num={a:.8f}, true={t:.8f}, err={abs(a-t):.2e}")
```

### 13.2 그래프: \(f(x)=\sin x\)와 \(f'(x)=\cos x\)

```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2*np.pi, 2*np.pi, 500)
y = np.sin(x)
dy = np.cos(x)

plt.figure()
plt.plot(x, y, label='sin x')
plt.plot(x, dy, linestyle='--', label="(sin x)'=cos x")
plt.axhline(0)
plt.title("sin x와 그 도함수")
plt.legend()
plt.grid(True)
plt.show()
```

### 13.3 SymPy로 합성/곱/몫 자동 미분 확인

```python
import sympy as sp

x = sp.symbols('x')
f1 = sp.sin(5*x**2)
f2 = x**2*sp.sin(x)
f3 = sp.cos(x)/(1+sp.sin(x))

print("d/dx sin(5x^2) =", sp.diff(f1, x).simplify())
print("d/dx x^2 sin x =", sp.diff(f2, x).simplify())
print("d/dx cos x/(1+sin x) =", sp.diff(f3, x).simplify())
```

## 14. 실전 응용 한 컷씩

- **진동계**: \(x(t)=A\cos(\omega t+\phi)\) → \(v=A\omega(-\sin)\), \(a=-\omega^2 x\).  
  주파수 \(\omega\)가 클수록 속도 진폭 \(A\omega\)가 커집니다.  
- **위상-진폭 통합형**: \(c_1\cos x+c_2\sin x=R\cos(x-\delta)\)  
  미분하면 같은 \(R\)에 대해 위상만 \(\pi/2\) 이동.  
- **스펙트럼 미분**: 푸리에 급수 \(f(x)=\sum a_k\cos kx+b_k\sin kx\) →  
  \(f'(x)=\sum (-k a_k)\sin kx + (k b_k)\cos kx\) (계수에 \(k\)가 곱해져 고주파가 강조).

## 15. 연습문제 (간단 해설)

1) \(f(x)=\sin(3x)\cos(2x)\) 의 도함수.  
   *풀이*: 곱미분 + 합성.  
   \(f' = 3\cos(3x)\cos(2x) + \sin(3x)\cdot(-2\sin(2x))\).

2) \(f(x)=\tan(4x+1)\).  
   *풀이*: \(f'=\sec^2(4x+1)\cdot 4\).

3) \(f(x)=\dfrac{\sin x}{1+\cos x}\) 단순화한 도함수.  
   *힌트*: 탄젠트 반각 치환 가능, 혹은 몫미분 → 깔끔한 상수형 나옴.

4) \(y=\arcsin(2x-1)\) 의 도함수와 정의역.  
   *풀이*: \(y'=\dfrac{2}{\sqrt{1-(2x-1)^2}}\), 정의역 \(0\le x\le 1\).

5) \(f^{(7)}(x)=?\) for \(f(x)=\cos x\).  
   *풀이*: 주기 4 → \(7\equiv3\pmod4\) → \(\cos^{(7)}x=\sin x\).
