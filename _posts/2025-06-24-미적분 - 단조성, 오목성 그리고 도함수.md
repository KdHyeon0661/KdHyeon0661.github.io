# 단조성, 오목성 그리고 도함수

단조성(증가/감소)과 오목성/볼록성은 **도함수의 부호**로 빠르게 판정할 수 있다.  
여기서는 1계·2계 도함수와의 관계, 정확한 정의, 반례/주의점, 다변수 확장까지 정리한다.  
(모든 수식은 $$\cdots$$, 코드는 ``` 로 감싼다.)

---

## ✅ 1) 단조성(Monotonicity)과 1계 도함수

### 정의(단변수)
구간 $$I$$ 에서
- **단조 증가(nondecreasing)**: $$x_1<x_2\Rightarrow f(x_1)\le f(x_2).$$  
- **엄격 증가(strictly increasing)**: $$x_1<x_2\Rightarrow f(x_1)<f(x_2).$$  
- **단조 감소/엄격 감소**도 유사.

### 1계 도함수와의 연결(Mean Value Theorem 스케치)
$$f$$ 가 $$I$$ 에서 미분 가능하면:
- $$f'(x)\ge 0\ \forall x\in I\ \Rightarrow\ f$$ **단조 증가**.  
  (평균값정리로 임의의 $$x_1<x_2$$ 에 대해  
  $$\displaystyle \frac{f(x_2)-f(x_1)}{x_2-x_1}=f'(\xi)\ge 0$$)
- $$f'(x)\le 0\ \forall x\Rightarrow f$$ **단조 감소**.
- $$f'(x)=0\ \forall x\Rightarrow f$$ **상수**.

> **엄격 증가** 보장: (i) $$f'(x)>0\ \forall x$$ 이거나,  
> (ii) $$f'(x)\ge 0$$ 이고 어떤 부분구간에서도 $$f'$$ 가 **항등적으로 0이 아님**.

### 빠른 예제
- $$f(x)=x^3:\ f'(x)=3x^2\ge 0$$ → **단조 증가**(단, $$x=0$$ 에서 $$f'=0$$ 이나 전체로는 증가).  
- $$f(x)=e^x:\ f'(x)=e^x>0$$ → **엄격 증가**.  
- $$f(x)=-\ln x\ (x>0):\ f'(x)=-1/x<0$$ → **엄격 감소**.

> **주의**: $$f'(c)=0$$ 이라도 극값이 아닐 수 있음 (예: $$x^3$$ 은 변곡점).  
> 판정에는 **부호 변화** 또는 **2계 도함수**가 필요.

## ✅ 2) 오목성/볼록성(Concavity/Convexity)과 2계 도함수

### 기하적 정의(접선 비교/현수선)
- **볼록(convex)**: 임의의 $$x_1,x_2$$, $$t\in[0,1]$$ 에 대해  
  $$f\big(tx_1+(1-t)x_2\big)\le tf(x_1)+(1-t)f(x_2).$$  
  (그래프가 **위로 볼록**; 접선이 아래에 깔림)
- **오목(concave)**: 부등호 반대. (접선이 위에 놓임)

### 2계 도함수 판정(두 번 미분 가능 가정)
- $$f''(x)>0\Rightarrow f$$ **볼록**.  
- $$f''(x)<0\Rightarrow f$$ **오목**.  
- $$f''(x)=0$$ 은 **변곡점 후보**일 뿐, 실제 변곡점이려면 **오목/볼록이 바뀌어야** 함.

### 예제
- $$f(x)=x^2:\ f''=2>0$$ → 항상 **볼록**.  
- $$f(x)=-x^2:\ f''=-2<0$$ → 항상 **오목**.  
- $$f(x)=x^3:\ f''=6x.$$  
  $$x<0$$ 오목, $$x>0$$ 볼록, $$x=0$$ 에서 **변곡점**(오목성 변화 확인).

> **정확성**: 2계 미분이 없더라도 볼록/오목은 **정의**로 판정 가능.  
> 2계 판정은 매끄러울 때의 **충분조건**(및 일반적으로 필요충분).

## ✅ 3) 1·2계 도함수로 형태 분석(종합 예)

함수 $$f(x)=x^3-3x$$
- 1계: $$f'(x)=3x^2-3=3(x^2-1)=0\iff x=\pm1.$$
  - 구간 부호:  
    $$x<-1:\ f'>0\ (\uparrow)\quad -1<x<1:\ f'<0\ (\downarrow)\quad x>1:\ f'>0\ (\uparrow)$$  
    ⇒ 극대 at $$x=-1$$, 극소 at $$x=1$$.
- 2계: $$f''(x)=6x$$ → $$x=0$$ 에서 변곡점(오목↔볼록 전환).

---

## ✅ 4) 1차/2차 판정 요약

- **1차 판정(부호 변화)**:  
  임계점 $$c$$ 근방에서  
  - $$f'$$ 가 **+→-** 로 바뀌면 **극댓값**  
  - **-→+** 로 바뀌면 **극솟값**  
  - 같은 부호면 **극값 아님**
- **2차 판정**: $$f'(c)=0$$ 일 때  
  - $$f''(c)>0$$ ⇒ **극솟값**  
  - $$f''(c)<0$$ ⇒ **극댓값**  
  - $$f''(c)=0$$ ⇒ **불충분**(고차 도함수/그래프/1차판정 사용)

---

## ✅ 5) 변곡점(Inflection Point)

- 정의: **오목성(부호)이 실제로 바뀌는** 점.  
- **필요조건 아님**: $$f''(c)=0$$ 은 흔한 신호지만,  
  $$f''(c)=0$$ 이어도 변곡점이 아닐 수 있고(예: $$x^4$$),  
  $$f''$$ 가 존재하지 않아도 변곡점일 수 있다(예: $$x^{1/3}$$).

---

## ✅ 6) 다변수 확장(개요)

함수 $$f:\mathbb{R}^n\to\mathbb{R}$$, 헤시안 $$H(x)=\nabla^2 f(x)$$.
- **볼록성**: 모든 방향으로 2차형이 비음이 아니면(양의 준정부호) **볼록**.  
  **엄격 볼록**: 양의 정부호.  
- **오목성**: 헤시안이 음의 준정부호/정부호.
- **최적화 결과**: **볼록 최적화**에서는 **국소 최소 = 전역 최소**.

## ✅ 7) 자주 하는 실수/주의

- $$f'(c)=0$$ 이면 무조건 극값? → ❌ (예: $$x^3$$).  
- $$f''(c)=0$$ 이면 변곡점? → ❌ (오목성 변화 확인 필수).  
- 단조 판정에서 미분 불능점(절댓값 등)도 흐름 바꿀 수 있으니 체크.  
- 정의역/경계 포함 여부(닫힌 구간이면 **끝점 비교** 필수).

## ✅ 9) 추천 시각화 코드 (Python / matplotlib)

```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3, 3, 500)
f  = x**3 - 3*x
fp = 3*x**2 - 3
fpp= 6*x

plt.figure(figsize=(10,6))
plt.plot(x, f, label='f(x) = x^3 - 3x')
plt.plot(x, fp, label="f'(x)")
plt.plot(x, fpp, label="f''(x)")
plt.axhline(0, linewidth=0.7)
plt.axvline(-1, linestyle='--', alpha=0.6)
plt.axvline( 1, linestyle='--', alpha=0.6)
plt.axvline( 0, linestyle='--', alpha=0.6)
plt.legend()
plt.title('도함수를 통한 단조성/오목성 분석')
plt.grid(True)
plt.show()
```

## ✅ 10) 한눈 요약 표

| 성질 | 판정(미분 가능 가정) | 해석 |
|---|---|---|
| 증가 | $$f'(x)>0$$ | 오른쪽으로 갈수록 값 ↑ |
| 감소 | $$f'(x)<0$$ | 오른쪽으로 갈수록 값 ↓ |
| 볼록 | $$f''(x)>0$$ | 접선이 아래(그래프가 위) |
| 오목 | $$f''(x)<0$$ | 접선이 위(그래프가 아래) |
| 변곡점 | 오목↔볼록 실제 전환 | 형태가 바뀌는 지점 |

---

### 마무리
- **1계 도함수**는 **방향(증가/감소)**, **2계 도함수**는 **굽힘(오목/볼록)** 을 말해 준다.  
- 실제 문제에서는 **부호표 → 극값/변곡 판단 → 경계 비교**의 루틴을 습관화하면 실수를 크게 줄일 수 있다.
