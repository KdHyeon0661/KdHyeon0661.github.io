# 실수와 직선 (확장 개정판)

## 0. 한눈에 보는 로드맵

- **실수의 세계**: 집합 $$\mathbb{R}$$, 순서/절댓값/거리, 구간, **밀도**, **완비성(최소상한 성질)**, **Archimedean 성질**  
- **수직선**: 실수 ↔ 점 **일대일 대응**, 스케일/단위, 좌표 눈금  
- **직선의 표현**:  
  - 기울기-절편형 $$y=mx+b$$  
  - 점-기울기형 $$y-y_0=m(x-x_0)$$  
  - 두 점형 $$y-y_1=\frac{y_2-y_1}{x_2-x_1}(x-x_1)$$  
  - 일반형/법선형 $$Ax+By+C=0 \;\Leftrightarrow\; \mathbf{n}\cdot\mathbf{r}+C=0$$  
  - 벡터/매개표현 $$\mathbf{r}(t)=\mathbf{r}_0+t\mathbf{v}$$
- **미적분 관점**: 평균변화율 ↔ 순간변화율(미분), 직선 아래 면적(적분), 접선 근사(선형화)  
- **심화**: 거리/사영/각, 평행·수직, 리만합으로 본 직선의 적분, $$\varepsilon\text{–}\delta$$ 연속성, Lipschitz  
- **오해/함정**과 **연습문제(풀이 포함)**

## 1. 실수란 무엇인가?

실수 $$\mathbb{R}$$ 는 **수직선 위의 모든 점**에 대응되는 수의 집합입니다. 자연수/정수/유리수/무리수를 모두 포함합니다.

### 1.1 구성: 자연수 → 정수 → 유리수 → 실수
- **자연수**: $$1,2,3,\dots$$  
- **정수**: $$\dots,-2,-1,0,1,2,\dots$$  
- **유리수**: $$\frac{p}{q}$$ (단, $$p,q\in\mathbb{Z},\,q\neq 0$$). **유한/순환소수**.  
- **무리수**: **비순환 무한소수**. 예: $$\sqrt{2},\,\pi,\,e$$  
- **실수**: 유리수 $$\cup$$ 무리수 = 수직선 전체.

> **예제 1.1**  
> (1) $$0.125=\frac18\in\mathbb{Q}$$ (유리수).  
> (2) $$0.\overline{142857}=\frac{1}{7}\in\mathbb{Q}$$ (순환소수).  
> (3) $$\sqrt{2}\notin\mathbb{Q}$$ (무리수 — 피타고라스의 증명).

### 1.2 순서/절댓값/거리
- **전순서**: 임의의 $$a,b\in\mathbb{R}$$ 에 대해 정확히 하나: $$a<b$$, $$a=b$$, $$a>b$$.  
- **절댓값**: $$|x|=\begin{cases}x,&x\ge0\\-x,&x<0\end{cases}$$ (원점까지의 거리)  
- **거리함수(메트릭)**: $$d(x,y)=|x-y|$$

**거리의 공리 확인** (모든 $$x,y,z\in\mathbb{R}$$):
1. **양의정치성**: $$d(x,y)\ge 0$$, 평등성 $$d(x,y)=0\Leftrightarrow x=y$$  
2. **대칭성**: $$d(x,y)=d(y,x)$$  
3. **삼각부등식**: $$d(x,z)\le d(x,y)+d(y,z)$$

> **증명 스케치(삼각부등식)**  
> $$|x-z|=|x-y+y-z|\le|x-y|+|y-z|$$ (절댓값의 기본 성질).

### 1.3 구간 표기와 집합 기호
- 닫힌: $$[a,b]=\{x\mid a\le x\le b\}$$  
- 열린: $$(a,b)=\{x\mid a<x<b\}$$  
- 반열림: $$[a,b),\;(a,b]$$  
- 전실수: $$\mathbb{R}=(-\infty,\infty)$$

> **예제 1.2**  
> 구간 $[1,4)$의 길이는 $$4-1=3$$, 끝점 4는 포함되지 않음.

### 1.4 밀도와 완비성
- **밀도(density)**: $$a<b$$ 이면, **유리수** $$q$$ 와 **무리수** $$\alpha$$ 가 항상 존재하여 $$a<q<b$$, $$a<\alpha<b$$.  
  - (유리수 존재 증명 스케치) Archimedean 성질로 $$n(b-a)>1$$ 되게 하는 $$n\in\mathbb{N}$$ 선택. 정수 $$k$$ 가 $$na<k<nb$$ 를 만족하도록 택하면 $$q=\frac{k}{n}$$ 가 중간에 존재.  
  - (무리수 존재) 위에서 얻은 $$q$$ 에 대해 $$q+\frac{b-a}{2\sqrt{2}}$$ 가 여전히 $$(a,b)$$ 안에 들도록 조정하면 비순환소수(무리수)가 하나 나온다.
- **완비성(최소상한 성질)**: $$\emptyset\neq S\subset\mathbb{R}$$ 가 위로 유계이면, **최소상한** $$\sup S$$ 가 존재.

> **예제 1.3 (최소상한)**  
> $$S=\{x\in\mathbb{R}\mid x^2<2\}=( -\sqrt2,\sqrt2 )$$ 는 위로 유계(상계=2 등). 이때 $$\sup S=\sqrt2$$.

> **직관**: 유리수만으로는 $$x^2<2$$ 의 상한을 “유리수로” 딱 잡아줄 수 없다. 실수는 그런 빈틈을 막아준다(연속적 수직선).

### 1.5 Archimedean 성질
- 임의의 $$x\in\mathbb{R}$$ 에 대해, $$n>x$$ 인 자연수 $$n$$ 존재.  
- 임의의 $$\varepsilon>0$$ 에 대해, $$\frac{1}{n}<\varepsilon$$ 인 자연수 $$n$$ 존재.

> **용도**: 극한/근사, **유리수의 밀도** 증명, $$\varepsilon\text{–}\delta$$ 논증에서 “충분히 큰/작은” 정수의 선택.

## 2. 수직선과 좌표축

### 2.1 실수 ↔ 점 (일대일 대응)
각 실수 $$x$$ 는 수직선 위 **한 점**에, 반대로 각 점은 **정확히 하나의 실수**에 대응.

- 원점 0, 단위길이 1 지정 → 눈금(스케일) 형성  
- 단위/스케일 변경은 **선형 사상** $$x\mapsto ax+b$$ 로 모델링

> **예제 2.1 (스케일 변환)**  
> 섭씨–화씨 변환 $$F=1.8C+32$$ 는 **직선**. 기울기 1.8, 절편 32.

### 2.2 좌표평면에서의 함수
함수 $$y=f(x)$$ 의 그래프는 점들의 집합 $$\{(x,f(x))\mid x\in\mathbb{R}\}$$.

- **증가/감소**는 기하적으로 **오르막/내리막**  
- 연속성은 **펜을 떼지 않고 그릴 수 있는가**의 정량화(정의는 $$\varepsilon\text{–}\delta$$)

## 3. 직선의 여러 표현

### 3.1 기울기-절편형
$$
y=mx+b
$$
- $$m$$: **기울기**(변화율/단위당 변화)  
- $$b$$: **y-절편**(원점에서의 함수값)

> **예제 3.1**  
> $$y=2x+1$$ — 기울기 2, 절편 1. $$x\to x+1$$ 때 $$y\to y+2$$.

### 3.2 점-기울기형
$$
y-y_0=m(x-x_0)
$$
점 $$(x_0,y_0)$$ 을 지나고 기울기 $$m$$ 인 모든 직선.

### 3.3 두 점형
$$
y-y_1=\frac{y_2-y_1}{x_2-x_1}(x-x_1)\quad (x_1\neq x_2)
$$

> **예제 3.2 (두 점 → 직선)**  
> $$(1,2),(4,8)$$ → $$m=\frac{8-2}{4-1}=2$$, $$y-2=2(x-1)\Rightarrow y=2x$$.

### 3.4 일반형/법선형
$$
Ax+By+C=0\quad (A,B\text{ 동시에 }0\text{ 아님})
$$
- **법선**: $$\mathbf{n}=(A,B)$$  
- **법선형**: $$\mathbf{n}\cdot\mathbf{r}+C=0$$, 여기서 $$\mathbf{r}=(x,y)$$  
- **점–법선 표현**: 점 $$\mathbf{r}_0$$ 을 지나고 법선 $$\mathbf{n}$$ 이면 $$\mathbf{n}\cdot(\mathbf{r}-\mathbf{r}_0)=0$$

### 3.5 벡터/매개표현
$$
\mathbf{r}(t)=\mathbf{r}_0+t\,\mathbf{v}=\langle x_0,y_0\rangle+t\langle a,b\rangle,\quad t\in\mathbb{R}
$$
- **방향벡터** $$\mathbf{v}=\langle a,b\rangle\neq\mathbf{0}$$  
- **속도**(미분) $$\mathbf{r}'(t)=\mathbf{v}$$: 직선은 **등속 운동**.

### 3.6 기울기와 각
$$
m=\tan\theta
$$
- 수평선: $$m=0$$  
- 수직선: 기울기 미정의(함수 그래프로 표현 불가)

## 4. 직선과 미적분의 연결

### 4.1 평균변화율과 순간변화율(미분)
$$
f(x)=mx+b\quad\Rightarrow\quad f'(x)=m\;\;(\text{모든 }x)
$$

- 평균변화율:
  $$
  \frac{f(x+h)-f(x)}{h}=\frac{m(x+h)+b-(mx+b)}{h}=m
  $$
- 순간변화율(극한):
  $$
  \lim_{h\to0}\frac{f(x+h)-f(x)}{h}=m
  $$

> **해석**: 직선에서 **모든 구간의 시컨트** 기울기 = **모든 점의 접선** 기울기 = $$m$$.  
> **평균=순간** → 평균·순간변화율의 차이를 가장 단순하게 보여주는 모델.

### 4.2 적분과 면적
$$
\int_a^c(mx+b)\,dx=\left[\frac{m}{2}x^2+bx\right]_a^c=\frac{m}{2}(c^2-a^2)+b(c-a)
$$

- 기하학: **사다리꼴** 넓이와 일치  
  $$
  \frac{(ma+b+mc+b)}{2}\cdot(c-a)
  $$

> **예제 4.1 (정적분)**  
> $$\int_{0}^{3}(2x+1)\,dx=[x^2+x]_0^3=9+3=12.$$

### 4.3 리만합으로 다시 보기 (직선의 적분)
균등분할 $$x_k=a+k\Delta x,\;\Delta x=\frac{c-a}{n}$$:
$$
\sum_{k=1}^n (m\,x_k+b)\,\Delta x
= m\sum_{k=1}^n x_k\,\Delta x+b(c-a)
$$
여기서 $$\sum x_k$$ 를 등차수열 합으로 정리하고 $$n\to\infty$$ 극한을 취하면 위 정적분 결과와 일치.

> **직관**: 직선은 구간 어디서나 같은 기울기 → **리만합의 수렴이 가장 투명**.

### 4.4 선형화(접선 근사)
일반 함수 $$g(x)$$ 에서 $$x=a$$ 근방:
$$
g(x)\approx g(a)+g'(a)(x-a)
$$
- 직선 $$y=mx+b$$ 의 경우 **정확히 일치**(근사가 아님).  
- 복잡한 곡선도 **국소적으로 ‘직선처럼’** 다루는 발판.

## 5. 계산·시각화 예제

### 5.1 기울기/절편/두 점
1) **기울기 해석**  
$$y=-3x+4$$ → 기울기 $$-3$$, 절편 $$4$$. $$x\!\uparrow$$ 1 ⇒ $$y\!\downarrow$$ 3.

2) **두 점을 지나는 직선**  
$$(1,2),(4,8)\Rightarrow m=\frac{6}{3}=2,\; y-2=2(x-1)\Rightarrow y=2x.$$

3) **평행·수직**  
- $$y=2x+1$$ 와 평행: $$y=2x+b$$ (임의의 $$b$$).  
- $$y=2x+1$$ 와 수직: $$y=-\tfrac12 x+b$$ (기울기 곱 $$=-1$$).

### 5.2 점–직선 거리와 사영
- 거리(일반형 $$Ax+By+C=0$$, 점 $$(x_0,y_0)$$):
  $$
  \mathrm{dist}=\frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}
  $$
- 사영:
  $$
  \mathrm{proj}_{\mathbf{v}}(\mathbf{u})
  =\frac{\mathbf{u}\cdot\mathbf{v}}{\mathbf{v}\cdot\mathbf{v}}\,\mathbf{v}
  $$

> **예제 5.1 (점–직선 거리)**  
> 점 $$(3,4)$$, 직선 $$3x-4y+5=0$$:  
> 거리 $$=\frac{|3\cdot3-4\cdot4+5|}{\sqrt{3^2+(-4)^2}}=\frac{|9-16+5|}{5}=\frac{2}{5}$$.

> **예제 5.2 (사영)**  
> $$\mathbf{u}=\langle2,5\rangle, \mathbf{v}=\langle1,1\rangle$$  
> $$\mathbf{u}\cdot\mathbf{v}=7,\;\mathbf{v}\cdot\mathbf{v}=2$$  
> $$\Rightarrow \mathrm{proj}_{\mathbf{v}}(\mathbf{u})=\frac{7}{2}\langle1,1\rangle=\left\langle\frac72,\frac72\right\rangle.$$

### 5.3 파이썬 코드 (수정·실행 예시)

> **주의**: 이 코드는 예시입니다. 환경에 맞춰 실행하세요.

```python
# 1) 직선 y = m x + b의 평균/순간변화율 확인
import numpy as np

m, b = 2.0, 1.0
def f(x): return m*x + b

x0 = 3.0
for h in [1.0, 0.1, 1e-2, 1e-3, 1e-6]:
    diffq = (f(x0+h)-f(x0))/h
    print(h, diffq)  # 항상 2.0 근처(정확히 2.0)
```

```python
# 2) 리만합으로 ∫_a^c (m x + b) dx 수치 검증
m, b = 2.0, 1.0
a, c = 0.0, 3.0

def integral_closed(m, b, a, c):
    return 0.5*m*(c**2 - a**2) + b*(c-a)

def riemann_sum(m, b, a, c, n=100000):
    xs = np.linspace(a, c, n+1)[1:]  # 오른쪽 리만합
    dx = (c-a)/n
    return np.sum(m*xs + b) * dx

print("Closed form:", integral_closed(m, b, a, c))
print("Riemann n=1e5:", riemann_sum(m, b, a, c, 100000))
```

```python
# 3) 점-직선 거리 함수
import math
def point_line_distance(A,B,C,x0,y0):
    return abs(A*x0 + B*y0 + C) / math.sqrt(A*A + B*B)

print(point_line_distance(3,-4,5,3,4))  # 0.4
```

## 6. 직선의 기하 심화

### 6.1 두 점 사이 거리
$$
\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}
$$

### 6.2 점–직선 거리(유도)
법선벡터 $$\mathbf{n}=(A,B)$$, 점 $$\mathbf{r}_0=(x_0,y_0)$$, 직선 위 점 $$\mathbf{r}=(x,y)$$.  
직선: $$\mathbf{n}\cdot\mathbf{r}+C=0$$.  
거리 = **법선 방향**으로 본 투영 길이:
$$
\frac{|\,\mathbf{n}\cdot\mathbf{r}_0+C\,|}{\|\mathbf{n}\|}=\frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}.
$$

### 6.3 평행/수직의 조건
- 평행: $$m_1=m_2$$ (방향벡터 **평행**)  
- 수직: $$m_1 m_2=-1$$ (방향벡터 **직교**)  
- 수직선/수평선 조합은 별도 처리(수직선은 기울기 미정의).

### 6.4 각도
직선 $$m_1,m_2$$ 의 끼인 각 $$\phi$$:
$$
\tan\phi=\left|\frac{m_2-m_1}{1+m_1 m_2}\right|
$$

## 7. 직선에서의 해석학(연속/미분/적분)

### 7.1 연속성의 $$\varepsilon\text{–}\delta$$ 정의 (직선의 경우)
$$
f(x)=mx+b
$$
임의의 $$\varepsilon>0$$ 에 대해 $$\delta=\frac{\varepsilon}{|m|}$$ (단, $$m=0$$ 이면 아무 양의 $$\delta$$ 가능) 을 택하면  
$$|x-a|<\delta\Rightarrow |f(x)-f(a)|=|m||x-a|<\varepsilon.$$

> **결론**: 모든 직선은 **전 구간에서 연속**.

### 7.2 미분가능성과 Lipschitz
- $$f'(x)=m$$ (모든 $$x$$) — 전 구간 **미분가능**  
- **Lipschitz**: $$|f(x)-f(y)|\le |m|\,|x-y|$$ — 상수 $$|m|$$

### 7.3 적분과 부호
- 정적분은 **부호를 가진 면적**: 그래프가 $$x$$-축 아래면 음수 가능  
- **기하학적 면적**을 원하면 구간을 나누어 절댓값을 취함

> **예제 7.1**  
> $$\int_{-1}^{2}(2x+1)\,dx = [x^2+x]_{-1}^{2} = (4+2) - (1-1) = 6.$$

## 8. 실전 상황으로 보는 직선 모델

### 8.1 평균속도/순간속도
- 일정 속력 $$v$$: 거리 $$s(t)=v t + s_0$$ → **직선**.  
- 평균속도 = 순간속도 = $$v$$.

### 8.2 환율/단위 변환
- 섭씨–화씨, 환율(수수료 무시), 픽셀→센티미터 등: 대부분 $$y=ax+b$$.

### 8.3 물탱크 채움(이상화)
- 일정 유량 $$Q$$ 로 시간 $$t$$ 동안: 체적 $$V(t)=Qt+V_0$$ (선형 증가).  
  - 실제: 초기/말기 압력 변화 → 선형에서 벗어나지만 **국소적으로는 선형 근사**.

## 9. 더 깊게: 선형 대수/최적화 연결

### 9.1 최소제곱 직선(간단 스케치)
데이터 $$(x_i,y_i)$$ 에 가장 잘 맞는 $$y=mx+b$$ 를 찾는 문제:
$$
\min_{m,b}\sum_{i}(y_i - mx_i - b)^2
$$
- **해**: 정규방정식(선형 시스템)으로 도출, 해석해를 갖음(공식 유도 가능).  
- **연결**: 벡터 사영(열공간에의 직교사영) — “자료를 직선에 투영”.

> **메시지**: 직선은 **근사/예측의 1차 모델**로 핵심.

## 10. 자주 하는 오해/함정

1) **수직선에 ‘틈’이 있다?** → 실수는 **완비**라서 틈이 없다.  
2) **수직선의 기울기?** → 수직선은 함수 그래프가 아님(기울기를 수로 표현 불가).  
3) **적분=항상 면적?** → 정적분은 **부호가 있는 면적**. 면적 문제면 절댓값/구간분할 필요.  
4) **평균변화율=순간변화율?** → 일반 함수는 다르다. 직선이 **특별히 일치**할 뿐.  
5) **평행/수직 결정 시 절편 비교** → 평행 여부는 **기울기**로, 절편은 무관.

## 12. 연습문제 해설

1. **해설**: $$a<b$$. Archimedean 성질로 $$n(b-a)>1$$ 인 $$n\in\mathbb{N}$$ 존재. 정수 $$k$$ 를 $$na<k<nb$$ 만족하도록 택하면 $$q=\frac{k}{n}$$ 가 $$a<q<b$$ 인 유리수.  
2. **해설**: $$S=(-\sqrt3,\sqrt3)$$, 위로 유계. **최소상한** $$\sup S=\sqrt3$$.  
3. **해설**: $$f(x)=mx+b$$, 임의 $$\varepsilon>0$$ 에 대해 $$\delta=\varepsilon/|m|$$ (단, $$m=0$$ 이면 임의의 양의 $$\delta$$). 그러면 $$|x-a|<\delta\Rightarrow |f(x)-f(a)|=|m||x-a|<\varepsilon$$.

4. **해설**: 점-기울기형: $$y-3=-4(x-2)$$. 정리: $$y=-4x+11$$.  
5. **해설**: 기울기 $$m=\frac{-1-5}{7-1}=\frac{-6}{6}=-1$$. 직선: $$y-5=-1(x-1)\Rightarrow y=-x+6$$.  
6. **해설**: 거리 $$=\frac{|3\cdot3-4\cdot4+5|}{\sqrt{3^2+(-4)^2}}=\frac{2}{5}$$.

7. **해설**: 평균변화율 $$\frac{f(x+h)-f(x)}{h}=\frac{3(x+h)-2-(3x-2)}{h}=3=f'(x)$$.  
8. **해설**: 적분값 $$[x^2+x]_{-1}^{2}=(4+2)-(1-1)=6$$. 사다리꼴: 밑변 길이 $$3$$, 윗변 높이 $$f(-1)=-1$$, $$f(2)=5$$ → 평균 높이 $$2$$, 넓이 $$2\cdot3=6$$.  
9. **해설**: 균등분할 $$x_k=a+k\Delta x$$, $$\Delta x=\frac{c-a}{n}$$, 오른쪽 리만합
   $$
   \sum_{k=1}^{n}(m x_k + b)\Delta x
   = m\Delta x \sum_{k=1}^n x_k + b(c-a)
   $$
   $$\sum x_k = \sum (a+k\Delta x)=na+\Delta x\frac{n(n+1)}{2}$$. 정리 후 $$n\to\infty$$, 결과 $$\frac{m}{2}(c^2-a^2)+b(c-a)$$.

10. **해설**: 수직 조건 → 기울기 $$-\tfrac12$$. 점 $$(2,0)$$ 통과: $$y-0=-\tfrac12(x-2)\Rightarrow y=-\tfrac12 x +1$$.  
11. **해설**: $$m_1=3,\,m_2=-\tfrac13$$.  
    $$
    \tan\phi=\left|\frac{m_2-m_1}{1+m_1m_2}\right|=\left|\frac{-\tfrac13-3}{1-1}\right|
    $$
    분모 0 → $$\phi=\frac{\pi}{2}$$ (직교).  
12. **해설**: $$|f(x)-f(y)|=|m||x-y|\le L|x-y|$$ 에서 $$L=|m|$$.

   - **과제**: 임의의 시간 간격에서 평균속도 계산 → 항상 $$v$$.  
3) **균일 요금제**: 기본료 $$b$$ + 사용량당 요금 $$m$$ → 비용 $$C(x)=mx+b$$  
   - **과제**: 사용량 2배 → 비용은 정확히 $$2m$$ 만큼 증가(절편은 고정).

### 13.2 코드 스니펫 (테스트/검증)
```python
# 직선 회귀가 아닌 '검증'에 초점: 평행/수직/거리 계산 유틸
import math

def slope_two_points(x1,y1,x2,y2):
    if x1==x2: return None  # 수직선
    return (y2-y1)/(x2-x1)

def is_parallel(m1, m2):
    return (m1 is None and m2 is None) or (m1 is not None and m2 is not None and abs(m1-m2)<1e-12)

def is_perpendicular(m1, m2):
    if m1 is None: return m2==0.0
    if m2 is None: return m1==0.0
    return abs(m1*m2 + 1.0) < 1e-12

def line_point_distance(A,B,C,x0,y0):
    return abs(A*x0+B*y0+C)/math.sqrt(A*A+B*B)

print(slope_two_points(1,2,4,8))  # 2.0
print(is_parallel(2.0,2.0))       # True
print(is_perpendicular(2.0,-0.5)) # True
print(line_point_distance(3,-4,5,3,4)) # 0.4
```

## 15. 마무리

- **실수의 완비성**은 수직선에 “틈이 없음”을 보장, 극한/연속/적분의 기반을 제공합니다.  
- **직선**은 “변화율=상수”라는 가장 단순하지만 강력한 모델: 평균변화율과 순간변화율이 일치, 적분은 정확한 사다리꼴.  
- **선형화**는 복잡한 곡선을 국소적으로 직선으로 다루는 핵심 아이디어.  
- 이후 다항/지수/삼각함수, 다변수/벡터미적분에서도 직선(접선/접평면)은 **첫 번째 근사**로 계속 등장합니다.
