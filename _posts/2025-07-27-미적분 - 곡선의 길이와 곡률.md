# 곡선의 길이와 곡률

> 핵심만 먼저  
> - **길이**: $$L=\displaystyle\int_a^b \|\mathbf r'(t)\|\,dt$$  
> - **곡률(3D)**: $$\displaystyle \kappa(t)=\frac{\|\mathbf r'(t)\times \mathbf r''(t)\|}{\|\mathbf r'(t)\|^3}$$  
> - **곡률(2D 매개)**: $$\displaystyle \kappa(t)=\frac{|x'y''-y'x''|}{(x'^2+y'^2)^{3/2}}$$  
> - **곡률(함수 \(y=f(x)\))**: $$\displaystyle \kappa(x)=\frac{|f''(x)|}{\bigl(1+(f'(x))^2\bigr)^{3/2}}$$  
> - **접촉원(오스큘레이팅 원)**: 반지름 $$\rho=1/\kappa$$, 중심 $$C=\mathbf r+\rho\,\mathbf N$$

---

## 1) 곡선의 길이 (Arc Length)

### 1.1 정의 (일반 매개화)
$$
\mathbf r(t)=\langle x(t),y(t),z(t)\rangle,\quad t\in[a,b]
$$
$$
L=\int_a^b \|\mathbf r'(t)\|\,dt
=\int_a^b \sqrt{(x')^2+(y')^2+(z')^2}\,dt.
$$

### 1.2 2차원·특수 형태
- 2D 매개: $$\displaystyle L=\int_a^b \sqrt{(x')^2+(y')^2}\,dt.$$
- 함수 \(y=f(x)\): $$\displaystyle L=\int_{x_0}^{x_1}\sqrt{1+(f'(x))^2}\,dx.$$
- 극좌표 \(r=r(\theta)\): $$\displaystyle L=\int_{\theta_0}^{\theta_1}\sqrt{r^2+\Bigl(\frac{dr}{d\theta}\Bigr)^2}\,d\theta.$$

### 1.3 예제 A — 포물선 \(\mathbf r(t)=\langle t,t^2\rangle,\,0\le t\le1\)
$$
\mathbf r'(t)=\langle 1,2t\rangle,\quad \|\mathbf r'(t)\|=\sqrt{1+4t^2}.
$$
$$
L=\int_0^1 \sqrt{1+4t^2}\,dt
=\left[\frac{t}{2}\sqrt{1+4t^2}+\frac{1}{4}\,\operatorname{asinh}(2t)\right]_0^1
=\frac{\sqrt5}{2}+\frac{1}{4}\operatorname{asinh}(2).
$$

### 1.4 예제 B — 원의 1/4호 \(x=a\cos\theta,\,y=a\sin\theta,\ \theta\in[0,\tfrac\pi2]\)
$$
\|\mathbf r'(\theta)\|=a,\quad L=\int_0^{\pi/2} a\,d\theta=\frac{\pi a}{2}.
$$

## 2) 곡률 (Curvature)

### 2.1 물리적·기하적 의미
- 곡률 \(\kappa\): **단위 길이당 방향 변화율**.  
- 평면에서는 접선각 \(\theta\)로 $$\displaystyle \kappa=\frac{d\theta}{ds}$$ (여기서 \(s\)는 호의 길이).

### 2.2 공식 모음
- 3D(일반): 
  $$
  \kappa(t)=\frac{\|\mathbf r'(t)\times\mathbf r''(t)\|}{\|\mathbf r'(t)\|^3}.
  $$
- 2D 매개:
  $$
  \kappa(t)=\frac{|x'y''-y'x''|}{\bigl((x')^2+(y')^2\bigr)^{3/2}}.
  $$
- 함수 \(y=f(x)\):
  $$
  \kappa(x)=\frac{|f''(x)|}{\bigl(1+(f'(x))^2\bigr)^{3/2}}.
  $$
- 극좌표 \(r=r(\theta)\) (평면):
  $$
  \kappa(\theta)=\frac{\big|\,r^2+2(r')^2-r\,r''\,\big|}{\bigl(r^2+(r')^2\bigr)^{3/2}},\quad {}'=\tfrac{d}{d\theta}.
  $$

### 2.3 예제 C — 원 \(x^2+y^2=R^2\)
상수 곡률:
$$
\kappa=\frac{1}{R},\qquad \rho=\frac{1}{\kappa}=R.
$$

### 2.4 예제 D — 포물선 \(y=x^2\)
$$
f'(x)=2x,\ f''(x)=2
\Rightarrow \kappa(x)=\frac{2}{\bigl(1+4x^2\bigr)^{3/2}},\ \rho(x)=\frac{(1+4x^2)^{3/2}}{2}.
$$
특히 \(x=0\)에서 \(\kappa(0)=2\), \(\rho(0)=\tfrac12\).

### 2.5 예제 E — 나선(헬릭스) \(\mathbf r(t)=\langle a\cos t,\,a\sin t,\,bt\rangle\)
$$
\|\mathbf r'(t)\|=\sqrt{a^2+b^2},\quad 
\mathbf r''(t)=\langle -a\cos t,-a\sin t,0\rangle.
$$
$$
\|\mathbf r'\times\mathbf r''\|=a\sqrt{a^2+b^2}\quad\Rightarrow\quad
\kappa=\frac{a}{a^2+b^2}\quad(\text{상수}).
$$
(참고: 비틀림 $$\tau=\dfrac{b}{a^2+b^2}$$.)

### 2.6 예제 F — 사이클로이드 \(x=a(\theta-\sin\theta),\,y=a(1-\cos\theta)\)
$$
x'=a(1-\cos\theta),\ y'=a\sin\theta,\ x''=a\sin\theta,\ y''=a\cos\theta.
$$
$$
\kappa(\theta)
=\frac{a^2|x'y''-y'x''|}{\bigl(a^2((1-\cos\theta)^2+\sin^2\theta)\bigr)^{3/2}}
=\frac{1}{4a\,|\sin(\theta/2)|}.
$$
끝점(첨점) \(\theta=0,2\pi\)에서 \(|\sin(\theta/2)|=0\Rightarrow \kappa\to\infty\) (첨점에서 곡률 무한).

### 2.7 예제 G — 로그 나선 \(r(\theta)=ae^{b\theta}\) (극좌표)
\(r'=br,\ r''=b^2 r\) 대입:
$$
\kappa(\theta)=\frac{1}{r\sqrt{1+b^2}}=\frac{1}{a\,e^{b\theta}\sqrt{1+b^2}}.
$$
(곡률이 지수적으로 감소 → 바깥으로 갈수록 완만.)

## 3) 프레네–세레(Frenet–Serret) 체계 한눈에
- 단위 접선 $$\displaystyle \mathbf T=\frac{\mathbf r'}{\|\mathbf r'\|}$$  
- 법선 \(\mathbf N\): \(\mathbf T'\) 방향  
- 이변량 $$\mathbf B=\mathbf T\times\mathbf N$$  
- 호길이 매개 \(s\)에서
$$
\mathbf T'=\kappa\,\mathbf N,\quad
\mathbf N'=-\kappa\,\mathbf T+\tau\,\mathbf B,\quad
\mathbf B'=-\tau\,\mathbf N.
$$
- **접촉원(오스큘레이팅 원)**: 반지름 \(\rho=1/\kappa\), 중심 \(C=\mathbf r+\rho\,\mathbf N\).

---

## 4) 수치 해석 팁 (표본점에서 길이·곡률)

### 4.1 길이(폴리라인 근사)
표본 \((x_i,y_i)\)가 촘촘하면
$$
L\approx \sum_i \sqrt{(x_{i+1}-x_i)^2+(y_{i+1}-y_i)^2}.
$$

### 4.2 곡률(중심차분 근사, 2D)
균일 간격 \(t\)라면  
\(x',y'\approx\) 중심차분, \(x'',y''\approx\) 2차 차분을 사용해
$$
\kappa_i\approx
\frac{|x'_i y''_i- y'_i x''_i|}{\bigl((x'_i)^2+(y'_i)^2\bigr)^{3/2}}.
$$
노이즈가 있으면 **스플라인 보간/스무딩** 후 미분 권장.

## 5) 코드 예제 (SymPy로 상징, NumPy로 수치)

> 그대로 붙여 실행할 수 있는 **학습용** 코드입니다. (인터넷 불필요)

### 5.1 상징 미분으로 길이·곡률 (포물선)
```python
import sympy as sp
t = sp.symbols('t', real=True)
x = t
y = t**2

# 길이: ∫ sqrt(x'^2 + y'^2) dt
xp, yp = sp.diff(x,t), sp.diff(y,t)
speed = sp.sqrt(xp**2 + yp**2)  # sqrt(1 + 4 t**2)
L = sp.integrate(speed, (t, 0, 1))
print("Arc length [0,1]:", sp.simplify(L))  # sqrt(5)/2 + asinh(2)/4

# 곡률: |x'y'' - y'x''| / (x'^2 + y'^2)^(3/2)
xpp, ypp = sp.diff(xp,t), sp.diff(yp,t)
kappa = sp.simplify(abs(xp*ypp - yp*xpp) / (xp**2 + yp**2)**sp.Rational(3,2))
print("Curvature κ(t):", kappa)  # 2 / (1 + 4 t**2)**(3/2)
print("κ(0) =", sp.simplify(kappa.subs(t,0)))  # 2
```

### 5.2 헬릭스 곡률·길이(수치 샘플 + 시각화용 값)
```python
import numpy as np

def helix_samples(a=1.0, b=0.5, n=10001, t0=0.0, t1=10.0):
    ts = np.linspace(t0, t1, n)
    x = a*np.cos(ts); y = a*np.sin(ts); z = b*ts
    # 길이(연속해: sqrt(a^2 + b^2) * (t1 - t0))
    speed = np.sqrt( (-a*np.sin(ts))**2 + (a*np.cos(ts))**2 + (b*np.ones_like(ts))**2 )
    L_num = np.trapz(speed, ts)
    kappa_true = a/(a*a + b*b)
    return ts, x, y, z, L_num, kappa_true

ts, x, y, z, L_num, k_true = helix_samples()
print("Numeric arc length ≈", L_num)
print("Helix curvature (theory) =", k_true)
```

### 5.3 임의 2D 매개곡선의 곡률(중심차분)
```python
import numpy as np

def curvature_from_param(x, y, t):
    # 중심차분으로 1·2차 미분 근사
    xp = np.gradient(x, t)
    yp = np.gradient(y, t)
    xpp = np.gradient(xp, t)
    ypp = np.gradient(yp, t)
    num = np.abs(xp*ypp - yp*xpp)
    den = (xp**2 + yp**2)**1.5
    kappa = np.zeros_like(x)
    mask = den > 0
    kappa[mask] = num[mask]/den[mask]
    kappa[~mask] = np.nan  # 속도가 0인 점(첨점) 등은 정의 불가
    return kappa

# 테스트: 사이클로이드
a = 1.0
t = np.linspace(1e-3, 2*np.pi-1e-3, 10000)  # 첨점 회피
x = a*(t - np.sin(t))
y = a*(1 - np.cos(t))
k_num = curvature_from_param(x, y, t)
print("Curvature stats — min:", np.nanmin(k_num), "median:", np.nanmedian(k_num), "max:", np.nanmax(k_num))
```

## 6) 접촉원(오스큘레이팅 원) 구해보기 (평면, 매개형)

- 단위 접선 $$\displaystyle \mathbf T=\frac{\mathbf r'}{\|\mathbf r'\|}$$  
- 단위 법선 $$\displaystyle \mathbf N=\frac{1}{\|\mathbf r'\|}\,\frac{d\mathbf T/dt}{\|d\mathbf T/dt\|}$$ (부호는 방향 선택)  
- $$\displaystyle \rho=1/\kappa,\quad C=\mathbf r+\rho\,\mathbf N$$

**예제: \(y=x^2\)에서 \(x=0\)**
- \(\kappa(0)=2\Rightarrow \rho=\tfrac12\).  
- 접선은 \(y=0\) (수평), 법선은 \((0,1)\) 방향.  
- 접촉원 중심 \(C=(0,0)+\tfrac12(0,1)=(0,\tfrac12)\), 반지름 \(1/2\).

---

## 7) 해석·실무 팁

1) **매개 불변성**: \(\kappa\)는 재매개화해도 동일. (단, \(\mathbf r'\!=\!0\)이면 정의 불가)  
2) **첨점/뾰족점**: \(\|\mathbf r'\|=0\)이면 곡률 정의 안 되거나 \(\infty\)로 발산(사이클로이드 첨점).  
3) **수치 안정화**: 미분은 노이즈에 민감 → 스플라인 보간 후 미분 or Savitzky–Golay 필터.  
4) **극좌표 곡률**: 장미곡선, 로그나선 등 극형식 곡선 분석에 매우 유용.  
5) **호길이 매개**: \(s\)-매개로 바꾸면 \(\|\mathbf r'(s)\|=1\), \(\kappa=\|\mathbf T'(s)\|\)로 단순.

## 8) 연습 문제 (풀이 포함)

### Q1. \(y=\ln x\)의 곡률을 \(x>0\)에서 구하라.
$$
f'(x)=\tfrac1x,\quad f''(x)=-\tfrac1{x^2}
\Rightarrow \kappa(x)=\frac{|{-}1/x^2|}{\bigl(1+1/x^2\bigr)^{3/2}}
=\frac{1}{x^2\,(1+1/x^2)^{3/2}}
=\frac{1}{(x^2+1)^{3/2}}.
$$

### Q2. 로그나선 \(r=ae^{b\theta}\)의 곡률.
앞서 유도: $$\displaystyle \kappa(\theta)=\dfrac{1}{a\,e^{b\theta}\sqrt{1+b^2}}.$$

### Q3. 포물선 \(\mathbf r(t)=\langle t,t^2\rangle\)에서 \(t=1\)의 곡률.
$$
\kappa(1)=\frac{2}{(1+4)^{3/2}}=\frac{2}{5^{3/2}}.
$$

### Q4. 원호 \(x=a\cos\theta,y=a\sin\theta\)의 곡률이 상수임을 보여라.
위 2.3에서 \(\kappa=1/a\) (상수).
